　　不过谁都知道，尸王后卿正在闭关炼化将臣之血，等他出关的时候，必然又将带来一股血雨腥风。
<br/>　　这是空界所有势力压在心头的一块大石头。
<br/>　　小九告诉他，在空界，这件事只对一个人有好处，那就是林三生，只要战争一天不结束，阐教和佛门就不能把他怎么样，只能眼看着他发展自己的势力，林三生当然不会错过这个机会。
<br/>　　两人正聊着，宫殿外面传来阿黄的声音：“主上，林大元帅到了。”
<br/>　　这么快就听到消息了！
<br/>　　叶少阳跟小九互相看了一眼，小九道：“有请！”
<br/>　　“等下！”叶少阳急忙说道，指了指小九的身上，“你这衣服见客可不行么，赶紧去换一套。”
<br/>　　小九红着脸冲他笑了笑，进去换了一套平时穿的古装，这才让阿黄请林三生进来。
<br/>　　林三生是跟瓜瓜一起来的——问了才知道，原来是瓜瓜一个在外面没意思，干脆去元帅府找林三生了，两人合计了下，为避免让叶少阳再跑路，干脆直接过来。
<br/>　　都是自己人，小九也没跟他们客气，领他们到客厅，泡茶说话。
<br/>　　叶少阳迫不及待地要去见道风，问林三生把山河社稷图带来没有。
<br/>　　“你过来找我，正是为了这个，怎么能不带。”林三生把山河社稷图拿出来，在桌子上展开。
<br/>　　叶少阳请他们跟自己一起进去，林三生婉拒了，小九本来是很想陪着他的，陡然想到芮冷玉在里头，笑着摇了摇头，道：“道风跟我们毕竟生分点，我们要是在，有些话他不好说，还是你自己进去吧。”
<br/>　　瓜瓜是无所谓的，跟着叶少阳一起进去。
<br/>　　洪荒世界。
<br/>　　大致都还是以前的样子。
<br/>　　叶少阳跟瓜瓜在石碑附近站了一会，辨认了一下方向，没走多远，就有几个道士过来，其中一个见过叶少阳，立刻上前打招呼。
<br/>　　交谈得知，他们是林三生安排在石碑附近的守兵，虽然说这山河社稷图外人进不来，但林三生还是不放心，派了一些自己的心腹在这里守卫。
<br/>　　得知他们是来看望道风和芮冷玉的，这个道士立刻为他们引路，陪他们上山，路上叶少阳问起道风二人现状，这道士答道：“风神一向在房间里，只有芮老师一个人伺候他，我们不可能见到的，但是芮老师跟我们倒是每天都能见到。”
<br/>　　“你们叫她啥？”叶少阳皱眉。
<br/>　　这道士笑道：“芮老师啊，你知道的，我们都是生在这个世界的，当初祖师为我们开悟，用的是人间之道，但我们对人间了解毕竟不多，正好芮老师来了，每天都会开坛讲课，传授我们人间的知识，大家都很喜欢她，因此叫她老师。”
<br/>　　叶少阳这才了然。其实“老师”这个词并不是当代才发明的，在古代就有，跟“师父”的意思差不多，不过与师父传弟子不同，师父对弟子，是言传身教地传授技艺，也就是一门技术。
<br/>　　而老师对学生，更多是传道，教的一般是文化方面的知识和做人道理等等。
<br/>　　孔子的那些弟子，叫孔子都是老师而不是师父。
<br/>　　过了结界，一路上山，来到道观前面，还没进去，叶少阳就听到了一阵整齐地朗读声，仔细听了一下朗读的内容，当场就懵了：
<br/>　　“一一得一，一二得二，一三得三……”
<br/>　　乘法口诀？？
<br/>　　叶少阳没让那道士禀报，自己带着瓜瓜，从偏殿绕进去，那道士就在殿门外呆着。
<br/>　　一个道士从后面匆匆赶来，对他耳语了什么，这道士勃然变色，跟着他一起朝后山走去，在一片树丛之中，发现了一具尸体，脑袋破碎，鲜血淋漓地躺在乱石中间，
<br/>　　这道士附身检查，发现尸体都硬了——他们都是长在这里的生灵，跟人间不同，他们在这里，肉身跟魂魄是一体的，肉身死了，也就魂飞魄散了。
<br/>　　“怎么样？”这道士问道。
<br/>　　“跟上周那具尸体一样，也是被吸干了血……应该是昨晚死的。师兄，会不会是化蛇偷袭？”
<br/>　　这道士摇摇头，“化蛇若是能打开这结界，死的就不是一个两个了。”
<br/>　　“可是……除了化蛇，还会有什么人这么干呢？”
<br/>　　“没准是附近的什么邪灵所为，你们快去把尸体掩埋，不要声张！”
<br/>　　这道士望着地上这具被吸干了血的尸体，展开了某种不可思议的联想。
<br/>　　叶少阳站在偏殿的外面，隔着一层布帘，朝内堂偷看。
<br/>　　内堂里，坐着很多道士，还有一些没化形的妖精，一人一只方凳，老实地坐着，一起背诵着乘法口诀，所有人都望着正前方，在他们对面，内堂三清神像的前面，竖着摆着一块很大的白色木板，上面写着一排排的毛笔字，正是乘法口诀。
<br/>　　黑板旁边站着一个身穿衬衫和短裙的姑娘，只是看到个侧影，叶少阳就认出她是芮冷玉。
<br/>　　瓜瓜想要叫她，被叶少阳拉住了，然后饶有兴致地是在一旁偷看。
<br/>　　一群异界的道士，还有妖精，一起背诵乘法口诀的样子，真的……让他有一种错乱的感觉。
<br/>　　等大伙都背诵完之后，芮冷玉走到讲台中间，摆手说道：“好了，这个你们回去自己接着背诵，我们接着上物理课，牛顿的三大定律……”
<br/>　　“噗……”叶少阳差点一口喷出来，被一个道士听到，转头看到他，举手告诉芮冷玉：“老师，那边有人偷听。”
<br/>　　芮冷玉转过头来，跟叶少阳的目光撞在一起，叶少阳吐了吐舌头。
<br/>　　愣了几秒钟，芮冷玉脸色泛红，对学生们说道：“好了，物理课下午再上吧，老师有点事，待会见。”
<br/>　　这些道士们一起站起来，那些长得像蜥蜴的、长得像蛇的、长得像老鼠的，等等妖精也都挺直了腰杆，齐声喊道：“老师再见！”
<br/>　　叶少阳又差点要吐血。
<br/>　　等他们走了之后，芮冷玉笑着走到叶少阳面前。
<br/>　　“嫂子！”
<br/>　　瓜瓜亲热地叫了一声。
<br/>　　芮冷玉揉了揉他的脑袋，微笑着望着叶少阳，半晌说道：“你怎么来了。”
<br/>(本章完)
