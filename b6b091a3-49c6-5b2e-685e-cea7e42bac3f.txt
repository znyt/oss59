　　“死吧！”
<br/>　　冲上天穹的雷灿，口中发出一声怒喝。
<br/>　　只见其身上雷光闪烁，噼里啪啦的爆响，其手中出现了一柄硕大的雷刀，恐怖的雷霆威压弥漫八方。
<br/>　　“斩！”
<br/>　　雷灿状若天神，双手持刀，猛然向下一劈。
<br/>　　霎时之间，一道恐怖的刀气从九天之上劈下，长达万丈不止。
<br/>　　这刀气，分明就是一道闪电，蕴含爆炸般的雷霆之力，速度快到巅峰，威力强大到极致，瞬息间撕裂无尽长空，向苏莫当头劈去。
<br/>　　刀气犹如是天地雷罚，蕴含强大的雷霆奥义，和一缕天道意志，威力无可匹敌。
<br/>　　天地间属性的种类奇多，除了基本的五行属性之外，风、雷、光、暗、阴、阳等等各式各样的属性。
<br/>　　若说攻击力最为强大的单一属性之力，莫过于雷属性之力了。
<br/>　　雷属性力量，刚猛霸道，克制阴邪，最主要的是，雷霆之力之蕴含一丝天道意志，所以才最为强大。
<br/>　　雷霆，是雷罚，是天道的武器。
<br/>　　“的确远超同阶武者数倍不止，但……还是太弱了！”
<br/>　　苏莫抬头望着急速劈下的闪电，微微摇头，随即他手中剑光一闪，一剑逆斩而上。
<br/>　　这一剑，苏莫催动了五种剑之战魂。
<br/>　　刹那之间，冲天的剑威撕裂了苍穹，耀眼的三色剑气裹挟无边的锋芒，以比闪电还要快的速度，直击当头而来的闪电。
<br/>　　瞬息之间，三色剑气与浩大的闪电相击在了一起。
<br/>　　轰！
<br/>　　一声爆响震动了整个战界，在所有人震惊的目光中，浩大的闪电仿佛是纸糊的一般，瞬间被撕的支离破碎。
<br/>　　锋芒绝世的三色剑气，斩碎了浩大的闪电之后，速度丝毫不减，裹挟惊天杀机，直接向雷灿的身上斩去。
<br/>　　“什么？”雷灿见此，顿时吓的亡魂皆冒。
<br/>　　这一剑速度太快了，这一剑的威势太恐怖了，被剑气锁定之下，他都感觉自己的身躯要被撕裂了，神魂即将离体。
<br/>　　他后悔了，他完全没有想到，苏莫的实力居然恐怖到了如此的地步！
<br/>　　但是，现在显然不是后悔的时候。
<br/>　　电光火石之间，他完全来不及躲避，只能仓促的将手中雷刀向前一横，企图挡住三色的剑气。
<br/>　　轰！
<br/>　　三色的剑气狠狠的斩在了雷刀之上，一声爆响，雷灿全身剧震，雷刀遭到强大力量的攻击，瞬间崩飞了出去。
<br/>　　而后，强大无比的三色剑气，直接斩在了雷灿的身上。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声响起，强大的剑气斩过，直接撕碎了雷灿的身躯。
<br/>　　鲜血喷涌，碎肉飞洒，雷灿的身体被剑气裹挟的锋芒撕成了碎片，连战魂都被撕碎了，没有逃生的机会。
<br/>　　整个战界一片死寂，所有的人，无一例外，全部抬头望着天穹上的这一幕，个个瞪圆了双眼。
<br/>　　三色剑气撕碎雷灿身躯的情景，在众人的脑海之中，仿佛成了一副永恒的的画面。
<br/>　　一招！
<br/>　　秒杀！
<br/>　　强大的雷灿，战力超强的雷灿，同阶之中，被一招秒杀了！
<br/>　　震惊！
<br/>　　震撼！
<br/>　　所有人都震惊的无以复加，心中掀起滔天骇浪，若不是亲眼所见，他们绝对不会相信这个事实。
<br/>　　众人都知道苏莫强大，越级战斗轻而易举，但何曾想到苏莫的实力强大到了如此地步！
<br/>　　雷灿的实力，比一般的武帝境五重武者都要强大，直追武帝境六重初期武者，
<br/>　　苏莫能一招秒杀雷灿，说明苏莫的战斗力，完全达到了武帝境六重初期的地步，甚至还不止。
<br/>　　啪嗒！啪嗒！啪嗒！
<br/>　　雷灿变成碎片的尸体跌落在海面上，传出一阵声响，将愣住的众人纷纷惊醒。
<br/>　　嘶！
<br/>　　一瞬间，倒吸冷气的声音此起彼伏。
<br/>　　随即，十几座观战的岛屿之上，立刻炸开了锅。
<br/>　　“我的天呐！苏莫的战力居然这么恐怖！”
<br/>　　“太逆天了，一招就秒杀了雷灿！”
<br/>　　“强大，这才是真正的强大，战力超凡啊！”
<br/>　　“我本以为苏莫就手段诡异一些，没想到自身战力也这么恐怖！”
<br/>　　惊呼声四起，整个战界都一片嘈杂，所有人都是被苏莫强大的战力震惊了。
<br/>　　“没想到我还是小看了你！”木璃导师立身于岛屿的边缘，清冷的目光凝视着海面上的苏莫。
<br/>　　从上次苏莫斩杀冷江一役，她就知道苏莫战力超强，但今日这一战，苏莫表现出的战力，似乎更强几分。
<br/>　　而且，苏莫不只是战力强，这修炼的速度未免也太快了吧！
<br/>　　火元圣师也是发现了苏莫修为的进展，所以放弃了教授苏莫炼器的打算，放任其自行成长。
<br/>　　这种修炼天赋，或许武道之路比炼器之道更适合。
<br/>　　各大派系的人震惊的同时，纷纷眸光闪烁，随即立刻派人离开战界，将此地的消息回禀派系高层，或者各自派系的圣子。
<br/>　　苏莫战力如此逆天，他们则不能这样静观其变了。
<br/>　　有人欢喜有人忧，其中一座小岛上，一大帮太史世家派系的人，个个脸色难看。
<br/>　　尤其是尤天韩，脸色已然铁青，双拳紧握，甚至都想亲自出手抹杀苏莫了。
<br/>　　圣子太史爵将此事交给他，原本以为轻易便能搞定，现在不仅没有搞定，反而出的丑更大了。
<br/>　　而且，每一个天才的陨落，都是太史世家派系的损失。
<br/>　　苏莫身形不动，伫立在海面上，随即他大手凌空一抓，雷灿的雷刀和储物戒飞射到了他的手中。
<br/>　　这雷刀居然是一件尊级下品宝器，相当不凡。
<br/>　　没有仔细查看，苏莫便将雷刀和储物戒收了起来。
<br/>　　随即，他目光转动，扫视十几座岛屿，最后落在了尤天韩的身上。
<br/>　　“你们太史世家派系，还有人要挑战吗？”苏莫淡淡的问道，眸中杀机一闪而逝。
<br/>　　当初赵城和郭剑青追杀他，不仅是受了太史轩之命，估计也受了此人之命。
<br/>　　只不过，对方修为高达武帝境九重巅峰，他目前还不是对手，不然绝不会饶了此人。
<br/>　　众人听闻苏莫之言，立刻安静了下来，所有人都望着尤天韩，想要看看对方如何应对。
<br/>　　太史世家派系，绝不可能就此罢手，定然会展开反击。
<br/>　　尤天韩铁青着脸，望着淡然而立的苏莫，他的眼眸微微眯了下来。
<br/>　　“林无影，你出手吧！”
<br/>　　少倾之后，尤天韩淡淡的说道。
