　　第612章
<br/>　　东方童一张娃娃脸，此刻却是怒目圆睁，谁也不会怀疑，这个一直笑眯眯的人，会立刻拔刀杀人。
<br/>　　但当他触及到程大雷的目光时，却突然顿住了。程大雷的目光太过平静，他就斜靠在石头上，却给人一种居高临下的感觉。这感觉令东方童觉得，如果他敢向程大雷出手，死的一定是自己。
<br/>　　他挤出一丝笑容：“不必了，我们自己忙得过来。”
<br/>　　“如果需要帮忙的话，一定要说话喔，我这个人很是助人为乐喔。”
<br/>　　“多谢，多谢。”
<br/>　　东方童虚伪的道了两声，几个人一起动手，点燃火把走出山洞。亏得现在雨小了许多，否则举着火把出去，立刻就会被雨淋灭。
<br/>　　本来拥挤的山洞此刻又变得空旷起来，小白狼和程大雷面面相觑，两个人都是一头雾水。
<br/>　　“他们是干什么的？”小白狼忍不住开口问道。
<br/>　　程大雷摊开双手：“我怎地知道，这帮人怎么看怎么都有些古怪啊。”
<br/>　　程大雷挠挠头，想不清楚东方童一行人为何如此重视这女人。不过，黑天半夜又下着雨，东方童想要找到那黑衣女子怕是不容易。
<br/>　　程大雷虽然有些好奇心，但也不会打破砂锅问到底。东方童走了更好，他还能待得宽敞些，等明天天一亮，就出发继续赶路。
<br/>　　又过去了很久，山洞外的雨还没有停，却突然响起脚步声。那黑衣女子竟去而复返，跑进山洞后直接向程大雷跪倒，口中道：“恩人救我。”
<br/>　　淋了一场雨，衣物贴在身上衬得这女人曲线玲珑，又因为天寒雨冷的缘故，她的娇躯轻轻颤抖。
<br/>　　如此一个女人跪在程大雷面前，程大雷又不是坐怀不乱之人，他赶忙伸手去扶，口中道：“快快请起，快快……”
<br/>　　小白狼轻咳一声，拦在程大雷面前。她摆出了随身的短刀，挑了挑眉毛道：“什么来历，赶紧实话实说，不要逼姑奶奶用刀和你说话。”
<br/>　　黑衣女子抬起头，楚楚可怜的看着程大雷，口中道：“敢问是蛤蟆大王程当家么？”
<br/>　　此时莫说小白狼，程大雷也是一惊，手握住剑一字一顿道：“你是什么人？”
<br/>　　程大雷一路行来，都小心不和人碰见，如果被识破身份，这对程大雷来说绝对不是什么好事。
<br/>　　女人摇摇头，道：“程当家勿怪，小人名叫周桃，乃是京州长安人，当初曾听说过程当家化名牛三斤大闹长安城的伟事，况程当家如此英雄魁梧的美男子天下又能有几人，所以小人才能一眼认出程当家。”
<br/>　　“呃……哈哈哈。”程大雷托着下巴，忍不住哈哈大笑，瞧瞧世间还是有识货之人的。
<br/>　　“姑娘莫慌，快快起来说话。”
<br/>　　“程当家不肯救我，小人不敢起来。”
<br/>　　“这……你究竟碰到什么事了？”
<br/>　　周桃欲言又止，轻轻叹了口气：“那东方童并非什么药商，乃是京州一地的大盗，杀人放火无恶不作。小人是长风关守将陆长迢的妾室，他们抓走小人，就是为了勒索长风关。”
<br/>　　这长风关位于京州境内，处于帝国和草原的接壤地带，距离此地也并不遥远。程大雷虽然没见过哪位陆长迢，但对于附近的情形还是了解的。
<br/>　　“你瞧瞧……”程大雷长叹一声，道：“姑娘快快请起，这件事包在我身上，有我在，没任何人能伤害姑娘。”
<br/>　　小白狼冷哼一声，心底对程大雷十分不屑。谁知道这女人说得是真是假，程大雷只听对方一面之词，委实有些太莽撞了。
<br/>　　况且，就算她说得是真的，但这件事和我们又有什么关系。就算东方童是坏人，可我们也并不是好人呐。
<br/>　　正这时候，外面响起脚步声，东方童一帮人骂骂咧咧的回来了。
<br/>　　周桃一惊，道：“程当家，你定要救我。”
<br/>　　“不用慌，你先到后面躲躲。”
<br/>　　山洞后面有狭长的暗道，黑天半夜也看不清楚，几乎是周桃刚刚挤进去，东方童一行人便走进了山洞。
<br/>　　外面下着细雨，林子里伸手不见五指，东方童在外面转了一圈，也不可能找到。无可奈何，只好先回到山洞，等天亮以后再说。
<br/>　　他冲程大雷拱拱手，没了之前的热情。一群人坐在山洞里烤火，虽然没有埋怨刘凯，但刘凯却紧张得直冒冷汗。
<br/>　　是程大雷主动与他们打招呼：“这个……没找到么？”
<br/>　　一句话打开了东方童的话匣子：“那女人实在狡猾，如果让我找到她，一定立刻杀掉她，决不能让她再从我手中逃了。”
<br/>　　小白狼多长了一个心眼，问道：“那女人是什么人呐，难道不是你们的同伴？”
<br/>　　东方童一怔，与身边几人对视一眼，忽然长叹一口气。
<br/>　　“也就不瞒二位了，在下并非什么药商，乃是长风关的守军，在陆将军手下做事。”
<br/>　　咦。
<br/>　　程大雷和小白狼同时睁大眼睛，事情变得古怪起来。小白狼追问道：“那女人呢？”
<br/>　　东方童压低声音：“二位可听说过江湖上有位鬼脸婆婆。”
<br/>　　小白狼一头雾水，对于这四个字很是陌生。程大雷却是睁大眼睛，难以置信道：“她重出江湖了？”
<br/>　　传说这位鬼脸婆婆，精通易容术，能在一瞬间改妆成完全不同的一人，于是才有鬼脸之称。当年她在京州一地，很是做了许多件大案，手下的人命不少。程大雷在长安时，也听过这位前辈的名声。
<br/>　　“那贼女人就是鬼脸婆婆。”东方童咬着牙道。
<br/>　　“这……不大可能吧。”程大雷吃惊道：“那鬼脸婆婆成名在二十年前，现在就算不死也是一大把年纪，可那女人……”
<br/>　　东方童摇摇头：“具体我也说不好，那鬼脸婆婆擅长易容术，有鬼神不测之术，乔装成年轻妇人也未必做不到。也有可能，这女人是鬼脸婆婆的传人，孙女徒弟什么的，也说不好。”
<br/>　　事情变得复杂起来，程大雷不知该不该相信东方童，但至少要对周桃有所保留，最关键的一点是……
<br/>　　此刻周桃就在他身后不远处。
<br/>(本章完)
