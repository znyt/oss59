　　囚室正按即行的轨道缓慢运作着，给他的感觉，像是在下降。
<br/>　　不知过了多久，刺眼的光亮照射了进来，有些刺眼，让刘哔眯了一下眼睛。
<br/>　　前方的金属大门上，逐渐打开了一道缺口，一个玻璃罩子将他和外界隔离了开来。
<br/>　　待到看清来人的脸庞时，刘哔有些意外，玻璃窗外站着的，是他原本的队友，旧二多福。
<br/>　　“刘先生，几天没见，你憔悴了好多。”看到眼前的囚室打开，旧二多福向前走了一步，将手中的盒子放在了台子上面。
<br/>　　看着多福的眼神有些平静，刘哔随意的站在囚室中央，没有接话。
<br/>　　他有些好奇，这家伙来找自己是要做什么，对于旧二多福，他还真没有太多的了解。
<br/>　　“呵呵，刘先生，我给你带了一些美味的食物，几天没有进食，你一定是很饥饿了吧。”多福嘴角露出微笑，非常客气的将盒子推进一个暗格之中。
<br/>　　随后，囚室右边的墙壁上，裂开了一道口子，一个样式简洁的饭盒，静静的被搁置在了其中。
<br/>　　目光稍稍闪动了一下，刘哔走了过去，伸手摸索了一下，拿出了饭盒。
<br/>　　暗格中的通道并不大，而且很曲折，想利用这里逃离出去，几乎是不可能的事情。
<br/>　　随着他将饭盒取出，暗格瞬间关闭，刘哔打开了饭盒的盖子，一股鲜美的人肉香气从中飘散了出来。
<br/>　　“说吧，有什么事情。”盖上饭盒的塑料盖子，刘哔向前走了几步，和旧二多福互相对视着。
<br/>　　刚进食过一具SSS级别喰种的尸体，他并不怎么饥饿，况且，CCG给提供的食物，他也不怎么想吃。
<br/>　　虽然特殊的体质让他不会中毒，可谁知道这份食物取材于什么地方。
<br/>　　对于食物的标准，他可是非常严苛。
<br/>　　“刘先生，总议长大人要我传话，如果你肯归顺和修一族，那我们可以立即将你释放，并为你在CCG创造一个全新的身份，不知道你的意思是？”
<br/>　　看到刘哔似乎对他提供的食物没有兴趣，旧二多福目光闪动了一下，依旧笑吟吟的开口问道。
<br/>　　他相信，这样丰厚的条件是不会有喰种拒绝的，有一个被官方承认的人类身份，这是多少喰种做梦都想做到的事情。
<br/>　　更不要说，对方的行动还受到了限制，这座牢狱当初CCG打造的时候花费了巨资，就连SSS级别的喰种都不可能从内部逃离出来。
<br/>　　与其说是询问，倒不如更像是一种威胁，如果不答应总议长大人的条件，那就做好死在其中的准备吧。
<br/>　　“哦？和修一组吗。”低喃一声，旧二多福的话语让刘哔提起了一些兴趣：“归顺的话，倒也不是不可能，只不过，在此之前先要谈谈有什么限制。”
<br/>　　“当然，需要动一个小小的手术，想必刘先生应该不会介意吧。”似乎已经预见了谈判的成功，多福脸上的笑容更加灿烂了。
<br/>　　当然要有所限制，否则他们家族又怎么得以控制那么多手下。
<br/>　　血缘？利益？人情？
<br/>　　别开玩笑了，这些东西，怎么可能约束喰种，更不要说是狡猾的人类了。
<br/>　　出乎意料的，刘哔嘴角带起一抹玩味的笑容，缓缓摇了摇头：“抱歉，我很讨厌别人在我身体上动手动脚，所以必须手术的话，我想我还是继续待在这里吧。”
<br/>　　开玩笑，这个地方他如果真想出去的话，还用得着答应什么条件吗？
<br/>　　之前和有马贵将的战斗，也是存留了想要试探其深浅的心思，如果他想要逃跑，有马也很难将他抓住。
<br/>　　甚至为了隐藏实力，一些底牌他还没有展露出来呢。
<br/>　　之所以故意被抓进来，是因为他知道，用不了多久，青铜树就会进攻这里，到时候，很多高阶喰种都会出现。
<br/>　　包括，SSS级的‘独眼枭’，这也是刘哔此行，最大的目标。
<br/>　　这份计划，在他向CCG透露古董消息的时候，就开始制定了，一直到现在为止，计划倒也没出现什么纰漏。
<br/>　　而店长这边的收获，也是出人意料的丰盛，以他现在的实力，‘独眼枭’恐怕已经不是他的对手了。
<br/>　　看到刘哔脸上坚定的神色，旧二多福有些气恼，他没想到对方会拒绝这样优厚的条件。
<br/>　　不过也没关系，机会还有很多，他有机会和这家伙慢慢耗下去。
<br/>　　囚室内的那股滋味，他可是也亲身体会过的啊，那种幽闭死寂地方，怎么可能忍受的了。
<br/>　　他相信，在这种环境下，用不了多长时间，刘哔就会彻底屈服。
<br/>　　“既然刘先生这样说了，那也没办法，过段时间，我会继续来看刘先生的。”旧二多福站起身来，继续说道：“希望刘先生到时候会改变现在的想法。”
<br/>　　说罢，多福没再给刘哔回答的机会，径直从一旁的金属门中离开，囚室前方的金属墙壁，也缓缓闭合了起来。
<br/>　　……
<br/>　　接下来的很长一段时间里，监狱都没有什么动静，没有任何光亮，四周的环境也总是死寂黑暗的，刘哔早已经缺失了时间的概念。
<br/>　　他终于知道先前旧二多福为什么会有那般信心了，这样的监禁方式，比世界上任何酷刑都要残忍。
<br/>　　孤独，如同潮水一般将他淹没，为了避免自己发疯，刘哔只能将全部的精力都投入到了修炼当中。
<br/>　　在这段时间里，他经过精挑细选，也终于从商城挑选出了适合自己的物品。
<br/>　　他不缺技能，功法也能使用一段时间，血脉什么的，也暂时不需要进行强化。
<br/>　　现在对他实力提升帮助最大的，就只有两种方法，或者提升自己的身体素质，要么就是兑换一些装备。
<br/>　　两者各有优劣，兑换属性点来提升身体素质的话，价格太高，一万多的积分花进去，基本起不到太大的效果。
<br/>　　但是兑换装备的话，虽然短时间内战力提升的要多一些，可性价比却远远不如前者了。
<br/>　　毕竟装备这种东西，总是会进行更换的。
<br/>　　思索再三之下，最终刘哔还是选择了兑换装备，在面对独眼枭之前，将自己的战力提高一些会更有把握。
