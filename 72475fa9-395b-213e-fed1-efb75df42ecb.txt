　　在街道两旁的建筑顶上飞跃着，刘哔已经跑了许久，早就逃离了正处于混战当中的古董咖啡厅。
<br/>　　但他依旧能感受到，身后有几道强横的气息，一直在自己后面穷追不舍。
<br/>　　追击他的，是那几个特等搜查官们，以及铃屋什造，在眼皮子底下被抢走战利品，他们自然恼怒至极。
<br/>　　有‘死亡之翼’的增幅，就算这些人的等级比刘哔高出了不少，可是想追上他，也是天方夜谭的事情。
<br/>　　眼看着即将脱身，刘哔嘴角露出一抹阴谋得逞的笑容，距离店长死去的时间才过了十几分钟，他的时间还很充裕。
<br/>　　突然之间，像是察觉到了什么，刘哔的身影向旁边一躲，脸上的笑容凝固了下来。
<br/>　　“砰！”
<br/>　　在他原本的前进路线上，炸起了一朵金色的电光，强大的冲击力，将整个楼顶都轰成了粉末。
<br/>　　停下自己的脚步，刘哔目光落在前方的身影上，脸色阴沉了下来。
<br/>　　那是一个长相英俊的年轻男人，身穿一件纯白色的风衣，戴着一副金丝眼镜，显的有些斯文。
<br/>　　稳住了身体，刘哔的视线，停留在了白衣男子胸前的牌子上，那是一枚金色楼边的身份牌，在整个CCG里，仅此一枚。
<br/>　　这代表着CCG明面上的最高战力，有马贵将。
<br/>　　“捉拿‘修罗’，将其驱逐。”有马伸出左手，声音有些机械，随即一团强烈的金光，无声无息的从其手中激射了出来。
<br/>　　速度太快，刘哔根本来不及躲闪，在他看见金光的时候，有马的攻击已经落在了他的面前，千钧一发之际，他只来得及将龙头拐杖横在胸前。
<br/>　　“轰！”
<br/>　　一股巨力袭来，他手中的龙头拐杖，脆弱的不像是一件鬼器，和金光接触的一瞬间，整根拐杖就碎裂成了无数碎片。
<br/>　　金光速度丝毫未减，按照原有的攻击轨道，落在刘哔胸口上，猛然炸开。
<br/>　　直至现在，尖锐的破空声方才响了起来，伴随着一道剧烈的轰鸣声，将他的身体轰进了下方的建筑里。
<br/>　　没给他任何喘息的机会，耳边的音鸣感还未褪去，有马的身影便贴了上来，以手为刀，顺着刘哔的头顶立劈了下来。
<br/>　　“有马。”联络器中传来一道低沉的男声，有马贵将眼中闪过一抹暴虐，像是在挣扎着什么。
<br/>　　最终，他还是将手臂向上微微抬了一些，修长的左手，劈开了刘哔的小丑面具，在其脸上留下一道浅浅的疤痕。
<br/>　　因为先前的攻击，刘哔身体内仿若有无数电流乱窜一般，动弹不得，有马贵将从腰间摸出一道手铐，将他的双手锁了起来。
<br/>　　远处一众特等搜查官们姗姗来迟，看到有马贵将之后，神态恭敬的走了过来。
<br/>　　“有马特等，这是你的库克因，手感超棒诶。”什造脸上带着轻松的笑容，将手中的长刀递了过来。
<br/>　　看到地上被束缚住的刘哔时，什造表情一僵，目光有些诧异。
<br/>　　但他背后的黑色翅膀做不了假，方才出手抢走他们战利品的，就是此人。
<br/>　　有马接过长刀，并没像其他人一样将其转化为手提箱，反倒是随手别到了自己腰间。
<br/>　　“修罗已捕获，正在押送。”拿起椭圆形的联络器，有马语气平静，提着无法行动的刘哔，上了一架直升机。
<br/>　　……
<br/>　　耳旁是呼啸的风声，还有螺旋桨转动的声音，似乎CCG对于他十分重视，有马贵将亲自押解，让他根本没有逃走的机会。
<br/>　　大概飞了半个钟的功夫，直升机降落在了一处停机坪前。
<br/>　　暗自在心中记下了路线，刘哔估摸着已经飞离了市区，有马要带自己去哪，他也不太清楚。
<br/>　　“下去。”直升机还未停稳，有马抓起刘哔，纵身跳到了下面的空地上。
<br/>　　这里似乎是一处天台，四周没什么人影，很是空旷，不远处有一个造型别致的金属门，充满了超前的科技感。
<br/>　　挣扎了几下，刘哔发现手中的镣铐异常坚硬，想要打开却不是短时间可以做到得了。
<br/>　　况且，就算解开了镣铐，有马贵将在旁边押解着，他也没有逃脱的机会。
<br/>　　拽着刘哔的衣领，有马开启了金属门，带他一起走了进去。
<br/>　　门后的空间很大，呈圆柱形，四周都是冰冷的金属墙壁，大厅中央有一根柱子，用十字形的过道连接到了各个楼层。
<br/>　　两人没有任何交流，有马乘坐电梯，带他来到了五楼的一间办公室内。
<br/>　　一个头发有些秃的值班人员正在玩着手机，见到有马贵将，立即从椅子上站了起来。
<br/>　　“新犯人，修罗，疑似小丑组织成员，单独关押。”推了推眼镜，有马脸上没有任何表情。
<br/>　　似乎是清楚有马贵将的脾气，值班人员并没有多问，只是按照流程，抽取了刘哔的血液和指纹，拍了几个角度的照片，用来做档案。
<br/>　　资料登记完毕，值班人员操作电脑，双手在键盘上操作了一番，站起身来。
<br/>　　“有马特等，113号囚室，已经可以进去了。”微微躬了一下身，值班人员脸上带着讨好的笑容。
<br/>　　点了点头，有马经车熟路的，带刘哔穿过了几个过道，来到一处空旷的场地前。
<br/>　　四周有几个金属铁门，其中一间敞开着，里面空间很小，空无一物。
<br/>　　看到这幅场景，刘哔也证实了心中的猜测，这里正是CCG关押喰种的地方，不知道有多少高阶喰种都被关押进了这里。
<br/>　　以他目前的实力，只要被关进了这里，基本上也没有什么希望可以逃离出来。
<br/>　　脸色有些阴冷，他心中有些懊恼，他很确定，自己不可能在有马贵将面前逃离出去。
<br/>　　提着刘哔的衣领，有马没有任何话语，将他往囚牢的方向扔去。
<br/>　　没有任何反抗，刘哔顺着有马贵将的力道，向前滑行了一段距离，在即将进入的时候，展开了背后挂满黑色羽毛的翅膀，双脚在墙上一蹬，向入口处冲去。
<br/>　　他想要尝试一下，凭借自己全力爆发的速度，或许他还有一丝逃离的契机。
