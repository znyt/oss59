　　“呼——！”
<br/>　　看着那片差点就把自己手中攀登绳给切断，让自己以自由落体的方式跟地面来一个亲密接触的直升机螺旋桨，重重的舒了一口气。
<br/>　　如果这片螺旋桨再往前划出一点，自己就死定了。
<br/>　　可是布谷悬着的心还没彻底放下，那片螺旋桨便是“嘎吱”一声，然后朝着地面坠落下来。
<br/>　　“妈的！”
<br/>　　布谷眼神凝沉，脚在大楼外墙上用力一蹬，抓着绳子向旁边荡开。
<br/>　　“呼！”
<br/>　　直升机螺旋桨呼啸着从布谷身旁坠落下去，直插在一辆停在路边的轿车上。直升机螺旋桨从车顶刺入，直刺进轿车中。之前被大楼上掉落下来的玻璃碎片刮花，只需要补个漆的轿车，这次是真的要返厂大修了。
<br/>　　那架被布谷当头一炮的“小鸟”直升机冒着黑烟歪歪扭扭的朝着旁边的一栋楼房撞去，直升机驾驶舱中，两个飞行员拼尽全力控制着直升机，想要避免坠机的结果，嘴里大声叫着“Mayday！Mayday！”
<br/>　　只是这个时候根本不可能有人来救他们，套用《九品芝麻官》中的一句话，就是“你叫吧，就算是叫破喉咙也不会有人来救你”。
<br/>　　在两个飞行员瞪大的惊恐眼神中和尖叫声中，“小鸟”直升机一头撞进了那栋大楼中，发生了剧烈的爆炸，一团火球冲破大楼，带出大量的混凝土碎块砸落在街道上。
<br/>　　被炸成了废铁，还在继续燃烧中的“小鸟”直升机晃动了一下，从楼房中掉落下来。
<br/>　　砸落在街道上，停在街边的几辆轿车瞬间就被直升机砸扁，彻底报废。
<br/>　　顺着索降绳滑到地面上，布谷快速解开身上的登山扣，摘下背在身后的M200狙击步枪，拉开枪栓，确认子弹已经上膛后，立刻从大楼和大楼之间的小路穿过，朝着跟天蛇的汇合点跑去。
<br/>　　……
<br/>　　博物馆内，罗昊从一名倒地的“黑箭”队员身边捡起一把安装了红点瞄准镜的M4A1战术步枪，可是刚把枪拿到手中，就发现步枪枪带被拉扯住了，扭头看去，发现那名“黑箭”队员只是受了重伤，并没有彻底端起，他满是鲜血的手丝丝的抓着枪带，不让罗昊把枪拿走。
<br/>　　罗昊也不多话，从那名“黑箭”队员身上拔出他的M9军刀，一道扎进他的脖子中，顺势一转刀柄，彻底切断了对方的生机。拔出沾满血的M9军刀一刀切断被对方拽在手中的枪带，又从对方摸了两个备用弹匣，一发枪榴弹，两个手雷和一把手枪在身上，翻滚到一边，对着那些从博物馆外索降跳入博物馆的force-recon队员射击。
<br/>　　“撤！快撤！”
<br/>　　齐凡大声叫喊着，扔出一颗从“黑箭”队员身上顺来的烟雾弹。
<br/>　　浓密的白烟瞬间弥漫在了博物馆中，白烟触发了博物馆中的灭火装置，安装在天花板上的喷水头弹出，喷出细密的水珠。
<br/>　　等到博物馆大厅中的白烟散开，博物馆中已经没有了战术中队的影子。维尔纳特推开身边的“黑箭”队员，满是狼狈的走到一名FoRCe-ReCoN队员面前，抬脚就朝着对方踹去，在对方充满怒火的瞪视下，捡起地上那支之前被对方踩在脚下的断手。
<br/>　　因为失血的关系，那只断手已经是变得惨白，毫无血色。以现在的医疗技术，断肢是完全有可能被重新接上的，但是运动功能受到创伤是肯定的，就算是现在就把断手接上，维尔纳特的右手也很难恢复不到以前的状态，一秒钟内完成拔枪、上膛、瞄准、射击更是不可能再做到。
<br/>　　好一点，维尔纳特可能会去军队的后勤单位工作；差一点，维尔纳特会被强制要求退役，而在他的服役经历上，除了说明他曾经在海军陆战队服役过，没有其他任何的经历和立功经历。
<br/>　　因为他的任务性质注定了他过去在force-recon中的服役经历不会出现在他的简历上。
<br/>　　离开军队维尔纳特能做什么？
<br/>　　物流公司的货车司机？如果他的手恢复的不够好的话，他可能连货车司机都做不了！他能做的可能就是外卖员，快递员这类的工作，在他人的抱怨和咒骂中，完成自己的工作，赚取微薄的收入。
<br/>　　当他回家的时候，他看到的可能不是温馨的晚餐，而是仍满地板的衣服，以及在房间大床上跟别的男人相拥在一起的妻子，然后在他的怒骂声中，他们之间的婚姻走到尽头。
<br/>　　那些负伤从force-recon退役的军官和士兵，以及还有那些在行动中阵亡的force-recon队员有什么样的待遇，维尔纳特见过很多。他们本应该是国家的英雄，可是他们任务的性质却使他们注定默默无闻！
<br/>　　不，这不是维尔纳特想要的，他是海军陆战队的明星，他以后注定会成为一个将军，然后在众人的注视下走进五角大楼国防部的大门！他不要复制过去那些force-recon队员退役后的生活！
<br/>　　维尔纳特想要报仇，可是他现在的身体状态根本不允许他有什么作为，仅仅只是往前走了几步，一股晕眩感袭来，维尔纳特便是眼前一黑，晕倒在地。
<br/>　　周围其他force-recon队员看着躺在湿漉漉地板上的维尔纳特，非但没有上去搀扶，眼中甚至还带着幸灾乐祸之色。
<br/>　　维尔纳特是海军陆战队中的明星，同样也是force-recon中的明星，早就使得维尔纳特自视甚高，在force-recon中人际关系并不算融洽。
<br/>　　如果维尔纳特不出事，那他就是明星，无人能撼动他的地位；可是一旦他像现在出了事，那些过去受到过他冷眼光的人恨不得再上去踩他一脚。
<br/>　　大量机动部队队员从博物馆大门口闯入，人数足有四五十人，齐凡听到从楼下传来的密集脚步声，说道：“走这边！”
<br/>　　行动前，齐凡已经对这个博物馆进行了研究，这个博物馆在这座城市中可以说是标志性建筑，要从晚上找到一些有关于它的资料很容易，齐凡对博物馆中所有的应急通道全都已经掌握清楚。
<br/>　　齐凡走到一扇卷帘门前，对着卷帘门的锁抬手就是一枪，把门锁打坏，对着孙涛使了个眼色。
