        “我也一直认为我不会恋爱，但是也没想到，自己会就这样轻易地陷入进去。”其实，傅夜沉至今都觉得自己对“苏漫雪”的感觉来得有些令他防不胜防。
<br/>
<br/>        那晚的回忆，至今令他记忆犹新。
<br/>
<br/>        他行踪败露后，歹徒对他穷追不舍，他被逼得走投无路，便纵身一跃。
<br/>
<br/>        他以为他那晚死定了，却发现自己跳到了她家阳台上，而她家阳台的玻璃门是虚掩着的。
<br/>
<br/>        当时，他推门准备进去的那一刻，就看到她伸出手来正巧挥开了玻璃门上的窗帘。
<br/>
<br/>        那时候的月光，洁白如雪，照在她脸上，犹如潺潺流水，在她脸上晃动着明媚的光芒。
<br/>
<br/>        其实，他见到她的第一眼，她便在他心里留下了深刻的印象。
<br/>
<br/>        混迹暗道这么多年，什么样的美女，他都见过，所以，“漫雪”吸引他的第一眼并不是来源于她的美貌，而是来自一种“岁月静好”时融合在她身上的恬静气息。
<br/>
<br/>        这样的气息，令他似曾相识。
<br/>
<br/>        或许，正是这种说不清理还乱的心灵直觉，才会让他在强吻她的那一刻，对她欲罢不能。
<br/>
<br/>        人的感情，微妙到无法用语言去形容。
<br/>
<br/>        傅夜沉沉浸在自己与“苏漫雪”初识的回忆里。
<br/>
<br/>        傅昼景看着大哥脸上洋溢着淡淡的幸福微光，会心一笑：“大哥，看样子，你是真的很爱那个准大嫂。”
<br/>
<br/>        只不过，他记忆里的那个准大嫂，似乎品德有点儿败坏门楣。
<br/>
<br/>        那次，准大嫂说要来滨江城玩，大哥没空去接准大嫂，所以派他去接的苏漫雪。
<br/>
<br/>        他接到苏漫雪时，苏漫雪还故意在他的面前卖弄风骚，袒胸露乳地想要勾引他。
<br/>
<br/>        这样一个看起来就像风尘世俗里的女子，是怎么入了大哥的眼和心的？
<br/>
<br/>        那苏漫雪无论样貌还是品行，都不及那个一直爱慕着大哥的李忆莎。
<br/>
<br/>        难道，真的就如大哥所说的，“救命之恩以身相许”？
<br/>
<br/>        或许，这就是爱情吧！
<br/>
<br/>        令人捉摸不透。
<br/>
<br/>        傅昼景和傅夜沉闲聊了一会儿，便抽身离开了傅氏集团总部大厦。
<br/>
<br/>        他手里还有一堆的项目单子，等着他去完成。
<br/>
<br/>        因为昨晚的事情，季思妍心里对傅昼景一直存在着芥蒂，她今天的所作所为，不再像前些天那样热情，而是实实在在地为了学习而工作。
<br/>
<br/>        她这样的心态，对傅昼景来说是最好不过的。
<br/>
<br/>        不然，前女友现在在自己手下打工，两个人抬头不见低头见，气氛着实尴尬。
<br/>
<br/>        以前每次开会，都是赵斌跟着，现在季思妍主动请缨，赵斌便给她这个新人机会。
<br/>
<br/>        傅昼景对季思妍做的会议纪要整理很满意，不得不说，季思妍在工作上确实很认真。
<br/>
<br/>        有些工作细节，就连工作了好几年的助理赵斌也没顾虑到，她却能想到。
<br/>
<br/>        其实，自从有了季思妍的协助，傅昼景在工作上的进展要快了许多。
<br/>
<br/>        一个项目，本来要半个月的样子才能完成，但是季思妍来了后，他可以提前一周搞定。
<br/>
<br/>        季思妍在他面前也是任劳任怨，从来不摆千金大小姐的架子。
<br/>
<br/>        这一个项目接着一个项目地做下去，傅昼景发现，原来自己也可以像普通员工一样拿到项目提成。
<br/>
<br/>        这些天，他仍旧早出晚归，没有多余的时间陪伴筱颖。
<br/>
<br/>        童筱颖同样也不闲着，每天早上五点钟就起来给傅昼景做早餐，等到傅昼景吃了早餐出门上班去了后，她又做完家务继续补个回笼觉，然后到了晚上近八点的样子，她才会出门动身前往会所驻唱。
<br/>
<br/>        在会所驻唱的这几天，每晚都还算顺利，有唐倩兮罩着，没有人故意找她的茬。
<br/>
<br/>        今天她特意提前了两个小时到了会所，正好是晚饭过后的时间里，来会所里娱乐消费的顾客就已经有很多了。
<br/>
<br/>        以前，她都没仔细看过这家会所的名字，现在仔细看的时候，才发现这个会所名叫“夜惑”。
<br/>
<br/>        来“夜惑”消费的人，大多数都是贪图享乐，沉迷于酒色的富人。
<br/>
<br/>        滨江城有个传闻，最美的女人在“夜惑”，最好的美食在“夜惑”，最香的美酒也在“夜惑”。

<br/>        今晚，唐倩兮休假。
<br/>
<br/>        童筱颖去唐倩兮的休息室找她时，发现她不在后，此时，从屋内的另一侧，突然走出来一个身材高挑，前凸后翘，披着大波浪卷发的红唇女人。
<br/>
<br/>        女人抬起做了白色美甲的手，撩开珠帘，从里边踏着高跟鞋，走到了童筱颖的跟前。
<br/>
<br/>        童筱颖脚上穿着的是平底鞋，却和这个女人差不多高。
<br/>
<br/>        “你就是小谢？！”女人打量了童筱颖一眼，甩手往童筱颖的身后走去，淡淡地说道，“你跟我来吧！”
<br/>
<br/>        小谢是童筱颖在这家会送里用的“艺名”，还是唐倩兮给她取的。
<br/>
<br/>        童筱颖转过身去，看着女人扭臀摆手，顿时起了一身鸡皮疙瘩。
<br/>
<br/>        “你——认识我？”童筱颖追上这个女人的步伐后，弱弱地问道。
<br/>
<br/>        女人对童筱颖翻了个白眼，嗤之以鼻地回答道：“小倩跟我经常提起你，若不是你会唱歌，以你这漂亮的姿色，我看你早就被费老板蛊惑去当出台小姐了。”
<br/>
<br/>        “什么是出台小姐？”童筱颖有点儿不解地反问。
<br/>
<br/>        女人再次对童筱颖翻了个白眼，冷嗤：“你这是把自己说得多纯洁啊！出台小姐，就是带出去开房的女人。还要我再细说吗？”
<br/>
<br/>        “谢谢，不用了。”童筱颖抿了抿嘴，感到有些尴尬，顿了顿后，她又接着问道，“请问，我该怎么称呼你？你这是要带我去哪里？”

<br/>        “我的艺名叫‘晚霞’，你叫我晚霞姐就行了。”晚霞妖娆地说道，然后领着童筱颖往舞池那边走去，“小倩说，你来咱们‘夜惑’也有些天了。但是对咱们‘夜惑’还不太了解。小倩让我今晚好好带你了解一下我们这里的规矩。另外，既然你是卖艺不卖身的，就记得带好你的假面眼镜！”
<br/>
<br/>        经晚霞这么一提，童筱颖连忙从裙兜里掏出那副白色羽绒的假面眼镜，戴在了自己脸上。
