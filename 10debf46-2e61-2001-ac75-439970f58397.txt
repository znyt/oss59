<br/>				<br/>　　    又过了两天。<br/>　　<br/>　　    这一天清晨，在听到凤九回来了的混元子匆匆的便赶到了天丹楼来，一进里面，便拉过冷华问着：“听说凤九那丫头回来了？在哪啊？老头找她有急事。”<br/>　　<br/>　　    “前辈，我主子正在炼丹房中炼丹，有什么急事都得等她出来了再说，要不您先到这边坐下歇会吧？先喝杯茶，主子一出来我便带您过去找她。”冷华温声笑着，将他带到一旁供客人歇息的桌边坐下。<br/>　　<br/>　　    听到这话的混元子皱了皱眉，坐下后道：“不是听说才回来没几天吗？怎么就这么拼的在炼丹了？我知道你们天丹楼最近的事情不少，她回来居然也能在那炼丹房呆得住？”<br/>　　<br/>　　    “主子回来后着手处理了不少事情，炼丹是昨天傍晚开始的，昨夜一整夜都没从炼丹房出来，我估计着，今天主子应该会出来了。”冷华说着，让人给混元子上茶。<br/>　　<br/>　　    “混元前辈，请喝茶。”白倾城端上茶水来，放在混元子面前的桌上，正要退下时，就听他的声音带着诧异的传来。<br/>　　<br/>　　    “你就是那个天阳宗的姓白的叫什么来着？”混元子敲了敲脑袋想了想，好半响也没想起那名字来。<br/>　　<br/>　　    “前辈，晚辈白倾城。”她垂眸恭敬的说着。<br/>　　<br/>　　    “哦，是叫白倾城啊？来来，抬起头来老头我瞧瞧。”混元子笑眯着眼睛说着。<br/>　　<br/>　　    闻言，白倾城略有迟疑，这时，便听冷华的声音传来。<br/>　　<br/>　　    “混元前辈的话没听到吗？把头抬起来给他老人家看看。”冷华的声音淡淡的，与平时的温和大不相同。<br/>　　<br/>　　    也是，知道这个白倾城曾经为难过他主子，还想算计他主子，他又岂会给他好脸色看呢！<br/>　　<br/>　　    “是。”白倾城的脸色微白，这才抬起头来。只是，此时只觉得屈辱又难堪。<br/>　　<br/>　　    想她原是堂堂天之骄女，不可一世，如今，却轮落到这个地步，被叫抬头就得抬头，这种屈辱感，也只有亲身经历的她才能懂得其中的难堪。<br/>　　<br/>　　    “呵呵呵，长得是挺不错的，倒也对得起倾城这名字，只不过，比起凤丫头还是差远了。”混元子端着茶水喝着，一边对着白倾城评头论足的说着。<br/>　　<br/>　　    她缓了缓神，以着平静的语气道：“倾城只是主子身边的婢子，自是无法与主子相提并论。”<br/>　　<br/>　　    “老夫跟你说吧！你也别放不下你白家大小姐和天阳宗天骄的架子，能跟在那凤丫头身边也是你的福气，只要是入了她眼中的人，她一向是不会亏待的，好好干吧！”<br/>　　<br/>　　    闻言，白倾城不由的看了混元子一眼，心头微暖，露出了一抹笑意：“是，谢前辈提点，晚辈一定不会让主子失望的。”<br/>　　<br/>　　    听到这话，一旁的冷华不由看了她一眼，见她神色不似作假，眸光不禁微闪，道：“你且退下吧！”<br/>　　<br/>　　    “是。”她行了一礼后，这才退了下去。<br/>　　<br/>　　    “冷华小子，你来来，坐下坐下，跟老头说说你家主子是怎么回来的？她这失踪了也有一年了，这一年都干什么去了？”<br/>　　<br/>
