　　就在刘哔有些不耐烦，想要用其他方法进去的时候，大门‘吱嘎’一声，却是自己缓缓打开了。
<br/>　　朱红色的大门露出一道缝隙，一个秃顶老头露出些许身影，看着刘哔，谨慎的问道：“你有什么事情吗？”
<br/>　　老头的声音让人有些难以描述，就如同老旧的风箱一般，破败刺耳，让人不由自主的起一身一批疙瘩，而他的长相，也算是对得起这无比难听的声音。
<br/>　　一身洗得发白的长衫，用那张满是老茧，褶褶皱皱如同风干橘子皮的手，拄着一根不知道何等材质的老旧拐杖，脸上布满皱纹，眼神阴翳。
<br/>　　令人印象最为深刻的，是秃顶老头的五官，像是闹着玩似得挤在一起，分不清哪是鼻子，哪是嘴巴，只有一双耳朵还算正常，但也只是相对而言。
<br/>　　说实话，第一眼看见这个秃顶老人的时候，刘哔心中着实吓了一跳，要不是对方开口说话，他还以为是哪个乱坟岗里面爬出来的僵尸呢。
<br/>　　“我来找任婷婷小姐，请问……”脸上堆起一丝笑容，刘哔随便找了一个借口，但是话还没有说完，就被秃顶老人给打断了。
<br/>　　像是极不待见刘哔一样，秃顶老人烦躁的挥了挥手，道：“小姐跟老爷出去了，你改个时间再来吧。”
<br/>　　说罢，也不等他回话，就伸手关上了大门，刘哔皱了一下眉头，心中隐隐觉得有些不太对劲。
<br/>　　大清早的，任发和任婷婷两人居然不在家里，况且这老头的反应也太不正常了。
<br/>　　虽然是不请自来，但他好歹也算是任府的客人，但是这个守门人的反应却像是巴不得他快点离开一样。
<br/>　　不行，一定要进去看一看，到底任家发生了什么事情。
<br/>　　心中犹豫了一下，刘哔下定决心，疾走两步，看左右无人，纵身翻上了高耸的围墙。
<br/>　　虽然太阳已经高悬，但清晨的薄雾还未完全散去，在晨雾的包裹下，整个任府都显得有些朦胧。
<br/>　　蹲在围墙上面，刘哔脸上的表情有些僵硬，之前上墙的时候，他不小心踩到了围墙上面的碎玻璃茬子。
<br/>　　真是该死，他应该提前想到的，刚过了两天安稳的日子，就让自己变得这么不谨慎了！
<br/>　　心中暗骂一声，刘哔慢慢将碎玻璃从自己的脚心拔出，一股鲜血瞬间流淌了下来，顺着玻璃落在了脚下的围墙上。
<br/>　　随手将沾染着血迹的玻璃揣进兜里，他也没贸然行动，而是仔细查探了一下周围的环境，确认安全之后，方才从围墙上跳了下来。
<br/>　　落地时发出了一丝轻微的响动，刘哔借助着四周的掩体，小心翼翼的向府邸深处前行着。
<br/>　　任府比他之前想象中的还要大很多，房屋不少，但都紧闭着屋门，只是让刘哔感觉诡异的是，他进来了这么久，都还没有见到一个人。
<br/>　　整个任府，都像是一座偌大的坟墓般，安静无比。
<br/>　　藏身在花圃后面，刘哔意外的发现，这花圃里面的鲜花却已经枯萎了大半了，一些枯掉的花瓣落在地面上，更增添了几分荒凉的感觉。
<br/>　　不经意的皱了一下眉头，他心中更加奇怪，这任府也算是镇上屈指一数的大家族，花圃枯萎了，按理来说应该会收拾一下才对。
<br/>　　况且，这任府的下人都跑到哪里去了，虽然现在还是早上，但也不应该是这般冷清的模样。
<br/>　　打起十二分的戒心，刘哔继续在庭院里乱转着，虽然古人府宅的构造都差不多少，但他却对此没有什么了解。
<br/>　　约莫一刻钟的功夫，才顺利的找到了正厅，因为不想暴露自己的踪迹，他的动作也是异常小心。
<br/>　　此时任府正厅的大门并没有关闭，向外大敞着，可惜一道屏风立在门口，将里面的场景遮挡的严严实实。
<br/>　　再次确认了一下自身周围的环境，刘哔犹豫了一下，闪身进入了正厅，没有了屏风的遮挡，大厅内的情景一览无余。
<br/>　　这是一个大概百多平的房间，家具奢华，四周的墙壁上还挂着一些字画，在大厅内部的红木桌子上，一杯还未喝完的茶水，正腾腾的冒着热气。
<br/>　　不好，有人，还没走！
<br/>　　心中悚然一惊，刘哔身影一晃，直接掠上了大厅的横梁，在任府的情况搞明白之前，他不想将自己暴露出来。
<br/>　　房梁上满是灰尘，刘哔将整个身体都贴在梁上，双眼如隼，紧紧注视着下面，五感此时都被他调动到了极致，甚至连门外风吹落叶所发出的‘沙沙’声都清晰可闻。
<br/>　　五分钟……十分钟……半小时……
<br/>　　整整四十多分钟的时间过去，下面都没有任何异样发生，刘哔心中犹豫，他开始怀疑，大厅喝茶那人早就已经离开了。
<br/>　　就在他等的有些烦躁时，情况却发生了一些变化，他之前在叩门时遇见的那个秃头老人，从门口迈了进来。
<br/>　　秃头老人走过了屏风，看四下无人，径直来到大厅的书柜旁，挪动了几本书的位置。
<br/>　　随后，书柜旁边的地板缓缓裂开，露出一个半人大小的洞口，秃头老人弓着身子，缓缓走了下去。
<br/>　　这一切动作都被藏身在房梁上的刘哔收入眼底，此时，连他心中也是无比惊讶。
<br/>　　他没想到在这个小小的大厅里还有机关，面对现在的状况，他一时间也不知道应该如何处理了。
<br/>　　暗道里面有什么，刘哔也不清楚，贸然跟下去的话，别说会打草惊蛇，就连他自身的安全都无法保证。
<br/>　　不过，对方却没给他考虑的时间，随着秃头老人下去，暗道缓缓关闭，原本被挪开的地板，又恢复成了原本的模样。
<br/>　　再三犹豫之后，刘哔放弃了进入的想法，依旧一动不动的趴在大厅横梁上，耐着性子继续等候着。
<br/>　　富贵险中求，就算是任府真的会有什么危险，凭借他增强过后的体质，也应该可以逃脱出去。
<br/>　　如果实在事不可为的话，他也不会继续坚持，但如今连危险都还没有看见，就此收手，他自然心有不甘。
