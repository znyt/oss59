　　人皇罐现在虽然残缺不全，但是一直以来都是楚风眠最为重要的一张底牌。
<br/>　　凭借人皇罐，楚风眠化解了无数次的危险。
<br/>　　虽然有极道仙兵碎片出世的消息，早就传遍了大罗天星域，但是楚风眠却是一直发现。
<br/>　　人皇罐碎片，会互相吸引。
<br/>　　现在楚风眠身躯之中的人皇罐突然被缓醒过来，给予楚风眠指引。
<br/>　　这也就代表着，根据这指引，楚风眠便是足以找到那一枚人皇罐碎片。
<br/>　　而这人皇罐，指引的方向……
<br/>　　楚风眠抬起头，看了过去，这方向正是楚风眠的正前方。
<br/>　　也就是如今楚风眠要一直向前去的地方。
<br/>　　“难道是那人皇罐碎片，就在那仙帝洞府之中？”
<br/>　　楚风眠心中惊疑一声，这方向的指引，也太过巧合了些。
<br/>　　应该没有那么巧的事情。
<br/>　　这一枚人皇罐碎片，真的藏在了仙帝洞府之中，楚风眠也不觉得有任何的奇怪。
<br/>　　毕竟楚风眠曾经得到了其中一枚人皇罐碎片，便是就在沧澜宫之中得到的。
<br/>　　沧澜宫是曾经沧澜仙尊留下的洞府，他的洞府之中就有着一枚人皇罐碎片。
<br/>　　这仙帝洞府的主人，也曾经得到了一枚人皇罐碎片，放入到了洞府之中，也极有可能。
<br/>　　“看来这仙帝洞府，是必须要闯一闯了！”
<br/>　　楚风眠目光闪烁，不止是因为那仙帝宝藏，还有那人皇罐碎片，也值得楚风眠冒险一闯。
<br/>　　这仙帝洞府位于赤红大陆的中心。
<br/>　　楚风眠跟燕皇的遁光在这赤红大陆穿梭，不一会，就在前方，一道耀眼的七彩霞光，直冲天际之上。
<br/>　　这七彩霞光，距离楚风眠如今足足有着两千里之远的距离，但是楚风眠依然是足以清晰的看得见。
<br/>　　甚至是足以感受到在这七彩霞光之中蕴含的滂湃灵力。
<br/>　　“笼罩两千里的霞光！”
<br/>　　楚风眠心中在看到这七彩霞光的一刻都不由的惊叹了一声。
<br/>　　在这赤红大陆之中所有开启的宝藏，都会随着霞光出现，霞光范围笼罩的越大，也就越是代表着其中蕴含的宝藏却是尊贵。
<br/>　　能够笼罩方圆百里级别七彩霞光的宝藏，都足以被称得上是大型宝藏了。
<br/>　　像是那狮霸武祖的洞府，其中开启之时显现而出的七彩霞光，便是足以笼罩方圆百里。
<br/>　　在方圆百里的武者，都可以清晰的看见那七彩霞光，眼前这一道霞光，居然足足两千里。
<br/>　　“不愧是仙帝洞府！”
<br/>　　两千里七彩霞光的笼罩，这宝藏纵然不是仙帝洞府，也相差不远了。
<br/>　　“我们靠过去。”
<br/>　　燕皇看了一眼，旋即向着七彩霞光的最中心飞了过去。
<br/>　　接近到了这七彩霞光之中，楚风眠便是感受到了无数强横的气势。
<br/>　　一股股的天地之力，冲天而起，这种压制不住的天地之力，便是九劫古帝的力量。
<br/>　　短短的刹那之间，楚风眠就发现了七八位九劫古帝，分布在这附近，这还是没有靠近最中心的区域。
<br/>　　至于七劫古帝，八劫古帝，甚至是真帝级别的武者，更是遍地都是，楚风眠一眼望去就有着数百人了。
<br/>　　这还只是很少的一部分，越是接近到这七彩霞光的中心，楚风眠便是感觉到了越来越多的强者。
<br/>　　“这一次到底是有着多少的强者来到了九域之中？”
<br/>　　楚风眠心中都是惊叹无比。
<br/>　　古帝，在大罗天星域之中虽然不算是难得一见，但是平日里，数位古帝一同出现，便就算是很少见的场面了。
<br/>　　可现在，楚风眠已经是感觉到了古帝，就有上百位了。
<br/>　　如果不是到来这里，楚风眠也不敢相信，居然是有着如此之多的古帝，都来到了九域之中。
<br/>　　同时楚风眠的心中也暗自庆幸，幸好是这些古帝到来的目标，是归墟天其中的宝藏，而没有前往其他九域的地方。
<br/>　　不然的话，凭借如此之多的古帝，只怕是让九域都会天翻地覆，纵然是九域的七大宗门，都承受不住如此之多古帝的冲击。
<br/>　　“大罗天星域之中的古帝，应该就在数百人左右，而且至少半数以上，都是属于四大宗门之中。”
<br/>　　“不过这一次九域现世，不止是大罗天星域的古帝，像是附近，阎罗星域，群龙星域，浮山星域，甚至是古老的净土星域之中的古帝，都来到了九域争夺宝藏。”
<br/>　　燕皇看出了楚风眠眼神之中的疑惑出声解释道。
<br/>　　净土星域的强者都前来了？
<br/>　　楚风眠记得，净土星域，是传说之中诸佛所在的星域，佛道，在曾经的九域之中也盛兴过一时。
<br/>　　当时在九域之中的一大宗门，净土宗，便是据说是从遥远的净土星域之中传递而来的。
<br/>　　不过后来净土宗也被毁灭，佛道也就是随即毁灭了，只有是在古籍之中楚风眠曾经读到过，关于佛道的事。
<br/>　　佛道无比神秘，在那个时代之中的记载都极少，但是在古籍之中，对佛道的记载都无比谨慎。
<br/>　　据说佛道擅闯蛊惑人心，在那个时代之中不知道多少强者，都被佛道强者度化，降伏佛道。
<br/>　　如果真的碰到来自净土星域的强者，楚风眠必须要小心行事才行。
<br/>　　楚风眠跟燕皇一路飞行，逐渐的来到了这七彩霞光的中心，有燕皇这样一位九劫古帝在身旁，也没有人会来找楚风眠二人的麻烦。
<br/>　　九劫古帝之间的拼杀，可是无比危险的，如果不是为了争夺宝藏，没有人愿意去跟一位九劫古帝拼杀。
<br/>　　楚风眠跟燕皇一路前进，逐渐的也看见了，在这七彩霞光的正下方，却是一座巨大无比的大门。
<br/>　　这一座佛寺，是百根白玉石柱所支撑，每一根白玉石柱都有着千丈之高，上面刻画这古老无比的禁制，这禁制的刻画，仿佛如同一枚枚经文一般。
<br/>　　佛寺？居然是一座佛寺？莫非是古老净土宗的一位强者？难道说净土宗之中，曾经诞生过一尊佛祖？
<br/>　　据说佛道的极致，便是佛祖，佛祖的实力足以跟仙帝平起平坐。
<br/>　　如果真的是一尊佛祖的寺庙，那其中的机缘宝藏，绝不会亚于仙帝洞府。
