　　闭关室内，萧逸盘膝而坐。
<br/>　　这里，是猎妖殿主殿的闭关室。
<br/>　　他的伤势，倒不算严重，只是之前不想有意外，击杀虎护法等人时强行凝聚了六色圆环，留了些许反噬。
<br/>　　他来，只是想静静地思索些事情。
<br/>　　从一月前开始，他重伤在湖底养伤，再到今日杀毒袍、十万邪修，吸收两条龙形煞气，借雷炼体等等。
<br/>　　事实上，他的修为、实力等等，几乎无半分提升。
<br/>　　那一月养伤，自不必多说，几乎一直在生死边缘与痛苦中消除着反噬与恢复伤势。
<br/>　　而今日，虽吸收了血煞、毒煞，两条庞大的龙形煞气。
<br/>　　但对他本身的修为而言，根本无半分影响。
<br/>　　整条巨大血煞，皆是被他体内的血丹所吸收。
<br/>　　而他的小世界和气泉之内，可是半点儿元力都没有增加，自然的，他的修为没有增加。
<br/>　　非要说实力的话，那便是自己气血一道上的手段，能借用这颗血丹发挥极强的威力了。
<br/>　　如血界四式。
<br/>　　这颗血丹，本就在以前吸收了大量的血气，惊人无比。
<br/>　　后又在剑帝碑外，吸收了天下剑修的血气，更为强大。
<br/>　　而今，加上这条孕育百万年的龙形血煞，其蕴含的力量以及威力，完全达到了相当可怕的层次。
<br/>　　若他现今将这颗血丹的力量完全激发，赖以施展血界四式的话，实力，绝对在传奇之上。
<br/>　　甚至于，爆发的实力，大大超过了之前脚踏血煞与毒袍一战之时。
<br/>　　不，甚至于，会比他想象的，还要高出数个层次。
<br/>　　但，如同以前一样，这颗血珠，有着影响武者心智、心神之效。
<br/>　　以这颗血丹现今层次，一旦完全爆发，根本不是他能压得住的。
<br/>　　之前与毒袍一战时，虽是脚踏血煞，且完全借用血煞的力量，方能将血界四式爆发惊人实力。
<br/>　　可那时，血煞之力在外，自己只是直接借用里头力量，故影响不大。
<br/>　　而现今，血煞之力，完全融入了自身体内血珠。
<br/>　　借用血丹力量，血珠会搅动自身血气，自身受其影响。
<br/>　　二者完全不一样。
<br/>　　再者，现在这颗血丹蕴含的力量，可比龙形血煞庞大多了。
<br/>　　说白了，萧逸还是像以前一样，无法轻易借用这颗血丹的力量。
<br/>　　甚至于，这颗血丹变得更强下，反倒对他的限制更大了。
<br/>　　萧逸也因此，更无法轻用体内这颗血丹。
<br/>　　至于毒丹，同样如此。
<br/>　　毒丹，吸收了整条龙形毒煞的力量，自也是威力惊人。
<br/>　　对于毒丹，萧逸倒没有对血丹那般忌惮。
<br/>　　毕竟，血丹的血气力量，会影响心神，乃至让人发狂疯魔，大失常性。
<br/>　　可毒丹却没有任何影响。
<br/>　　且，萧逸也想因此验证万毒宝典之内的一项项疯狂记载和想法。
<br/>　　不过，想法也终归是想法，毕竟未经验证，萧逸也不敢轻用。
<br/>　　说白了，毒丹在体内，萧逸并不担心，但是否使用，他还是要斟酌，也需要之后的验证。
<br/>　　故，即便吸收了这两条庞大无比的血煞、毒煞。
<br/>　　可萧逸本身的修为和实力，却无半分提升。
<br/>　　至于借雷炼体，这不过是当日巨木封印内，萧逸想着反正也要吸收两条煞气的力量，且抵抗雷煞的雷电，这才借以炼体罢了。
<br/>　　简单来说，肉体力量的提升，不过如同锦上添花。
<br/>　　此刻的肉体力量，已然达到绝世层次，萧逸感知过，大概等同正常绝世强者9992道左右的层次吧。
<br/>　　而这，仅仅是正常的肉体力量罢了。
<br/>　　萧逸以前的肉体力量，之所以强，之所以屡屡能赋予他莫大战力，是因为他修的乃是修罗战体。
<br/>　　修罗战体赋予的肉体力量，才是真正的强大莫测。
<br/>　　而现在的借雷炼体，虽让他肉体力量达到绝世，却只是等同普通绝世体修的肉体力量。
<br/>　　简单来说，他现在的肉体力量，与那些修炼诸如雷罡战体等等体修功法达到绝世层次的体修，没什么两样。
<br/>　　而他的修罗战体，终归还停留在圣皇境七重的层次。
<br/>　　修罗战体的修炼与提升，唯一的办法，只有吸收妖兽精血，化作修罗之力，淬炼自身。
<br/>　　那才是真正的强悍。
<br/>　　如果说，体修武者，是碾压一切职业的存在。
<br/>　　那么，修罗战体所赋予的肉体力量，则是碾压一切体修。
<br/>　　二者，自然不可同日而语。
<br/>　　当日了，日后，修罗战体也修炼到绝世境界时，那时，修罗战体赋予的肉体增幅，还有现今的肉体增幅，就会成为双重增幅。
<br/>　　届时，他的肉体力量，将更加强悍。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　说起来，他虽得了血煞、毒煞，可本身，却几乎没半点实质的提升。
<br/>　　倒是那条风煞。
<br/>　　如今，整条风煞，都在风圣壶内。
<br/>　　这么一条龙形煞气，而且还是孕育百万年，里头力量，庞大得惊人。
<br/>　　无论是雾妖，还是那把千风风煞而成的镰刀，假以时日，都会得到极大的成长和提升。
<br/>　　若是日后整条风煞被吸收殆尽，雾妖会成长到何等地步，萧逸也期待万分。
<br/>　　现今的雾妖，已是武道大能实力。
<br/>　　他日，必将是他身边一大助力。
<br/>　　“呵，算了。”萧逸不禁笑笑。
<br/>　　虽说血煞、毒煞，没给他什么实质性的提升。
<br/>　　但雾妖能吸收那条风煞，已是意外收获。
<br/>　　再者，他本来的目的，只是要击杀毒袍罢了。
<br/>　　想起毒袍，萧逸皱了皱眉，手中光芒一闪，取出一个乾坤戒。
<br/>　　这是毒袍的乾坤戒。
<br/>　　萧逸感知了一眼，从中取出一根破裂的白骨手杖。
<br/>　　正是之前那根毒袍赖以凝聚太荒之火的手杖。
<br/>　　“嗯？”萧逸看了几眼，细细触摸手杖，忽然脸色一变。
<br/>　　“这是真正的荒影蛛之骨？”
<br/>　　萧逸陡然一惊。
<br/>　　……
<br/>　　同一时间。
<br/>　　六龙城，那片早已在大战之下狼藉一片，破烂不堪的废墟之地内。
<br/>　　毒袍冰冷的尸体中，蓦地闪出一道诡影。
<br/>　　诡影，是一细小之物，却速度极快。
<br/>　　正是鬼螳螂。
<br/>　　鬼螳螂，攀爬至毒袍额间，重重地咬了一口。
<br/>　　毒袍原本冰冷的尸体，蓦地渐渐温热。
<br/>　　干枯的尸体，忽然变得充盈。
<br/>　　紧闭的眼眸，猛地一睁。
<br/>　　“易霄。”毒袍阴蛰的眼眸中，冰冷异常。
<br/>　　下一秒，原地一阵黑雾涌现。
<br/>　　黑雾，疾速而离，无声无息，转瞬间已消失在远方。
<br/>　　原地，毒袍的‘尸体’，以及鬼螳螂，已消失无踪。
