　　“武道真意？”萧逸皱了皱眉。
<br/>　　指尖，陡然一引。
<br/>　　身旁那条火焰长河，瞬间被他牵引而至。
<br/>　　到了他现今层次，事实上已知晓武道真意是何物。
<br/>　　所谓武道真意，便是一类武道的精粹。
<br/>　　起码需要一千条同类型武道的融合之下，方能凝聚而出。
<br/>　　拿现在这条‘火焰长河’来说，它便是萧逸汇聚了一千条完整的火类武道而成。
<br/>　　当然了，其中几乎都是二流火焰的武道。
<br/>　　“千火真意。”萧逸看着这条武道真意，自语一声。
<br/>　　或者可以更简单来说，当领悟出千火真意时，便代表着这一类火焰，武者已融会贯通，悉数参悟掌控透彻。
<br/>　　千火真意之内，便蕴含了这一类火焰的所有武道知识。
<br/>　　这就是武道真意。
<br/>　　武道中的真意。
<br/>　　武道真意，也有强弱之分。
<br/>　　拿‘千火真意’来说，不过是一种简单的火类真意罢了。
<br/>　　或者说，以剑道的角度去视之，会更加直接。
<br/>　　拿剑道来说，有无数种剑道。
<br/>　　但剑道又可以分为许多类，风之剑道、冰之剑道、无属性的纯粹剑道、植物类的剑道，如万花剑道等等。
<br/>　　比如说风之剑道，风之剑道中，又可细分为许多不同的剑道。
<br/>　　如其中最普通的疾风剑道，还有将风属性发挥至极致的寂空剑道。
<br/>　　而在普通和极致二者间，还会有无数种，诸如清风剑道、风暴剑道、柔风剑道、回风剑道…等等。
<br/>　　当然，这其中又分为了快剑、慢剑、狠辣之剑、延绵之剑等等。
<br/>　　如果剑修，将风属性一类的所有剑道，悉数参悟，并且掌控，便会衍生出武道真意。
<br/>　　这种武道真意，便是‘风剑真意’。
<br/>　　而‘风剑真意’，毫无疑问乃是极强的武道真意，绝对远远强于‘千火真意’。
<br/>　　而要领悟‘风剑真意’，极难，其中蕴含的，绝不止一千条剑道。
<br/>　　这甚至是绝世强者这个层次都不一定够格领悟出的武道真意。
<br/>　　因为没人知道风剑真意到底蕴含多少种风属性剑道真意。
<br/>　　也更无法知道自己差哪几种。
<br/>　　这甚至需要武者极长时间的去闭关、参悟武道，等等。
<br/>　　这便是武道真意，武道中的真意，一条真意，蕴含无数条武道的精粹与奥妙，蕴含了天地间的至理知识。
<br/>　　这也是为什么武者天极境之后的修炼，走的是融合武道。
<br/>　　因为武者便是需要一条条完整武道的融合，方能悟出天地武道真意。
<br/>　　而另一点，剑道，乃是所有武道中最难领悟出真意的一道。
<br/>　　剑道与剑道之间，几乎难以融合。
<br/>　　拿萧逸的霸道剑道、寒冰剑道来说。
<br/>　　二者同为顶尖剑道，同为剑道，但一为寒冰、一为霸道，二者根本没有契合点，也就是说根本无法融合。
<br/>　　反倒是火焰一道，火焰本身就有融合性，各种火焰武道之间，要融合不难。
<br/>　　故其他种类的武道真意，相对容易悟出。
<br/>　　而剑道，则极难。
<br/>　　这也是剑修，被誉为最擅长战斗的武者的原因。
<br/>　　说回武道真意，同样是剑类。
<br/>　　风属性一道，如疾风剑道，若是与冰雪属性的剑道融合，便有了风雪剑道。
<br/>　　而风雪剑道，既属于风属性剑道，又属冰属性剑道。
<br/>　　或者更简单来说，天地武道无数，彼此皆可融合，便看自己能否悟透悟彻，将其融合了。
<br/>　　不同的融合，便有了不同的一道。
<br/>　　无数种融合下，便有了无数种可能。
<br/>　　自然的，领悟而出的‘武道真意’，也会有无数种。
<br/>　　天地规则，各有不同，却又互有联系，共同组成了这一天地轮回，世间万物。
<br/>　　“呼。”萧逸不禁长长地呼出一口气。
<br/>　　他到达圣皇境，除了修为上的突破，实力上的增加，更有着天地至理的明悟。
<br/>　　原本以为圣皇境，不说实力，单说境界，已经是一个很不错的境界了。
<br/>　　但真正达到后，方知这天地奥妙，远超自己想象。
<br/>　　他甚至感觉，圣皇境，领悟出武道真意后，不过是一个开端。
<br/>　　难怪武道之路，如此难走。
<br/>　　亦难怪，炎龙大陆千千万年来，无人能达武神之位。
<br/>　　要达武神，便要掌控天地武道，明悟世间规则，一念之间，堪破天地轮回。
<br/>　　这等层次，非人所能想象。
<br/>　　“唉。”萧逸不禁重重地叹了口气。
<br/>　　难怪那些个武道大能，瓶颈困个数百年都是家常便饭。
<br/>　　难怪那些个绝世强者，千年难以突破些许修为。
<br/>　　难怪以星幻前辈那等人物，穷极一生，都无法突破绝世强者的下一境界。
<br/>　　即便突破了，距离武神之位，想来仍旧有极长极长的路程。
<br/>　　萧逸甚至觉得，那可能还会是另外一个开端。
<br/>　　真正的尽头，只有掌控所有天地武道。
<br/>　　嘭…
<br/>　　忽然，身旁一道火焰忽然凝聚，随后爆发。
<br/>　　“嗯？”萧逸脸色一变。
<br/>　　还未反应过来，已被一道紫炎轰得灰头土脸。
<br/>　　“嘶。”萧逸倒吸一口凉气，当即脸色大变。
<br/>　　“体内火焰不稳，甚至溢出体外，自动爆炸，伤及本身？”
<br/>　　一个武者，连自身力量都控制不住。
<br/>　　而且还是他赖以自豪的控火手段，赖以自豪的紫晶灵炎，竟是差点伤了他？
<br/>　　“差点被心魔所困。”萧逸心头一冷，脸色陡然一凛。
<br/>　　不错，萧逸刚才的叹气，还有思索，差点让自己产生了心魔，而且程度还不低。
<br/>　　否则他不会忽然体内火焰紊乱。
<br/>　　若他刚才不是及时反应过来，恐怕现今已身死道消，体内的火焰足以将他自己染成灰烬。
<br/>　　这是他以往从来没遇到过的事。
<br/>　　刚才，念及这武道之路的无比困难，竟是差点让他心魔噬身。
<br/>　　萧逸摇了摇头，脸色恢复了正常，眼眸也恢复了清明。
<br/>　　他深知，这就是武道之路的艰险，一个不慎，便会陨落在武道之路上。
<br/>　　这甚至无需敌人杀他，自己便会心神泯灭。
<br/>　　这也是武者须得时时谨慎，稳固道心的原因。
<br/>　　心魔，绝对是武者之路上的最大阻挠和危险。
<br/>　　“武道之路虽难，但我能做的，便是足够坚强。”
<br/>　　“一路走来，我挺到了现在，依依未带回，易老未救回，现今便想让我知难而退？”
<br/>　　“天地武道，你是不是太少看我萧逸了？”
<br/>　　一抹浓郁到极点的坚毅，陡然涌上萧逸眼眸之内。
<br/>　　那双明亮的眸子，仿佛看透了山洞之内的山石之壁，直视天际。
<br/>　　人需仰天，人需敬天。
<br/>　　可这双眸子，却带着惊人的不屈与倔强。
<br/>　　天地，折不了他，服不了他。
<br/>　　反倒是他的眸子，仿佛看透了这天地至理。
<br/>　　山洞之外，遥遥天际。
<br/>　　不知何时起，虚空崩溃，空间碎裂。
<br/>　　一道道无形的漩涡，横亘天地。
<br/>　　几乎是一瞬间，大陆之上，各处古老之地内，同时有了反应。
<br/>　　修罗地域，修罗总殿之内，修罗总殿主，猛地睁开了眼睛，直视天际。
<br/>　　万炎地域，炎殿总殿之内，炎殿总殿主，猛地睁开了眼睛，直视天际。
<br/>　　…
<br/>　　各处古老之地，同时有了躁动。
<br/>　　风刹地域，风刹总殿之内，风刹总殿主，双眼一眯，直视高空。
<br/>　　“那位冕下，果然已重现大陆。”
<br/>　　“承风，查。”
