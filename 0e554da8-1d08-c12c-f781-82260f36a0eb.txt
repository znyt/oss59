　　猎王神色认真起来：“怎么说？”
<br/>　　工作人员将档案调出来，道：“B12的任务也不知道是谁发布的，内容为在我们总部附近一个村落里，寻找发布者走丢的女儿。”
<br/>　　猎王的眉头再次皱起：“村落？我们总部设立在死火山内部，周围都是山林，哪来的村落？”
<br/>　　江凌轩也开口道：“也就是说这个任务，可能完成不了？或者说应该是，时间要很长？”
<br/>　　猎王眯起眼睛，想了一会儿后下令道：“关注王之暴君的动向，一旦出现在总部附近，立即向我汇报！”
<br/>　　“是！”
<br/>　　江凌轩有些惊讶，靠近猎王小声问道：“为何对王之暴君这么防备？”
<br/>　　猎王看了他一眼，道：“我们怀疑此人不是人类。”
<br/>　　江凌轩瞳孔剧烈收缩，乍然听闻这一消息后震惊到失声，不是人类？
<br/>　　这位在末世纪元之初就加入到猎者联盟的猎者，竟然不是人类？
<br/>　　王之暴君的身份早在猎王与楚涵某次会面的时候，就已经被私下曝光，只不过处于各方面的考虑，并没有广而告之，但却不妨碍猎王以猎者联盟创始人的身份，在各部门对王之暴君的全面监控。
<br/>　　异种王的另一个身份竟然是猎者，这本身就不可思议！
<br/>　　“老大。”忽的又一人进入屋内，冲着猎王道：“有一项S级任务，制定要您。”
<br/>　　猎王再次皱眉：“内容？发布者身份？”
<br/>　　“隐藏了身份。”那人回答道：“内容则是和南方的尸潮战事有关，要您支援南方的大战，在南部联盟军抵达战区之前，暂时守护段氏基地。”
<br/>　　江凌轩分析道：“这应该是某位人类阵营中的大人物发布的任务，南方的战事我也有所耳闻，据说第一场大尸潮战役打的相当激烈，狼牙战团正处于休战调整中，南部联盟军还在路上，如果第二场尸潮爆发，段氏基地就会有危险，这时候请猎者联盟的高手前去暂时顶替一下，也无口厚非。”
<br/>　　“是嘛。”猎王的声音有些诡异的平静，而后双眼眯了眯：“但我怎么觉得，有些巧合的过分呢？”
<br/>　　江凌轩有些没明白：“您的意思是……有人故意发布任务，调虎离山？”
<br/>　　几名工作人员也都看了过来，纷纷加入猜测讨论。
<br/>　　“可是把猎王调走，目的是什么呢？”
<br/>　　“猎王离开总部，我们这里也还有各种防护机关，先不说一般新人类根本破不了，再说哪怕神秘家族人来了，也有何培源长老坐镇，谁都攻不破这里。”
<br/>　　“不，有三个人不在此列。”猎王眼神冰冷。
<br/>　　“哪三个？”江凌轩立即问道。
<br/>　　猎王沉静回答：“王之暴君，帝之罗刹，美阎罗。”
<br/>　　“这三人都是我们联盟内的猎者。”一名工作人员不解的开口道：“没有理由冲进来搞破坏啊？况且以这三人的猎者身份，他们如果想，进入总部根本不需要这么麻烦。”
<br/>　　“高级猎者，是有特权的。”
<br/>　　这些工作人员不能理解，但知道一些辛秘的江凌轩，则是目露担忧的看向猎王：“这个S级任务，还是不要接了吧？”
<br/>　　“不。”哪知猎王却否定，道：“我接，明日就启程去段氏基地。”
<br/>　　江凌轩瞪大眼睛，不能理解猎王这一举动的目的，如果这一S级任务真的有诈，不就是故意给对方机会么？
<br/>　　——————
<br/>　　第二天，猎者联盟的最高级别猎者，猎王亲自出动前往南方执行任务的消息，一不小心走漏了风声，在四处传开。
<br/>　　很多听闻这一消息的猎者纷纷启程，将目的地设在了段氏基地，原因很简单，那就是他们万分的想知道猎者联盟中名气最大，身价最高，也是权力最大的猎王，究竟长什么样。
<br/>　　猎者联盟发展至今，已经聚集了相当多的一批猎者，但只有高级猎者才能进入总部，大部分的普通猎者都还只在边缘地带打转。
<br/>　　这一组织的核心地，只对高战力者开放！
<br/>　　而当今好不容易有这么一次机会，等到猎王亲自执行任务，任务内容还曝光了出来。
<br/>　　那些对猎者联盟总部好奇，对这一组织的创始人更好奇的普通猎者，怎么可能放过这次机会？
<br/>　　于是在猎王的刻意走漏消息中，大批的猎者就这么莫名前往南方，一不小心就让南方一整条分界线上的战区，新人类数量猛增，对抗丧尸群的战力也增强。
<br/>　　这一计中计，将大批不想归顺任何组织的猎者都坑了进来，搞出这一大场面的猎王自己却一点愧疚心里都没有。
<br/>　　猎者联盟的确打着他的名号创立，但唯有猎王心里清楚，出功出力的其实都是楚涵，包括当下在总部将管理系统运转起来的那些人，大多数也都来自狼牙。
<br/>　　他们牺牲了自己的前途，改名换姓，常年久月的待在猎者联盟内，就是为了将这一组织扩大再扩大。
<br/>　　因为猎者联盟，是如有万一后，狼牙和楚涵的最后退路！
<br/>　　猎王在将消息曝光后，也确实出现在段氏基地，表达了来意之后，得到了段氏基地很多高层的感激，猎王将要支援南部联盟军对抗尸潮的消息，也在这之后不仅在猎者之中流传，还蔓延到了各处地方。
<br/>　　同样得到这一消息的，还有白家，以及异种王。
<br/>　　在猎王前脚离开前往段氏基地的时候，异种王就后脚踏进了猎者联盟总部。
<br/>　　王之暴君的忽然出现，让总部内的很多猎者都异常激动，很多工作人员也纷纷前来迎接。
<br/>　　这位大佬在猎者圈中的名气，那可是高居不下，只不过太神秘也太高冷，导致整个猎者联盟内都没人对他有所了解。
<br/>　　江凌轩也在这些人之内，他远远的站在一边，看着眼前这些兴奋的人将王之暴君围起来，心头闪过猎王的那句话，而从提防之心更重。
<br/>　　太巧！
<br/>　　猎王刚走，王之暴君就来了！
<br/>　　说这其中没有阴谋，鬼才信！
