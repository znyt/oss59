        听到她这三连问，傅夜沉有些哭笑不得：“你怎么还在在意苏漫雪的事情？”

<br/>        “她怀过你的孩子，还有医院的手术单，是你签的字！”童筱颖一边控诉着，一边推搡着压在自己身上的这个男人。
<br/>
<br/>        傅夜沉也不得不先从这小女人身上抽身离开，有些不耐烦地说道：“我真听不懂你在胡说八道些什么。”
<br/>
<br/>        童筱颖见傅夜沉还嘴硬，一把推开他后，从床头柜里拿出平板电脑，将那封电子邮件打开，然后递到了傅夜沉的眼皮子底下。
<br/>
<br/>        “我有凭有据，是你敢做不敢当！还说我胡说八道！”童筱颖没好气地哭诉。
<br/>
<br/>        傅夜沉坐起身来，接过平板电脑看了看里面的内容，顿时忍俊不禁起来。
<br/>
<br/>        童筱颖见他在笑，更加恼火，气得一脚朝这坏男人踢去。
<br/>
<br/>        傅夜沉眼疾手快地抓住童筱颖的脚腕，另一只手则托住她的后背，将她整个身体往自己怀里揽。
<br/>
<br/>        “嘶”地一声，某处被坐痛了后，让童筱颖柳眉微蹙。
<br/>
<br/>        她双手搭在傅夜沉的肩头上，自己倒是脖子和肩膀因为某处痛得缩成了一团。
<br/>
<br/>        “上面的字迹倒是模仿得挺像，但毕竟不是我本人的亲笔签名，压根就做不能呈堂证供。”傅夜沉会心一笑，倾身吻了吻童筱颖嘟起来的粉唇。
<br/>
<br/>        童筱颖半信半疑地睨着傅夜沉，抿着嘴唇不吭声。
<br/>
<br/>        傅夜沉接着说道：“这种黑锅，我可不背。宝贝，你请的那私人侦探也太不专业了。我手下倒是有一支非常专业的侦探队伍，要不要我介绍给你？”
<br/>
<br/>        “大哥！”童筱颖差点气结，不禁恼羞成怒，“你出轨了，你还有理了，是不是？”
<br/>
<br/>        “我发誓，我傅夜沉除了童筱颖一个女人以外，就没碰过其他女人！”傅夜沉举起手来，扣起三指，一本正经地说道。
<br/>
<br/>        童筱颖看着傅夜沉一脸严肃的样子，突然间就心软了，好像又相信了他的话，但是心里却又莫名其妙地感到很慌。
<br/>
<br/>        “那我梦里那个两岁大的小男孩是怎么回事？我脱光了衣服躺床上等你，你去跑去了另外一个女人那儿，又是怎么一回事？还有，我总是梦见你跟另一个和我长得一模一样的女人在亲热，又是怎么一回事？大哥，你跟我说实话，好不好？”童筱颖鼻子一酸，委屈地滔滔不绝起来。
<br/>
<br/>        傅夜沉深吸了口气，搂着童筱颖的身子，将她按倒在大床上。
<br/>
<br/>        “筱颖，乖，别折磨我了。我忍不住了。”
<br/>
<br/>        语毕，他俯身低头，噙住了这小女人那喋喋不休的小嘴。
<br/>
<br/>        童筱颖刚开始半推半就，最后还是从了他。
<br/>
<br/>        她突然觉得自己太没骨气了，身体上对他的反应也表现得太过诚实。
<br/>
<br/>        本来还想继续跟他闹脾气的，结果闹到最后一点脾气也没有了。
<br/>
<br/>        “大哥，我想知道真相。”问这句话的时候，童筱颖是心平气和的。
<br/>
<br/>        傅夜沉见她执着于此，不得不跟她坦白以前所发生过的那些事情。
<br/>
<br/>        他最不愿意提起的阿景，也跟她提起来的。
<br/>
<br/>        “阿景，应该算得上是你真正的初恋吧！你以前，很爱他。”说这话时，傅夜沉的心是痛的。
<br/>
<br/>        童筱颖这才明白，那天傅昼景对她所说的那些话是什么意思。
<br/>
<br/>        “筱颖，不要再爱上阿景，好吗？”他很卑微地恳求。
<br/>
<br/>        童筱颖回过神来后，对上了傅夜沉那双阴郁的黑眸。
<br/>
<br/>        她没有说话，只是主动向着傅夜沉的薄唇吻了上去。

<br/>        翌日，日上三竿。
<br/>
<br/>        傅夜沉是被窗台上雀跃的几只鸟儿的啄窗声给吵醒。

<br/>        他掀开薄被，蹑手蹑脚地到浴室里去冲了个澡，洗走了身上的疲倦后，精神回笼，总算是神清气爽。

<br/>        走出浴室后，傅夜沉轻轻地回到了床边坐下，床上的小女人还在沉睡中，她怀里抱着枕头。
<br/>
<br/>        童筱颖的睡相算是再次诱发了他傅夜沉的欲望，他抬起手来。

<br/>        被大哥折腾了一夜，她到现在仍旧觉得有些精疲力尽。

<br/>        傅夜沉随之在童筱颖的身侧躺下，找准了时机，便凑到她的耳畔，问道：“我听到你的肚子在叫。”

<br/>        他的长臂从她的肩头圈过，大手将她整个身子揽入了怀中。
