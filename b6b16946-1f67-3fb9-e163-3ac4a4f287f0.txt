　　不知何时？盘古终于从呆滞的目光之中清醒过来，看着眼前破败的天地，盘古斧不知何时已经重新分散开来，太极阴阳图在镇压暴虐的地水火风，混沌钟在顶住下降的天，盘古幡静静地飘在盘古面前。
<br/>　　“何为不朽？何为不朽？何为不朽？”盘古问道三声，三声之后，一片寂静，不等回应，盘古就大笑道：
<br/>　　“哈哈哈！盘古真身，顶天立地！”
<br/>　　一刹那，亿万丈青色不朽光芒闪烁，盘古化身亿万丈与天地苍穹齐高，他这里，不朽境的肉身力量，爆发出强烈的不朽力量，双手顶天，双脚踏地，目视前方，全身散发着一种坚忍不拔，永不屈服，与天斗，与地斗，与众生斗，舍我其谁的意志。
<br/>　　“这，就是不朽！！！”
<br/>　　“今，盘古氏，愿身化万物，重塑乾坤，造化世界！！！”
<br/>　　“以吾不朽真身，不朽之力，赋予此方天地，长存混沌！”
<br/>　　“吾之躯，当化万物，天地玄黄，宇宙洪荒，乃吾之躯体！”
<br/>　　“洪荒不灭，吾即不朽！”
<br/>　　……
<br/>　　随着盘古这一道道声音落下，盘古真身，闪烁着纯青色不朽之光，璀璨无穷，覆盖着整个新开辟出来的洪荒天地，融入到整个天地本源之中，这个世界本就是他开辟出来的，这很容易做到。
<br/>　　接着，天地边缘，渐渐出现了一层青色的光罩，这光罩，挡住了混沌的冲击，却又在吸收着混沌之气，增强光罩的同时，也化为天地灵气反馈到洪荒天地。这就是不朽之力！
<br/>　　天地不再闭合，就这样，天日高一丈，地日厚一丈，十二万五千六百年过去了，盘古也油尽灯枯了，天无限高，地无限厚，洪荒世界外，不朽青光也彻底凝练了。顿时，青光不再，化为一道无色之光，守护洪荒！
<br/>　　“盘古大千世界，洪荒天地，成！”盘古笑了，他笑得很开心，这笑容中没有痛苦，没有放弃，更没有后悔，但却有苦涩，有回忆，有悲哀，有执着，还隐隐含有期待……
<br/>　　至此，一场葬送了三千魔神的混沌劫，彻底结束了。
<br/>　　三千魔神是失败者，却又是成功者，众魔神成功阻止了盘古证道不朽，却又都死在盘古斧下；盘古成功的开辟出新天地，却没有成功证道不朽。
<br/>　　在这一量劫之中，成功者，唯有大道，当然，还有全新的洪荒天道。至于混沌天道，那是赔了夫人又折兵！
<br/>　　……
<br/>　　“当——！”
<br/>　　“当……！”
<br/>　　此刻，天地间，混沌开天斧所化的混沌钟却在自动敲响，像是悲鸣，像是在诉说……
<br/>　　这时，盘古身体内飞出仅剩的十二滴青色的精血及三道元神清气，闪耀着青色的璀璨不朽之光，散发着古老而沧桑的气势，惊天动地。这就是以后的十二祖巫与三清（太清，玉清与上清）。
<br/>　　下一瞬，盘古真身轰然解体，演化天地万物。他呼出的气，化成风云，声音化为雷霆，左眼化为太阳，右眼化为月亮，四肢五体化为四极五岳，血液化为江河，经脉化为道路，肌肉化为田土，头发化为星辰，皮肤化为草木，齿骨化为金石，骨髓化为珠玉，汗流化为雨泽……
<br/>　　直到此刻，盘古大千世界，洪荒天地，彻底成型，苍穹大地，无限广阔！
<br/>　　与此同时，那神秘的天道空间之中，那些已经破碎的规则细线及法则链条在此刻却神奇的连接了起来，而且，还多出许多锁链，它们如同三千大道法则锁链一般，散发出一道道耀眼的光芒，向天地间蔓延而出，锁住整个天地。
<br/>　　天道本源光球也跳动起来，它变得更加璀璨，更加惊人了，更有一股惊人的意志诞生了，同时吸收紫气的速度也更快了。
<br/>　　但，却还有两个角落里的法则与规则，却是没有被连接上。在那里，一处散发着惊人的轮回道韵，而另一处却是一股百折不挠，坚忍不拔，煌煌人威，众生如龙的道韵！
<br/>　　……
<br/>　　就在此时，一股玄妙的韵律在空中回荡！苍穹之上，青天白日，风云倒卷，轰隆隆之间，顿时，风云变色，乾坤倒转，星移斗转，天空响起九道悲鸣之声。同时，一道亿万丈之长的玄黄光芒，照亮了整个洪荒天地，天空凝聚无数太初紫气，地上涌现无穷玄黄之气，天地弥漫祥瑞浮云，使得天地每一角，都震动起来！
<br/>　　一片玄黄在虚空汇聚着，散发出无量霞光，玄之又玄！古开天是劫，但同时也是功。对于大道来说，开天就意味着破灭混沌，重塑天地，自然是要降劫，而对于洪荒天地而言，开天利于天地，即为功，自然要降下无穷功德！而且还是最为特殊的天地玄黄功德。
<br/>　　玄黄功德为开天功德，是天地间最初功德，可不是后世的天道功德、先天功德、后天功德所能比的。
<br/>　　此刻，天空的玄黄功德骤然炸开，分成数份，各自降临洪荒各处。其中三成分别落在三道元神清气之上，三成落在十二滴青色精血上。
<br/>　　一成融合无穷玄黄之气化作一尊三十三层之高的金色小塔，散发着万法不侵的光辉，镇压无穷气运，为后天第一防御功德至宝，天地玄黄玲珑宝塔！
<br/>　　一成融合无尽太初紫气化作一柄紫金色小尺，闪动着天地奥妙之力，杀人不沾因果煞气，为后天第一攻击功德至宝，天地玄黄量天尺，也称鸿蒙量天尺！
<br/>　　两件至宝虽为后天生成，但含有功德与天地之气，无论玄黄之气还是太初紫气，都为天地之初的祖气，蕴含无量造化，不可多得。一攻一防，犹如阴阳，一动一静。
<br/>　　最后剩下两成洒向洪荒天地，为新生的天地创造无穷造化！
<br/>　　就在此时，虚空之中不知何时出现了一道谁也看不见的白色细线，这是因果之线中的因道细线。在这道因道细线出现的同时，三清之中的一道清气，同时也出现一道黑丝细线。在黑色细线的吸引之下，白色细线仿佛找到了家，顿时化作一道流光飞向黑色细线。两道细线相遇，顿时化作一道黑白双色线，为因果线。之后，迅速没入那道清气之中。
<br/>　　这时，“嗖！”的一声，天地玄黄玲珑宝塔飞入第一道清气，也就是太清气上。而另一件天地玄黄量天尺却在某种气机的牵引之下想要飞走，但此刻三清气之中的玉清气以迅雷不及掩耳之势在它飞走之前卷走它，之后，三道元神清气、十二祖巫分别向不同的方向飞走。
<br/>　　混沌开天斧所化的三大先天至宝也在冥冥之中的牵引之下，飞向洪荒各处。
<br/>　　一些先天灵物也各自孕育，天地开始发展。
<br/>　　混沌的时代结束，一个全新的时代正式拉开了序幕！
