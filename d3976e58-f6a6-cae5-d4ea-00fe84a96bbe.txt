　　　　“久仰妖帝大名，今日后辈人族斗胆前来挑战，还不忘吝惜赐教！”
<br/>　　妖帝刚要提剑上前，却被燧人氏等三祖与剑皇赤松子四人拦住，想要与他一战。
<br/>　　帝俊眼神一凝，脸上闪过一丝怒气，深深吸了一口气，缓缓道：
<br/>　　“好！好！好！，好你个人族！真是好胆！本帝不发威，你们真以为本帝好惹！今日本帝便让尔等人族看看，我妖族不是什么人都可以惹的。死来！”
<br/>　　帝俊语气之中带着浓浓的威压、霸道、不屑，更有一股高高在上之意！
<br/>　　“哼！”
<br/>　　帝俊冷哼一声，手持金色帝剑，一举一动都带着莫大的威势。他全身帝皇之威在此刻完全释放出来，在他的周身，虚空震荡，风暴迭起，威压众生。
<br/>　　“嗡！”
<br/>　　燧人氏等人也毫不示弱，虚空一阵波动，在燧人氏的周边，出现一个虚空旋涡，从里面走出一道身穿赤色王袍，全身燧火环绕的燧皇法相，在它的身上，能够看出刀耕火种，驱赶野兽，捕杀妖族，点燃火种，率领人族大战等各种异象，是他开创了人族的文明。
<br/>　　此刻，燧皇法相手中持有一根巨大的燧木，燧木之上，燃烧着一团熊熊的燧火，那是开启万族文明进化的火种，此乃功德圣器。
<br/>　　“嗡！”
<br/>　　“嗡！”
<br/>　　“嗡！”
<br/>　　紧接着，有巢氏等三人也时现出自己的法相。有巢氏的天地法相是一座洁白高耸的大厦，碧丽堂皇，仙气缥缈，鳞次栉比，其内一片安宁。
<br/>　　那是一片天堂！人类理想中的天国！
<br/>　　有巢氏头顶天堂法相，隐现于云雾之中，有些虚无缥缈的感觉，又带着一股煌煌之威。他一手托着一间简陋的木房，上面散发着金色的功德金光，这也是一件功德圣器！
<br/>　　缁衣氏身边飞来一件金色的法衣，法衣随风而涨，瞬间便涨到半边天空，将天边都遮蔽，金光闪闪，绽放出耀眼的光芒，将周边虚空禁锢，抵抗着帝俊的帝皇威压。
<br/>　　这是缁衣氏的天地法相，圣衣法相！
<br/>　　同时，她手中也多了一件金色的简陋兽皮衣衫，闪烁着功德金光，与燧木、木房三者连成一个三角之阵，来抵御帝俊的进攻之势。
<br/>　　赤松子的法相相对来说就要简单许多了，他的法相是一柄青色巨剑，剑光青翠欲滴，剑刃锋芒内敛，就仿佛一柄普通木剑。木剑遇风而涨，瞬间便涨得巨大无比，赤松子以身御剑，身剑合一，配合燧人氏等三人杀向帝俊。
<br/>　　四人身形一纵，化为四道流光冲天而起，瞬间便将帝俊围在中央，摆出四方之势，身后各自一道神轮，发出耀眼的光芒，笼罩八方，与帝俊对抗。
<br/>　　那是他们突破转轮境的转轮。一道转轮就是准圣初期，燧人氏的转轮是赤红色的，隐隐有燧火在上面燃烧着，那是燧火轮。有巢氏的转轮是白色的，那转轮宛如圆月，上面刻着各种建筑，美如天堂。那是一种信仰，为信仰轮。
<br/>　　缁衣氏的转轮是金色的，金光闪闪，释放出无量神光，抵御敌人攻势，这是一道偏向防御的神轮，她称之为神光轮。赤松子的神轮为青色的，上面环绕着锐金之气，转轮旋转，释放出无尽杀意，将空间切割。
<br/>　　四道转轮在四人脑后缓缓旋转，释放着无尽神通，配合其主人的攻击。
<br/>　　“哼！”
<br/>　　帝俊冷哼一声，手持金色帝剑，面色略带凝重，毕竟四人都是准圣，虽然不如自己，但要收拾他们，也不是那般容易的。他也不废话，直接开启大神通，一举一动，带有莫大威势，无双帝威向着四人压来，不过瞬间就被三件功德圣器挡住了。
<br/>　　有三件功德圣器在手，就相当于防御无敌，先天立于不败之地。
<br/>　　四人各自使出自己的神通，相互攻击，各种神通无量，招式凶险，相互对轰，这一片空间都被破碎了，不断有黑洞衍生。
<br/>　　帝俊的实力的确是比燧人氏四人强大不少，但他一人想要攻破四人的防守，也不是那般容易的。几人的战斗，你来我往，一时之间难以分出胜负，洪荒中的大能们纷纷惊骇不已，想不到人族竟然有如此地牌，能够对抗妖族。
<br/>　　“轰隆隆！”
<br/>　　“杀杀杀！”
<br/>　　下方大地之中，人妖两族依旧在不停地厮杀，两族互有伤害，但更多的是妖族陨落。人族的凶残与狠厉，震惊了无数人。胜利已经渐渐倾向于人族了。
<br/>　　一旁的东皇太一见此，想要上去帮忙，可是，人族另外五大转轮武皇境强者正在紧紧盯着他，见他一动，顿时五人便围了上来，他们使出天地法相与东皇太一对抗起来。
<br/>　　五人并没有功德圣器，但他们有自己的兵器，虽然不能完全抵抗东皇之威，但五人联手，也足够抵制东皇太一了。
<br/>　　各自使出自己的法相，自己的本命神通，转轮神通，一时之间，三处战场，就这样僵持了下来，各自分不出胜负。
<br/>　　……
<br/>　　盘古殿中，十二祖巫再次聚首。
<br/>　　“人族不可留！如今人族就可以与妖族对抗，若是再让他们如此下去，到时哪还有我巫族之事？”空间祖巫帝江大声道。
<br/>　　公共祖巫接着说道：“各位兄弟、妹妹，我手下有人来报告，人族的怨魂之中，有一种奇特的怨力，能够伤妖族的元神，若是我们能够得到更多的人族这种怨力，炼制十二柄戮妖幡，我们对付妖族就指日可待了。”
<br/>　　“竟有此事，此话当真？”
<br/>　　空间祖巫连忙急声道。此时事关重要，他不得不再次确认一遍，就连其他祖巫也将目光投向共工。
<br/>　　“不错，的确如此。”
<br/>　　“真的？太好了，真是天助我巫族，妖族小二们，等着受死吧。”众祖巫十分高兴，终于可以解决妖族了。
<br/>　　“好，既然如此，那大家发表意见吧，杀不杀人族？”帝江祖巫沉吟片刻，对着众人说道。
<br/>　　“大哥，你知道我的时间法则有些诡异，我又是能够借助时间法则看到未来的一些画面。”这时，时间祖巫烛九阴对着帝江说道。
<br/>　　“不错，你看到了什么？”帝江问道。
<br/>　　“我看到，在那些画面之中，人族是未来的主角。那时没有巫族，也没有妖族。”烛九阴脸色有些凝重道。
<br/>　　“什么？竟会有如此之事？”
<br/>　　帝江大惊，实在是自九阴说的话有些骇人听闻了，巫妖两族都不见了，只有人族，这怎么可能？
<br/>　　“不错，那一段画面，只是未来的一种可能，未来是多变的，是不可预测的，它也有可能是假的。”时间祖巫烛九阴不确定道。
<br/>　　“嗯。就算只是一种可能，也不能让它发生，人族当灭！”
<br/>　　帝江面色凝重，毫不犹豫下达命令，屠杀人族。若是刚才还有些犹豫的话，现在他完全是下定决心了。对于他来说，他绝对不会让任何威胁巫族的因素存在，哪怕它只是一种可能，也不行。
