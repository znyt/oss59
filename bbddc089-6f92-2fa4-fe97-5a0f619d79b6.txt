        童筱颖本能地将小尚轩抱了起来，护在怀中，季思妍那几道手劲，狠狠地落在了童筱颖的臂膀上。
<br/>
<br/>        但是，当傅昼景收拾完行李箱，从屋内走出来时，季思妍又及时收了手，一改刚刚泼妇之态，向小尚轩伸出手来，温柔地哄道：“小景，到妈妈这里来！妈妈抱抱，好不好？”
<br/>
<br/>        “不要！不要！不要！”小尚轩紧紧地抱着童筱颖的脖子，嗷嗷大哭。
<br/>
<br/>        最后，还是傅昼景从童筱颖怀里抱过小尚轩，在傅昼景的陪同下，小尚轩才同意跟季思妍一起回滨江城的外公外婆家先去暂住一段时间。
<br/>
<br/>        傅昼景先护送季思妍母子去了滨江城，随后才赶去了云城西镇的民政局，与童筱颖约好了时间，在大门口碰面。
<br/>
<br/>        童筱颖和傅昼景的离婚程序很简单，离婚协议书上，她童筱颖净身出户。
<br/>
<br/>        她当初是怎么嫁给他的，她就选择怎么离开。
<br/>
<br/>        与结婚证的大喜红色不同的是，现在的离婚证是一本暗红色，她和傅昼景各持一本。
<br/>
<br/>        从民政局里领了离婚证出来后，童筱颖将脖子上那块玉坠取了下来，递给了傅昼景。
<br/>
<br/>        玉坠是一个冷字上缠绕一只栩栩如生的凤凰。
<br/>
<br/>        与大哥傅夜沉手里的那块冷字上缠绕一只栩栩如生的龙，是一对祖传玉坠。
<br/>
<br/>        童筱颖清楚的记得，当年也是在这民政局门口，傅昼景把这块玉坠交付给了她。
<br/>
<br/>        如今，她完璧归赵。
<br/>
<br/>        “把这块玉坠，给尚轩吧！”童筱颖努力地挤出一抹笑容来。
<br/>
<br/>        傅昼景愣了半晌，才缓缓地抬起手来。
<br/>
<br/>        童筱颖一松手，玉坠随着红绳，一起跌落在了傅昼景的手心。
<br/>
<br/>        从这块玉坠上传来了属于她童筱颖的体温，傅昼景忽然间黯然神伤，握住玉坠的同时，也将童筱颖拉入了怀中。
<br/>
<br/>        童筱颖身心一怔，惊慌失措地抬起头来。
<br/>
<br/>        和当年的情形几乎一模一样，她淡然的眸子里，闪过这道令她熟悉得不能再熟悉的阴影。
<br/>
<br/>        傅昼景微微俯身，低头，犹如羽毛般的吻，轻轻地印在了童筱颖的额头上。
<br/>
<br/>        他们不同于其他离异的夫妇，见面如同敌人，分外眼红。
<br/>
<br/>        反倒像是一对被迫分离，彼此有苦衷却不能明说的苦命鸳鸯。
<br/>
<br/>        “筱颍，思妍她得了抑郁症。等她病好了后，我就来找你。你等我，好吗？”傅昼景深情款款地凝视着童筱颖的眼睛，语态诚恳。
<br/>
<br/>        童筱颖没点头，但也没拒绝。
<br/>
<br/>        到此刻，她不想否认，她其实还爱着这个男人。
<br/>
<br/>        “这段时间，我要去滨江城，处理我和思妍之间的事情。你先回澳洲等我。这是机票。”傅昼景说着，随后从裤袋里掏出钱包，将里面的一张写有她童筱颖的名字的返程机票递给了童筱颖。
<br/>
<br/>        童筱颖垂眸看着傅昼景递过来的这张机票，毫不犹豫地就接了。
<br/>
<br/>        现在的她，还能去哪？
<br/>
<br/>        于他而言，除了澳洲悉尼的那个家，她真的无家可归。
<br/>
<br/>        傅昼景将童筱颖送到云城机场后，童筱颖在傅昼景的目送下进了登机口。
<br/>
<br/>        云城这三年来的变化真的很大，不仅有了飞机场，而且还被傅氏集团投资修筑成一个大型影视城，集影视、旅游、度假、休闲、观光为一体的大型综合性旅游区。
<br/>
<br/>        就连西镇那穷得鸟不拉屎的地方，也发展成了一个比较有少数民族风情特色的旅游景点。
<br/>
<br/>        而这些，都是傅夜沉的“功劳”。
<br/>
<br/>        但是，傅昼景转身离开后，童筱颖却从登机口里走了出来。
<br/>
<br/>        她撕了机票，将机票扔进了垃圾桶中。
<br/>
<br/>        离开机场，童筱颖搭了一辆出租车，回到了西镇的深山区。
<br/>
<br/>        离婚的时候，傅昼景为什么让童筱颖“净身出户”？
<br/>
<br/>        因为他觉得，无论是过去、现在、将来，他都是她唯一的可以依靠的人。
<br/>
<br/>        哪怕她跟他离婚了，她也只能依靠着他才能活。
<br/>
<br/>        所以，傅昼景笃定，他的筱颍，一定会回到他们在澳洲悉尼的那幢别墅里。
<br/>
<br/>        可终究，他还是太高估了自己。
<br/>
<br/>        她的人，爱他的全部，但不代表，他可以控制她的灵魂。
<br/>
<br/>        童筱颖不想当傅昼景手中圈养的金丝雀。
<br/>
<br/>        她不傻，她有自己的想法。
<br/>
<br/>        只是，会因为爱一个人，而让自己变得很低微。
<br/>
<br/>        童筱颖下了出租车后，拖着行李箱，沿着田间蜿蜒的小路，听着蛙叫虫鸣，一步一步，向着曾经养育过她的那片深山大林里走去。
<br/>
<br/>        天色渐晚，她的那抹白色背影，慢慢地被苍茫暮色给吞噬，隐隐地消失在了林间雾霭之中。
