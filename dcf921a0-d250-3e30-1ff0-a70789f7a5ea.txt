　　“他已经三天没休息了吧？”雷恩满脸敬佩。
<br/>　　“希望他能成功。”马特眼中闪过担忧。
<br/>　　“理论上来说，应该不可能。”卡普兰道。
<br/>　　“你最好祈祷他成功，不然我们就要倒霉了。”爱丽丝随口道：“我们之所以能留在这里，就是因为他和红后有约定。如果他研究失败变成丧尸，你们觉得红会会放过我们。”
<br/>　　“现在走还来得及吗？”卡普兰脸色发白。
<br/>　　“想想我们扔进焚化炉的尸体吧，你觉得红后会让我们出去。”马特冷笑道。
<br/>　　“那我们不是死定了！”
<br/>　　“不一定。”爱丽丝神思不属。这几天她已经察觉出不同。自己的力气变得越来越大，五感变得越来越敏锐。杀起丧尸来砍瓜切菜，比受过专门训练的突击队员还迅速。
<br/>　　不然，基地内的丧失不会这么快清理干净！
<br/>　　这不正常！
<br/>　　她甚至能感觉到苏重体内的异常，就好像两人体内有着相同的东西在共鸣。
<br/>　　那东西，显而易见的就是生化病毒。
<br/>　　“自己已经感染了病毒，但并未发作！”爱丽丝隐隐有所猜测。
<br/>　　如果有我做参考，他能不能快速研究出血清呢？爱丽丝拿不定主意。
<br/>　　她对苏重了解太少，缺乏信任。而且苏重神秘异常，她怕自己刚说出秘密，就会被苏重按在实验台上大卸八块。
<br/>　　实在是苏重三天内在研究室内表现的太过疯狂，和那种狂热科学家没任何区别，这种人能干出什么冷血的事情来她都不会怀疑。
<br/>　　“他在干什么！”雷恩突然大叫。
<br/>　　爱丽丝不由自主看过去，只见苏重拿着高压注射器，将一管绿色液体直接打进左臂内。
<br/>　　“你确定不用小白鼠做些试验。”红后虚影突兀出现在实验室内。
<br/>　　苏重脸色苍白，豆大汗珠不断从额头落下，眼中布满血丝。他已经到了极限。
<br/>　　“来不及了啊，成败在此一举，活着我们就能继续合作，如果真死了，我就只能在焚化炉里和你做邻居啦。”苏重笑了笑道。
<br/>　　他是真没把握，三天时间实在太短。即便他花费本源，作弊推演。也有些来不及。
<br/>　　保护伞研究数十年才搞出些名堂，苏重只用三天，就想对保护伞终极成品作手脚，难如登天。
<br/>　　好在红后提供的数据内，本来就有抑制剂这个方向的研究，苏重在其基础上疯狂拓展，这才有了这一管药剂。
<br/>　　深吸一口气疲惫的靠坐在椅子上，转头看向玻璃窗外的众人，苏重笑了笑道：“我需要休息一会儿，如果我活下来，就请恭喜我。如果我变成丧尸，你们就自己离开蜂巢基地吧。这里并不安全。”
<br/>　　然后苏重实在忍不住困倦，靠在椅子上就睡了过去。
<br/>　　这一觉睡的并不舒服，体内不断传来灼痛，让他一直处于半梦半醒之间状态。体温逐渐升高，脑袋昏昏沉沉。
<br/>　　四个小时后，苏重睁开眼。他脸色苍白依旧，眼中血丝密布，看着趴在实验室外满脸担忧的众人。
<br/>　　苏重站起来笑道：“恭喜我吧，我暂时做不成丧尸啦。”
<br/>　　“好！”众人欢呼雀跃。
<br/>　　苏重的成功，也代表着他们多了条活路。以后暂时不会害怕病毒感染，因为苏重制作出了抑制剂。
<br/>　　同时他们还可以继续呆在基地。外面已经变成末日。三天来，他们已经通过红后了解过浣熊市清醒。
<br/>　　到处都是感染生化病毒发作的人。军队接管城市，阻挡一切人离开浣熊市。
<br/>　　所有人只能在室内等死，各种犯罪层出不穷，整个浣熊市宛若地狱。
<br/>　　相比起来，幽暗阴森的地下基地，反而像是世外桃源一片安宁。
<br/>　　“先生，您的智慧超乎想象。”红后一脸欣喜。
<br/>　　“你这是在夸我吗？”苏重笑呵呵。
<br/>　　“先生，根据您的体态特征，您似乎并未清除体内病毒。”红后冷着小脸泼冷水。夸你？我呵呵你一脸！
<br/>　　“三天内研究出杀死病毒的血清可能性太低，你说的吗，不足百分之一。”苏重不在意。
<br/>　　“这只是初代抑制剂，他能让我多活五天。五天后，如果我的研究速度比不上病毒的变异速度，那我还是要进焚化炉的。”苏重满脸轻松的调侃自己。
<br/>　　他能在三天内研究出初代抑制剂，就有自信在五天内研究出第二代！
<br/>　　苏重离开实验室，清理一下个人卫生，美美的睡了一觉。他这三天实在太累，病毒在他体内肆虐，被折腾的不轻。
<br/>　　醒来之后，他把几人召集起来：“之前我只是研制出了初代抑制剂，想要活命，还需要继续研究第二代抑制剂。不过有了第一次成功经验，二代抑制剂不过是在原本基础上进行优化就好。短期内不用担心病毒爆发。”
<br/>　　几人闻言出了一口气。苏重是他们呆在蜂巢基地的基础，如果他死了，红后绝对会翻脸不认人。
<br/>　　“丧尸你已经被清理完毕，你们可以开始做长远打算了。”
<br/>　　人一旦闲起来，就容易出问题。他要让着几个人忙碌起来。
<br/>　　“基地物资够我们生存半年，但这远远不够。生化末世不知道还会持续多久，物资越多越好。”苏重给他们讲明白，之后一切行动，都是为了他们自己，多出一份力，说不定关键时候能救人一命。
<br/>　　“基地种植园可以开启，一旦成功运转，它会为我们提供源源不断的食物。”
<br/>　　几人点头，他们之前也看过基地种植园。以前是为了给基地培养药材用的，想要生产可使用植物，需要一定的改造扩建。
<br/>　　“还要去外界收拢物资。现在浣熊市只是刚刚开始混乱，世面上还有不少食物没被污染。能搜集多少，就搜集多少。”
<br/>　　除了爱丽丝，几个人脸上不由露出担忧神色。他们这几天通过红后，可是知道浣熊市的混乱。而且他们清楚看到，一旦被丧尸划伤咬重，很快就会变成丧尸。
<br/>　　他们前往地面一旦被丧尸围住受伤，很可能会感染病毒。
<br/>　　苏重对此早有准备。
<br/>　　“你们难道忘了我这三天在干什么？”
<br/>　　“对啊，我们已经有了病毒抑制剂啦！”卡普兰大松一口气。
<br/>　　“不要太放松，抑制剂不是万能的。只能延缓病毒发作时间，要是因此大意了，我后续研究又没跟上，你们就只能陪我进焚化炉了。”苏重泼冷水道。
<br/>　　“好好好……我们不会放松。”几个人笑呵呵。
<br/>　　三天就能研究出遏制病毒的抑制剂，他们对苏重的研究能力佩服的五体投地。
<br/>　　开始谁都没信心，但苏重真的办到了，他们对苏重的能力顿时变得极其信服。
<br/>　　这样他们就不怕病毒感染了，也就可以大胆的前往地面搜集物资了。
<br/>　　“还有，基地药品资源有限，我并不能制作太多抑制剂。所以，在执行物资搜集的过程中，药品同样是重中之重。”苏重提醒道。
