　　数百道毁灭之雷降临，楚风眠挥手剑锋一动，便是全然斩灭。
<br/>　　这种实力让那些隐藏在周围虚空之中的半步天人境强者心中都是震惊不已。
<br/>　　哪怕是让他们来承受这数百道毁灭之雷，只怕是都要耗费大量的手段，甚至都无法做到毫发无伤。
<br/>　　可如今楚风眠却是如此轻而易举的便是将这些毁灭之类去全然打破。
<br/>　　“难道说此人可以度过这逆天大劫？”
<br/>　　有些半步天人境强者喃喃自语道。
<br/>　　逆天大劫，在古籍的记载之中，如果一旦是可以度过，将会得到无比巨大的好处。
<br/>　　一旦是度过逆天大劫，踏入天人境，便是板上钉钉的事情。
<br/>　　甚至是哪怕未来踏入天人境之中，都会成为天人境之中的大人物。
<br/>　　逆天渡劫，一旦是成功，便是未来名震天下的人物。
<br/>　　“没那么容易，逆天大劫如果那么好度过，也不配逆天二字了。”
<br/>　　五行刀圣眼神冰冷的说道，他充满杀意的双眼已经的死死的看向楚风眠，在算计着，如何才能够杀了楚风眠。
<br/>　　现在这逆天大劫的范围之内，没有任何一尊老古董敢靠近进去，毕竟谁如果进去，就是要跟楚风眠一样，承受逆天大劫的威能。
<br/>　　可一旦是逆天大劫结束，如果楚风眠没死，就是他们这些老古董出手的机会了。
<br/>　　数百道毁灭之雷，只是一个前奏。
<br/>　　位于楚风眠头顶之上的劫雷之中，更多的滚滚劫雷，咆哮凝聚而出。
<br/>　　这劫雷，都是变的越发的愤怒了起来。
<br/>　　轰隆！
<br/>　　惊雷一声响彻而起。
<br/>　　更多的劫雷，咆哮而下，这些咆哮的劫雷，足有数千道，凝聚成了一条条雷龙，向着楚风眠轰击而来。
<br/>　　这每一条雷龙的力量，简直都不亚于一名半步天人境的强者。
<br/>　　一共十条雷龙，同时向着楚风眠发起了攻击。
<br/>　　“来的正好！”
<br/>　　楚风眠抬起头，看向天空之上这一条条的雷龙，从他的头顶之上，一件古老的瓦罐，突然浮现而出。
<br/>　　这古老瓦罐浮现出的一刻，一股浩瀚的人皇之力降临了。
<br/>　　人皇罐。
<br/>　　已经是凝聚了七枚碎片的人皇罐，被楚风眠拿了出来。
<br/>　　在这逆天大劫之下，楚风眠也知道，他自身已经是不能够有着任何的保留了。
<br/>　　而且楚风眠同样也感觉到，这逆天大劫，是现在对楚风眠最大的考验，一旦是楚风眠可以度过，楚风眠就可以横扫另外三大势力。
<br/>　　另外那五行刀圣，已经是知道人皇罐在楚风眠的手心之中了，既然如此，楚风眠更是不需要在任何的隐藏下去。
<br/>　　“化矛！”
<br/>　　楚风眠长啸一声，人皇之力爆发而出，那人皇罐便是就在楚风眠的头顶之上，化为了一根古朴的长矛。
<br/>　　人皇之矛！
<br/>　　“给我灭！”
<br/>　　轰！
<br/>　　这人皇之矛凝聚而出的一刻，一把对着劫云之中咆哮而下的其中一条雷龙刺去。
<br/>　　人皇之矛的锋芒贯穿而去的一刻，那一条雷龙被当即贯穿，破灭，化为了无数飘散的雷霆。
<br/>　　楚风眠大手一挥，猛然对着那些飘散的雷霆抓了过去，只见这一道道雷霆都被楚风眠捏在了手心之中，一把吞噬到了本命剑元之中去。
<br/>　　楚风眠的本命剑元，乃是以造化之力凝聚，这些雷霆被楚风眠捏在手心之中，打入到了本命剑元之中去，直接化为了无比纯粹的造化之力，化为了楚风眠本命剑元的补品。
<br/>　　对楚风眠来说，每一次度过雷劫，也是提升他本命剑元力量的时刻。
<br/>　　瞬息之间，在吞噬了那一条雷龙的力量之后，楚风眠身上的气息节节攀升，强横的气息传递了出去。
<br/>　　对外人来说惧怕无比的劫雷，现在却只是楚风眠的补品，来为楚风眠提升力量。
<br/>　　世间最为纯粹的力量，便是劫雷之力，属于天道的力量，现在楚风眠便是要用这劫雷之力，提升他的实力。
<br/>　　“统统都给我破！”
<br/>　　感受到本命剑元力量的暴涨，楚风眠大笑一声，催动人皇之矛，再度向着其他九条雷龙刺杀而去。
<br/>　　人皇之矛，无坚不摧，就在这劫云之中，向着那九条雷龙追杀而去。
<br/>　　一条条雷龙，都在人皇之矛的面前，纷纷破灭，被楚风眠抓取，融入到本命剑元之中，化为了楚风眠的力量。
<br/>　　这一副场景，根本不像是楚风眠在渡劫，而是楚风眠在捕捉着劫雷，来提升他自己的力量。
<br/>　　轰！
<br/>　　终于是在这人皇之矛疯狂屠杀劫雷的一刻，这劫雷之中又是显现出了更加恐怖的一幕。
<br/>　　一道道灰色劫雷，缓缓降临。
<br/>　　当着灰色劫雷降临的一刻，楚风眠身上都感觉到了一种战栗的感觉。
<br/>　　“混沌！”
<br/>　　“混沌劫雷？”
<br/>　　“这不是只有天人大劫的之后，才会出现的劫雷吗？”
<br/>　　这灰色劫雷之中，蕴含的力量，所代表的正是混沌，天下一切，都是从混沌之中所诞生。
<br/>　　混沌劫雷轰然降临，轰击在人皇之矛上，楚风眠感觉到，这人皇之矛都在剧烈的抖动，向着下方逃去。
<br/>　　眼下楚风眠手中的人皇罐，毕竟还不是真正的极道仙兵，无法硬抗这混沌劫雷，重新回到了楚风眠的手中。
<br/>　　轰隆隆！
<br/>　　失去了人皇之矛的抵挡，这混沌劫雷直指楚风眠奖励了。
<br/>　　这混沌劫雷，才是真正的逆天大劫。
<br/>　　之前的，不过只是一个前奏。
<br/>　　楚风眠站立在星空之中，看着那混沌劫雷的降临，他身上的神树，十二万九千六百枚血肉晶体，龙皇血脉，两把灵剑，也同时出现在了楚风眠的手心之中。
<br/>　　楚风眠的所有力量，在这一刻爆发了出来。
<br/>　　楚风眠所有的积蓄，他现在所有的力量，都在这一刻爆发了。
<br/>　　那圣院宗灵泉的灵力，也被楚风眠全然吞噬，全力的将那一缕太初气息跟楚风眠的左手融为一体。
<br/>　　楚风眠感觉到，他的血肉都在蜕变，这一缕太初之气一旦是融入其中，那楚风眠便是不再是血肉之身了。
<br/>　　也不像是不朽真身，甚至是仙人的存在，而是超越了仙人，甚至是仙帝，天地所化的一种存在，天地所化的异类。
