<br/>				“我体内有混沌青莲之事，只有少数的几人知道，而我也相信，这几人并不会将这事与外人说，而且，这些人明显不是这片上游大陆的人，却又知道我身怀混沌青莲，那么……”<br/>　　<br/>　　她脑海中有着一个念头浮起，只是，不太愿意去相信自己的猜测。<br/>　　<br/>　　轩辕墨泽看了她一眼，低沉的声音带着严肃的道：“若不是知道的几人说出去的，那么，极有可能是有人落入那些人的手中，从中被得知了混沌青莲的下落。”<br/>　　<br/>　　他的声音一顿，问：“除了我之外，还有谁知道你身怀混沌青莲？”<br/>　　<br/>　　“混元老头，以及卓君越。”她的声音一顿，眸光微闪：“还有一人，就是当年将青莲给我的那个神秘的扫地老者。”<br/>　　<br/>　　“混元子为了你可以不顾性命的以死相护，如今又在星云仙宗养伤，自然不可能是他，而卓君越……”他的声音微顿，看向她。<br/>　　<br/>　　“卓君越一直在凤府里，而且他我也相信，不是会出卖我的人。”凤九缓声说着，她微低下头，手指划过丹田处被划过的那一道伤口。<br/>　　<br/>　　“那些人是想从我的身体里将青莲挖出来。”说着这话，她唇角微勾起一抹弧度，眼中也浮现着一抹冰冷的寒光。<br/>　　<br/>　　“当年那个老者既然将青莲给我，那就不会将青莲在我身上的消息告诉那些人，哪怕是死，也一定不会说。”<br/>　　<br/>　　她的眸光幽幽，神色冷然：“但，这些人既然能找上我，而且知道青莲在我身上，必定是得知了消息，所以我猜测，当年的那扫地老者，只怕落到他们的手中已经丧命。”<br/>　　<br/>　　“搜魂术。”轩辕墨泽缓声说着，看着她，道：“定是被用了搜魂术得知了事情。”<br/>　　<br/>　　两人在这里猜测着，推敲着，却不知，这事情竟如他们推敲的大致一般。当年那在桃花坞中扫地的神秘老者，正正是落入那些人的手中，被那些人用了搜魂术得知了混沌青莲的下落，从而找上了凤九。<br/>　　<br/>　　“这伤很奇怪，青莲之力治不好。”凤九的目光落身上的伤上，眉头微拧：“看来是有备而来。”<br/>　　<br/>　　“我会让阎殿的人去查，看看这些人的来历。”轩辕墨泽说着，转身进了里面衣箱中帮她再取出一套衣裙来。<br/>　　<br/>　　“先将衣服穿上。”<br/>　　<br/>　　“嗯。”她应了一声，在他的帮忙下将衣服穿上，只是，微扯动着身上的伤口时，伤口处传来丝丝的疼痛。<br/>　　<br/>　　“你到床上躺着吧！”轩辕墨泽没有再让她出去，而是扶着她先去里间床上躺着。<br/>　　<br/>　　“你跟他们说一下，让他们不要太担心。”<br/>　　<br/>　　轩辕墨泽点了下头，这才走出外间，将房门打开。<br/>　　<br/>　　候在院中的众人一见房门打开，连忙走上前来：“怎么样？伤口严重吗？”<br/>　　<br/>　　轩辕墨泽微侧过身让他们进去：“已经包扎好了，如今躺床上休息着，应该没什么大碍，不必担心。”<br/>　　<br/>　　听见这话，众人松了一口气，又问：“是什么人对你们出手？”<br/>
