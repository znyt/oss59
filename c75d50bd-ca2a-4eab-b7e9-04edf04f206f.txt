　　楚风眠看的出来，这太古蜈蚣脑袋之中的禁制，应该只是一名半步天人境武者布下的。

<br/>　　掌心凝聚灵力轰击而下，打入到了这太古蜈蚣的脑海之中，顿时这太古蜈蚣脑海之中的禁制，都被楚风眠轰杀打碎。
<br/>　　同时楚风眠的一股灵力，也紧随其至打入其中，在这太古蜈蚣之中，种下了楚风眠的禁制。

<br/>　　楚风眠一步踏入，坐在了这太古蜈蚣的头上，这太古蜈蚣也只是顺从的低吼的两声。

<br/>　　纵然是这太古蜈蚣，一身境界已经是达到了九劫古帝，可就连交谈都做不到，仅仅只能够是感受到楚风眠一点点的意思。
<br/>　　不过荒兽一族，也并非是不能开启灵智，只要是踏入了天人境，这太古蜈蚣也就足以开启灵智，成为妖神。

<br/>　　楚风眠看着坐下的太古蜈蚣开口道。
<br/>　　“吼吼。”
<br/>　　这太古蜈蚣，似乎是听懂了楚风眠的意思，兴奋的吼了两句，变得更加顺从了。
<br/>　　“没想到这太古蜈蚣，居然被你收复了。”
<br/>　　看着楚风眠已经是降伏了这太古蜈蚣，海公主也是从方舟之中走了出来，感叹道。
<br/>　　“这太古蜈蚣，藏身在碎星带之中，不知道是猎杀了多少的武者妖神，都没有人可以将其降伏，你降服了他，也算是为碎星带除了一个祸害。”
<br/>　　“恩？这太古蜈蚣，早就被人降伏过了。”
<br/>　　楚风眠听到海公主的话，也是心念一动，看来这是太古蜈蚣已经是被人降伏的事情，根本是没有人知晓。
<br/>　　这太古蜈蚣，早就被人降伏了，却是故意让这太古蜈蚣，留在这碎星带之中，只怕就是为了猎杀其他经过碎星带的武者妖神。
<br/>　　“早就被人降伏了？”
<br/>　　海公主听到楚风眠的话，不由一愣的问道。
<br/>　　“你看。”
<br/>　　楚风眠手心一动，只见从太古蜈蚣的身上，飞出来了几百枚空戒。
<br/>　　许多空戒之上，都沾染着鲜血。
<br/>　　这些空戒，都是被太古蜈蚣曾经猎杀过的武者妖神的，如今都被积攒了下来。
<br/>　　看到这些空戒的一刻，楚风眠的心中也明白了，这太古蜈蚣藏身在碎星带之中，实际上就是为了猎杀经过的武者妖神。
<br/>　　不然如果不是有人命令这样做的话，这太古蜈蚣，绝不会留下这些空戒的。
<br/>　　毕竟太古蜈蚣，没有灵智，自然也无法打开这些空戒，应该是有人故意交代之下，太古蜈蚣才会将这些空戒留下的。
<br/>　　“竟然是这样……”
<br/>　　海公主看到这些空戒的一刻，也相信了楚风眠的说法。
<br/>　　这楚风眠偶然之下，倒是发现了一个隐秘，不过到底是谁，让这太古蜈蚣潜伏在碎星带之中，去猎杀武者妖神，楚风眠也懒得管。

<br/>　　楚风眠看着海公主开口道。

<br/>　　“怎么回事？”
<br/>　　“太古蜈蚣被人降伏了？还是被人诛杀了？我留在这太古蜈蚣身上禁制消失了？”
<br/>　　就在远处，一座大陆中的宫殿之内，一尊老者突然睁开了双眼，眼神之中露出几分冰冷道。
<br/>　　“这太古蜈蚣，可是我们吴家的根本，不能丢失！”
<br/>　　“查！查清楚！”
<br/>　　在这宫殿之中，已经是掀起了一阵轩然大波。
<br/>　　这一切楚风眠当然不知道。
<br/>　　楚风眠现在跟海公主，坐在太古蜈蚣的背上，向着阎罗大陆的方向飞去。
