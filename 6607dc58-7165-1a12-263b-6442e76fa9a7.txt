　　炎良的这句问话中所透露出的信息量很大，以至于不少火族人都震惊当下，显然他早就知道眼前这个淡蓝色衣衫的女人会出现，并且熟知对方身份。
<br/>　　但所有人都确定这是他们火族第一次见到此女，更在此之前没有与任何人有过联系。
<br/>　　所以炎良如此突兀的忽然问话，没有任何开场白的相互认识，在当下便显得尤其古怪。
<br/>　　对比火族人的惊讶，鱼柔女则就显得气定神闲，她面不改色的从炎良面前走过，直径走向出口通道。
<br/>　　只是在与炎良擦肩而过时她忽然开口：“火族就这么点人保护我么？”
<br/>　　鱼柔女的此番话，信息量更大！
<br/>　　水族圣女没有让水族人保护，反而一上来就开口说出让火族保护的话来，这水火两族相争并且不合的事可不是秘密，不仅四大神秘家族知道，连楚涵等对阴阳谷情况略知一二的人都很清楚。
<br/>　　两个世代互为仇敌的种族，怎么可能去保护对方的圣女？
<br/>　　就在全体火族人惊呆之中，炎良却异常淡定：“人数是不多，但各个都是精英，况且圣女您的战力，恐怕根本不不需要任何人保护吧？”
<br/>　　此话一出，四周的火族人全体哗然。
<br/>　　圣女？
<br/>　　整个阴阳谷能称之为圣女的，只有水族圣女！
<br/>　　眼前这个女人是水族圣女？
<br/>　　那岂不是……
<br/>　　此时鱼柔女意味不明的一笑，忽然停下脚步在炎良身旁站定，她目不斜视的看着前方，没有将一丝余光放在炎良身上。
<br/>　　炎良有些惊讶，忍不住侧过身子看向身边的这个女人，但这时候一种淡淡的颤栗感，却忽然从炎良等火族人的心中冒出。
<br/>　　“这一代的火族，当真放肆啊！”鱼柔女轻叹一声，偏头笑看着炎良。
<br/>　　那双瞳孔里带着笑容，却没有丝毫笑意，那勾起到完美无瑕的唇角弧度，也感受不到任何温度。
<br/>　　冰冷的笑容，可怕的强者之尊！
<br/>　　炎良自然早就猜到了眼前这个水族圣女可能是谁，但他依然在第一次与之见面的时候试探，目的只是想知道到底是不是历史遗落的笔记上记载的那般神奇？
<br/>　　而事实证明，水族的确恐怖如斯！
<br/>　　鱼柔女的话让很多火族人都气愤异常，他们本就对水族有天生的厌恶感，这时候在被一个圣女如此呵斥，内心早就憋不住恨不得将这个圣女揍一顿。
<br/>　　可还不等这些火族人朝鱼柔女发难，炎良的举动就让他们全体愣住，又不可置信。
<br/>　　“火族少主炎良，恭迎圣女大驾。”炎良口齿清晰的说着，单膝跪下在鱼柔女身侧。
<br/>　　死一般的寂静在现场蔓延，足足一分钟内都没人说话，所有火族人都觉得自己是不是在做梦。
<br/>　　那个高傲不可一世的火族少主，竟然在他们火族的死敌水族圣女面前，行礼？
<br/>　　这到底怎么回事！
<br/>　　但是少主都跪了，那些火族成员更没有理由不跪，于是他们带着不甘心和愤怒，全体朝着鱼柔女行礼。
<br/>　　“我的族姓一开始就是鱼，因为我本来就是一条鱼。”鱼柔女忽然有所感叹的说着，望着天边的夕阳道：“没想到这么多年过去，一代代水族盛衰下来，鱼氏一族却是水族中一个微不足道的支脉。”
<br/>　　好似在自言自语，鱼柔女轻轻抬起手看着自己的手掌，其上有着一道肉·眼不可见的波纹在不停游走。
<br/>　　“是这个孩子告诉我的，还差最后两道禁制，却来不及了……”
<br/>　　鱼柔女的声音有些缥缈，说话间全体火族人就半跪在她身边，整个场面看起来尤其壮观。
<br/>　　早就肯定了眼前此女身份的炎良，却在此时抬眼道：“肖家第七代神女恐怕就是猜到了这一点，才故意避开了阴谷。”
<br/>　　鱼柔女的眼神一瞬间犀利，冷笑道：“我真是没想到肖家还藏了这么一手，也不知道他们是如何做到的，这种古老的禁术不早就消亡了么？除却水族的种族特色传承，其他家族想要得以还魂，需要付出极大的代价！”
<br/>　　“所以肖家整个家族都灭族，这恐怕就是他们的最终目标，为了某个目标牺牲整个家族，这魄力这谋略，这千年来的推演……”
<br/>　　说着鱼柔女双眼眯起：“若非不知敌我，我当真会佩服至极。”
<br/>　　“最重要的是肖家隐藏太深，他们既不是白家那般明面上要与回炉作对的家族，更不是高家那样忠心的家族。”炎良皱眉道：“肖家的风向，道现在我们都摸不透，不知道目的为何。”
<br/>　　“进去会会不就知道了？”鱼柔女说罢，便先一步朝前迈开，直接走了出去。
<br/>　　炎良起身，打着手势让众火族人跟上圣女的步伐。
<br/>　　此时走在最后的一名火族成员凑到他身边，不甘又不解问道：“少主，我们刚刚为什么跪她？”
<br/>　　“火神若能再世，他们水族也会跪。”炎良只能用另一种方式来解释：“水火两族是不和，但那是在大事情之下，初代圣女已经不是普通的水族人了，她超出了那种范畴，说她是我们的对家早就不合适，或者说我们不够资格来与她作对。”
<br/>　　那名火族成员听的迷迷糊糊，最终只能眨巴着眼睛又问了一句：“水族有圣女，我们火族也有火神吗？我们的火神能回来吗？”
<br/>　　炎良无奈的笑了笑，将这名火族人带着走出阴谷，并随口而答道：“回不来了，永远都回不来。”
<br/>　　当炎良带着最后一名火族人走出阴谷时，鱼柔女已经站在了那座石台上，她的身边站着一圈火族成员，各个都朝鱼柔女投去好奇探究的目光。
<br/>　　除却千年乃至万年来的针锋相对，这还是他们这一代火族第一次见到圣女，也是第一次见到水族人。
<br/>　　他们很想知道这个神奇的种族双脚到底是不是尾巴，于是不少火族人都拼命朝鱼柔女的双脚处看去，根本不管对方是否神色不耐。
<br/>　　“你的族人好奇心很重。”看到炎良出来，鱼柔女立即带着不满的口气开口。
<br/>　　这样被一群火族人盯着看，在万年前简直是不可能发生的事，胆子也太大了！
