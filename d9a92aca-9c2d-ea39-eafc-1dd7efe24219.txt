　　三道身影降临在了广场上，落在了苏莫的前方。
<br/>　　这是三名青年，个个相貌堂堂，气质非凡。
<br/>　　为首一人，约莫二十五岁左右，身穿蓝色长袍，五官英挺，面容俊逸，身上气息更是浩大精纯。
<br/>　　从其身上逸散出的气息，让得周围空间，温度急剧下降，虚空中凝结出一粒粒冰渣。
<br/>　　“汪晟！”
<br/>　　“居然是北海冰宫的汪晟，他怎么来了？”
<br/>　　“汪晟是万尊榜第六十二名的年轻至尊，难道他也要来比试吗？”
<br/>　　广场上，无数的武者惊呼出声，因为这到来的三人中，为首的那名蓝色长袍青年，不仅是北海冰宫的弟子，更是万尊榜上排名第六十二的年轻至尊。
<br/>　　万尊榜上的年轻至尊，数年难得一见，众人没想到，今日能见到排名前一百之人。
<br/>　　不少年轻武者都激动了，万尊榜年轻至尊，这可是他们的偶像，亦是他们的奋斗目标。
<br/>　　“汪晟！”青雅看到汪晟，顿时俏脸微变，面上闪过一丝慌乱。
<br/>　　看台上，一众皇室的强者，包括青俞皇帝在内，都是心中一惊，完全没料到汪晟居然会来。
<br/>　　看台上数十名强者，唯有那名消瘦的鹰钩鼻老者，面色波澜不惊，仿佛一切都与他无关一般。
<br/>　　苏莫暂时停止了出手，他也不急于一时，那名鹰钩鼻老者不过是武皇境二重修为，根本逃不出他的手掌心。
<br/>　　转头看向汪晟，苏莫心中暗暗点头，此人武王境九重巅峰修为，着实不简单。
<br/>　　“青师妹，许久不见，没想到你居然招驸马了！”
<br/>　　这时，汪晟凝视青雅公主，淡笑着说道。
<br/>　　青俞帝国，是依附于北海冰宫的势力，青雅以前也曾拜入过北海冰宫。
<br/>　　所以，汪晟才称呼青雅为师妹。
<br/>　　“汪晟，你来做什么？”青雅俏脸冰寒，皱眉说道。
<br/>　　对于汪晟，青雅没有任何好脸色，有的只有无尽的厌恶。
<br/>　　她以前也是北海冰宫的弟子，本以为加入了北海冰宫，前途不可限量。
<br/>　　但在北海冰宫，她不时的会受到汪晟的骚扰，她实力低微诚惶诚恐。
<br/>　　半年前，她为了自身的安危，迫不得已之下，便离开了北海冰宫，回到了青俞岛。
<br/>　　没想到，现如今对方居然追到了这里！
<br/>　　“师妹要招驸马，为兄自然不能不来！”
<br/>　　汪晟微微一笑，随即转头望向看台上的青俞皇帝，抱拳道：“岳父大人，汪某愿意做青俞帝国的驸马，迎娶青俞师妹！”
<br/>　　汪晟此言一出，全场顿时为之一静。
<br/>　　要知道，青俞皇室为青雅公主比武招亲，能打败青雅公主，才能成为驸马。
<br/>　　这汪晟倒好，直接就称呼青俞皇帝为岳父了。
<br/>　　不过，众人倒也没有太过惊讶，虽然汪晟的做法有些不妥，但对方的实力却是完全有资格做驸马。
<br/>　　而且，以汪晟的天赋和实力，若是成为了青俞帝国的驸马，绝对是青俞帝国之幸。
<br/>　　“这……！”青俞皇帝闻言一怔，微微皱眉问道：“你喜欢小女？”
<br/>　　对于汪晟的天赋和实力，青俞皇帝也是非常看重的，但对方的做法，却是让他很是不喜。
<br/>　　因为对方没有征求他的意见，便如此唐突，明显是不给他面子。
<br/>　　不过，对方的身份摆在那里，他也不能拿对方怎么样！
<br/>　　“岳父大人，晚辈对青师妹一往情深，定然不会辜负她！”汪晟笑着说道，仿佛他已经成为了驸马。
<br/>　　“汪晟，你不要痴心妄想了，我是不会嫁给你的！”
<br/>　　青雅公主闻言，顿时一声娇喝，随即看向青俞皇帝，满脸坚定的道：“父皇，我是不会嫁给他的！”
<br/>　　青俞皇帝见此，顿时眉头紧皱，看来青雅和对方有些过节啊！
<br/>　　对于女儿为何会离开北海冰宫，青俞皇帝不甚了解，只是听其所说，北海冰宫弟子间竞争太过激烈，压力太大，所以不想待在北海冰宫了。
<br/>　　对此，青俞皇帝便也没有强求。
<br/>　　但现在一看，青雅和这位汪晟，似乎有不小的过节。
<br/>　　“汪晟，小女并不愿意……！”青俞皇帝正欲开口，拒绝汪晟，但他话未说完，便被对方打断了。
<br/>　　“岳父大人，对于这个驸马，我汪晟志在必得！”
<br/>　　汪晟面色冷傲，轻笑一声道：“你应该知道，家师乃是北海冰宫三长老，只需一句话，便能让青俞帝国灰飞烟灭！”
<br/>　　“你……！”青俞皇帝闻言，顿时怒急，对方居然以势压人，直接威胁于他。
<br/>　　不过，面对汪晟，面对强大的北海冰宫，青俞皇帝心中，却是提不起多少反抗之力。
<br/>　　别说是北海冰宫三长老，就算是任何一位长老，都能灭掉青俞帝国。
<br/>　　众人见此，均是心中狂震，不愧是万尊榜年轻至尊，汪晟居然丝毫不给青俞皇帝面子！
<br/>　　不过，就算不以势压人，汪晟自身的实力，也不比青俞皇帝弱多少吧！
<br/>　　众人暗暗猜测！
<br/>　　“汪晟，你无耻！”青雅公主面色冰寒无比，怒声娇喝，对方居然出口威胁，简直是无耻至极。
<br/>　　“呵呵！青师妹，为兄对你的心意，你难道不清楚吗？”
<br/>　　汪晟轻笑一声，眸光打量着青雅窈窕的身姿，嘴角勾起一丝笑意，面前的绝美佳人，注定要成为他的女人。
<br/>　　“我是不会嫁给你的！”青雅面色坚定，她就算是死，也不可能嫁给对方。
<br/>　　“呵呵，青师妹，怕是由不得你了！”汪晟淡然一笑，毫不在意。
<br/>　　青雅俏脸苍白，心念急转，他是无论如何，也不会嫁给对方。
<br/>　　可是，对方乃是万尊榜排名第六十二的年轻至尊，不仅自身实力强大无比，还有北海冰宫这个庞然大物撑腰，她如何反抗？
<br/>　　就在此时，青雅的眼角余光，瞥见了身旁的苏莫，顿时脑中灵光一闪。
<br/>　　此人之前一指就弹断了他的青玉剑，实力强大绝伦，定然也是万尊榜年轻至尊，就算实力不如汪晟，估计也不会相差太大。
<br/>　　而且，万尊榜的年轻至尊，全部都是大势力之人，说不定能对抗北海冰宫。
<br/>　　青雅心中立刻有了办法，她现在别无选择，只能一试了。
<br/>　　念头及此，青雅脚步轻移，一步便来到了苏莫的身边，一只玉臂直接挽在了苏莫的臂膀上。
<br/>　　“汪晟，不好意思，我已经选好了驸马！”
