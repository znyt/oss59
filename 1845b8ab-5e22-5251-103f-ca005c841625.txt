　　青云子虽然没收他做内门弟子，但传授了他不少内门法术，后来叶少阳也指点了他很多，苏钦章的实力也是突飞猛进，早已晋升天师牌位，距离地仙也不远了，在这个魔法时代，这实力在年轻一代之中也算是佼佼者了。
<br/>　　不过，他毕竟弟子薄，与绝世强者相比，还是差了一截，最重要的是，青云子选中他掌管山门，实际看中的不是他的实力，而是他在管理方面的能力。悦然从小在轩辕山上修行，历经十几年，虽然不是七大弟子中的最强者，但是法术公会既然让他出头，说明也是信任他的实力，在硬实力上自然比苏钦章要强不少，几大宗师看在眼里，实在有点愤然，但也已经预料到了结果。
<br/>　　“悦然师侄，不可伤了和气。”张无生忍不住出声说道，心中非常不爽，但也没办法，以他的地位和辈分，实在不好亲自对悦然动手。
<br/>　　悦然自然也是知道这一点，知道这些宗师都不会对他动手，因此才会大张旗鼓地挑衅苏钦章。
<br/>　　苏钦章虽然实力不行，但毕竟是茅山的脸面，只要赢了他，这件事很快会在法术界流传开来：身为法术界最大门派之一的茅山的代理掌教，居然打不过法术公会一个最年轻的弟子，对于整个法术界来说，都是一件很伤士气的事情，对于茅山来说，更是耻辱……
<br/>　　两人的手印隔空相撞，却吸在一起，互相消耗着对方的修为。
<br/>　　悦然神情自若，苏钦章却渐渐吃力，一张面孔潮红起来……
<br/>　　“苏道长，你可不要让我……”悦然脸上的笑容看上去更加欠打。
<br/>　　苏钦章用尽全力，苦苦支撑，咬牙说道：“我到底是个外门弟子，欺负我不算本事，可恨我掌门师兄不在……”
<br/>　　“你是说叶少阳？”悦然嗤然一笑，“可惜他失踪了，不然，我倒想来与他一战，人间第一天师，呵……”
<br/>　　神道清微微笑着，看着这一幕，对悦然说道：“点到即可，给茅山留点颜面，毕竟是道门第一宗派……”
<br/>　　他这话听上去像是劝解，实际上更是对茅山的讽刺。悦然回忆，轻念一声佛号，一连变幻了三个手印，法力疯狂喷吐。苏钦章再也支持不住，踉跄后退，险些摔倒，一只手捂着胸口，恨恨地看着悦然，刚要开口，喉头一甜，差点一口血喷出来。
<br/>　　败了……苏钦章心如死灰，一时间想到的不是自己，而是茅山的名节，想说点什么，又怕那样更加给人嘲讽的机会，一时间站在原地，窘迫得不知道怎么办好。
<br/>　　“茅山掌教，不过如此。”悦然轻轻一笑，“哦，茅山代理掌教。”
<br/>　　“你——”
<br/>　　苏钦章刚要开口，突然感到脑后生风，转头一看，却是一道青衣，从山门的方向直飞上来，心中一颤，莫不是……
<br/>　　青衣飞来，落在苏钦章身后。
<br/>　　苏钦章激动得几乎要哭了，急忙拜道：“大师兄！”
<br/>　　来的正是道风。
<br/>　　身穿一件长长的青色道袍，衣角和披肩的长发在风中翩翩飞舞着，看着还是那么帅，那么装逼。
<br/>　　曾经的人间道神。
<br/>　　“道风！”张无生等人纷纷倒吸口气。
<br/>　　玉玑子和慈心师本能地就往人群后面躲，生怕被道风看到。悬空观一战，他们都见到过道风的雷霆手段，那个时候的道风，就能跟无极天师打成平手，这又修行了一段时间，传闻他修为更近一步……
<br/>　　不过，他们的担心是多余的，道风根本没注意到他们，目光落在悦然身上，看上去有点气定神闲，盯着他，淡淡说道：“我茅山开派以来，从来没有人敢打上山门，你是第一个……你欺我茅山无人么？”
<br/>　　悦然微微一怔，一时间没能开口。
<br/>　　这是他第一次见到道风。从前只是听说过他的名字，心中也怀有挑战的心思，不过真的见到了，他也不知道为什么，道风虽然什么都没做，但却有一种特殊的气质，让他觉得自己矮了一头，这感觉很不舒服，悦然一咬牙，针锋相对地说道：“既然道风子你这么说，想要怎样？”
<br/>　　道风根本没有回答，人影一动，抬起了一只手，手里抓着一这青翠的鞭形法器，闪耀着神光，就这么对着他砸了下去。
<br/>　　打神鞭！
<br/>　　没有什么花样，也没用什么法术，就这么直接对着他的脑袋砸下去。
<br/>　　“怕你怎的！”
<br/>　　悦然虽然看着不妙，但是当着这么多人的面，为了面子，一咬牙，再度结印，祭出佛门大手印，将打神鞭托住。
<br/>　　好像并没有想象中那么可怕。
<br/>　　悦然松了口气，笑道：“人间道神，不过如此啊。”
<br/>　　道风嘴角弯起，露出一个轻蔑的微笑，再一用力，打神鞭将悦然祭出的佛门大手印生生压了下去。
<br/>　　悦然这才震惊地发现，自己根本没有力量抵抗。
<br/>　　一旁的神道清也是大惊，但也不好插手，急忙拱手说道：“风之谷主，你已不是茅山的人，这么做怕是不妥。”
<br/>　　道风看也没看他一眼，仍旧盯着悦然，说道：“你欺我茅山无人么？”
<br/>　　语气很轻，似乎只是静静地与人交谈，但是听在悦然耳朵里，却是格外的恐怖。
<br/>　　“我……”悦然的语气软了一下，随即一咬牙说道：“道风，我师父是星月奴，我不信你敢杀我！”
<br/>　　道风微微垂下眼帘。
<br/>　　悦然以为他被说动，咧嘴刚要笑，道风却睁开眼睛，用更加清晰的声音吐出一句话：
<br/>　　“辱我山门者，罪无赦。”
<br/>　　打神鞭用力砸了下去。
<br/>　　“道风你敢！”神道清大惊，什么也顾不上了，瞬间扑了上去，宛如一道罡风吹过。
<br/>　　道风分出一只手，摊开手掌，放出三清鬼符，迎了上去，将神道清瞬间缠住，同时手中打神鞭重重落下，砸在了悦然的脑袋上……
<br/>　　砰的一声，没有任何意外，悦然脑浆迸裂，魂魄被打神鞭的法力驱赶，直入地下，连一秒钟都没停留，直接往六道轮回去了。
