　　阵法之外，那考核长老，坐在椅子上，打量着阵法，很是悠闲。
<br/>　　“这两个小子，居然你还没有出来？本来以为让他们见到那阿修罗，这两人就会乖乖跑出来的，怎么这个时候还没有出来？”
<br/>　　考核长老不由的自言自语起来。
<br/>　　“如今寒月漓，可是学院那些老古董眼前的红人，若是楚风眠死在这考核之中的话，寒月漓找过来可不好办啊。”
<br/>　　“不过这是圣党交代下来的任务，就算是这两个家伙死了也无妨，两个内门弟子死在考核中，不算什么大事。”
<br/>　　考核长老冷冷的想到。
<br/>　　就在这个时候，一阵光芒，突然从阵法中散发出来。
<br/>　　在洞府中，两道身影，楚风眠跟白萱，直接是从阵法中走了出来。
<br/>　　“哦？你们出来了？没有通过也无妨，下一次得到资格，还可以来参加考核。”
<br/>　　考核长老笑着开口道。
<br/>　　“这次没有通过考核，你们可以离开北邙圣域了，没有晋升种子弟子，都没有资格在这北邙圣域中修行。”
<br/>　　脸上笑着，考核长老的心中却是在冷笑，得罪了圣党的人，还想要晋升种子弟子，那有那么容易。
<br/>　　楚风眠嘴角露出一抹诡异的笑容道。
<br/>　　“哦？谁说我们没有通过考核的？”
<br/>　　“不就是斩杀一头阿修罗吗？没想到这考核，居然如此简单。”
<br/>　　“恩？你说什么？”
<br/>　　考核长老听到楚风眠的话，先是一楞，随后不由的开口道。
<br/>　　“你们完成了这任务，斩杀了那阿修罗？”
<br/>　　考核长老的言语中都是震惊无比道。
<br/>　　这阿修罗的实力，考核长老可是知晓的，就算是他自己进入其中，只怕都无法斩杀，反而是会被阿修罗斩杀。
<br/>　　“楚风眠，你可要清楚，你现在撒谎也没有任何意义，我们一查就可以查出来！”
<br/>　　考核长老冷冷的开口道。
<br/>　　“考核失败，自己乖乖离开便是，不要再多找麻烦。”
<br/>　　楚风眠没有开口，看了一眼旁边的白萱，白萱手一动，一把骨剑直接扔到了地板上。
<br/>　　这骨剑的坚固，令地板都被砸出来了一个大坑。
<br/>　　“这是？”
<br/>　　考核长老看到这骨剑，尤其是看到上面的恐怖魔意的一刻，都是一惊。
<br/>　　“阿修罗的骨剑？如此浓厚的魔意，你不止是斩杀了阿修罗，更是将他炼化到了这骨剑之中？”
<br/>　　考核长老开口的时候，语气都在颤抖。
<br/>　　这骨剑，正是阿修罗的骨剑，阿修罗是绝对不会丢弃自己的兵刃的，这骨剑也就代表着一名阿修罗的性命。
<br/>　　尤其是上面的魔意，显然是将阿修罗的精血，全部融入到其中，才可以做到的。
<br/>　　两个神力境极限的武者，居然是斩杀了一尊生死境高阶的阿修罗。
<br/>　　阿修罗本来就是天魔一族中最难对付的存在之一，每一尊都足以媲美人类中的天才，越级战斗的恐怖存在。
<br/>　　可现在居然是被直接斩杀，甚至是一身精血都被融入到了这骨剑之中，祭奠成了一把灵剑。
<br/>　　这让考核长老看到，就有些头皮发麻。
<br/>　　“不可能，这不是真的！”
<br/>　　考核长老有些不可置信的说道。
<br/>　　“这肯定你曾经得到过的奇遇，早就准备好的。”
<br/>　　“那就请考核长老检查吧。”
<br/>　　楚风眠也不废话，直接将这骨剑扔给了考核长老。
<br/>　　考核长老接过骨剑，半信半疑的拿了过来，还是有些无法相信。
<br/>　　他手心一动，在这骨剑面前，便是出现了一个水镜，来推算这阿修罗。
<br/>　　只见刚刚在法阵的身影，逐渐出现，阿修罗败在楚风眠的手中，被直接镇压，灭杀，一身精血融入到骨剑中，将骨剑锻造出来。
<br/>　　这一切，全部都被考核长老看在眼中，每一步都是无懈可击。
<br/>　　看完这一切，考核长老在看向楚风眠的时候，都宛如是看向一个怪物。
<br/>　　这生死境高阶的阿修罗，居然是在楚风眠的手中都毫无还手之力都被镇压灭杀，这等实力简直可怕至极。
<br/>　　尤其是那战龙之爪，其中蕴含的威力，令他的水镜上都是出现了丝丝裂痕，差一点都要被震碎。
<br/>　　“本以为只有寒月漓一个怪物，此人的实力也真是可怕！”
<br/>　　考核长老看向楚风眠，目光闪烁，许久才缓缓的开口道。
<br/>　　“不错，这阿修罗是被你斩杀，这考核你是通过了。”
<br/>　　“不过。”
<br/>　　考核长老的脸上突然露出几分阴冷，话锋一转的说道。
<br/>　　“这道考核虽然是通过了，不过你们二人还无法晋升成为种子弟子，还必须要进行一道考核才行。”
<br/>　　“什么？还有一道考核？”
<br/>　　白萱听到这考核长老的话，眉头一皱道。
<br/>　　“种子弟子的考核，不一直只有这一项吗？”
<br/>　　就连楚风眠的眼神中，都露出了几分不善。
<br/>　　这种子弟子的考核，他也有所听闻，这一道考核算是刁难，但是也算是在规则之中。

<br/>　　“恩？”
<br/>　　听到这考核长老的话，楚风眠跟白萱的脸色都是微微一变。

<br/>　　白萱有些担心的说道。
