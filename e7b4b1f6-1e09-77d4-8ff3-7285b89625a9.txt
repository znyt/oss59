　　吧嗒！
<br/>　　半截雪茄烟掉落在地上，火星飞溅而起，雪茄烟烟头砸得熄灭掉……
<br/>　　船舱门口，西伦科特看到坐在直升机舱门口的詹姆斯举起狙击步枪，看到狙击镜在空气中形成的银色反光，为之大惊。原本西伦科特认为自己手中抓着詹姆斯的把柄，詹姆斯一定会跟自己谈判，并且最终做出让步；但是西伦科特发现自己错了，詹姆斯从一开始就没打算要跟自己进行谈判，他从一开始就决定要自己的命！
<br/>　　砰！
<br/>　　咻！
<br/>　　以超音速飞行的子弹，带着尖锐的破空声，飞向站在船舱门口的西伦科特，子弹击中了西伦科特的右肩，他握着沙漠之鹰的右手被子弹打断，整条手臂飞了出去，鲜血从肩膀断口处喷射而出。
<br/>　　西伦科特发出一声如受伤野兽般愤怒的咆哮声，对着坐在直升机上的詹姆斯嘶吼道：“詹姆斯你想要杀人灭口，既然这样，那我们就同归于尽！”说着，西伦科特把握在左手中的背包甩了出去。
<br/>　　直升机上，詹姆斯没有理会西伦科特的咆哮，拉动枪栓，重新推上一颗子弹，从抛壳窗飞出的弹壳在机舱地板上弹跳着，发出“叮当”的金属脆响声，滚落下直升机，然后又一次扣动了扳机，这次子弹击中了西伦科特的胸膛，子弹贯穿了西伦科特的身体，造成了一个贯穿伤，在他背后破开一个血洞，大量的鲜血和内脏碎片喷溅而出，喷洒在船舱舱门上，而被西伦科特甩出去背包，划出一道抛物线朝着海面坠落下去。
<br/>　　货轮上每一个人都是被眼前这出乎意料的一幕惊讶到了，没有人想到詹姆斯会如此果决，根本不跟西伦科特进行谈判，直接选择了对他进行狙杀。至于西伦科特在看到詹姆斯所说的那些话，此时也没有人留心，因为此时所有人的心都被那只飞出货轮的背包所牵动着。
<br/>　　看着即将要落入海水的背包，罗昊迅速丢掉自己手中的武器，冲向已经飞出货轮的背包，纵身一跃，扑向下坠的背包。
<br/>　　罗昊的手勾住了背包，一把将背包抱在怀中，而他自己却是坠入了海中，溅起数尺高的水花。
<br/>　　游艇上，那个吸着粗雪茄的佣兵队长看着罗昊落水的位置，眼睛眯起，在思考着什么。
<br/>　　P病毒现在在黑市上是有市无价，西伦科特当时联系他的时候，就是以一瓶P病毒作为代价。
<br/>　　“Boss，我们要怎么办？”一名雇佣兵轻声向佣兵队长问道。
<br/>　　“撤！”佣兵队长没有被巨大的利益冲昏头脑，扔掉手中的烟头，冷静地说道：“P病毒我们拿不到了，就是拼死抢到了手里我们也拿不住，特种部队会对我们展开无休无止的追上，我们甚至根本靠近不过去，只要我们的游艇接近那艘货轮，头顶上那两架阿帕奇立刻就会用导弹击沉我们。”
<br/>　　问话的雇佣兵立刻点头将佣兵队长的决定吩咐下去，游艇立刻再次发动，掉头离开，没有丝毫的犹豫。
<br/>　　“昊子！”
<br/>　　“狂龙！”
<br/>　　“Alpha1！”
<br/>　　“……”
<br/>　　货轮上的人快速冲到船舷边，朝着海面张望，叫喊。
<br/>　　没过一会儿，还没彻底平静的水面出现了波动，紧接着抱着背包的罗昊冒出了水面，货轮上，周柏立刻扔下了绳子，罗昊抓住绳子游向货轮并且爬上了甲板。
<br/>　　而这时詹姆斯所搭乘的直升机也开始降落，直升机还没完全落下，詹姆斯就跳下了直升机，朝着人群的方向走来。
<br/>　　爬上甲板后，罗昊第一件事情就是打开手中的背包，检查背包中P病毒的情况，可是当罗昊打开背包后，瞬间愣住了。背包中只有两瓶P病毒！
<br/>　　罗昊抹了一把脸上的水，开始飞速思考起来，背包被西伦科特扔出货轮，被自己抓住之后就一直抱在自己怀中并没有打开过，所以不可能有P病毒掉落进海里的情况发生。发生现在这种情况，只有两种可能，西伦科特在登上这艘货轮的时候，将其中一瓶P病毒交给了别人，第二种可能就是从一开始就只有两瓶P病毒，山顶别墅地下室冷藏柜中那三个凹槽影响了罗昊的判断。
<br/>　　此时罗昊也拿不准到底是怎么回事。
<br/>　　就在这个时候，詹姆斯走向罗昊，看到背包中那两瓶P病毒的时候，眼睛微不可察的眯了眯，随后对罗昊说道：“Alpha1，你做得很好，现在把P病毒交给我带回去吧。”
<br/>　　听到詹姆斯的话，龙隐部队这边的人脸色都是一变，李然上前一步挡在詹姆斯面前，看着他说道：“怎么，将军想就这么带着两瓶P病毒离开吗？”
<br/>　　“难道有什么问题吗？”詹姆斯耸了耸肩膀，对自己提出的要求显得十分理所当然。
<br/>　　“你一开口就想要把两瓶P病毒全都带走，难道不觉得太过分了吗？”罗昊走上前，眯着眼看着詹姆斯冷声说道，说话的口气也不再有之前那么客气。
<br/>　　詹姆斯冷眼看着罗昊还有其他龙隐部队队员，轻笑一声，显得十分不屑，“你们除了答应我的要求，难道你们还有其他的选择吗？别忘了，你们的人现在还在联合行动司令部中，而其中就有罗队长的妻子。”
<br/>　　“威胁我？”罗昊眼中寒光闪烁。
<br/>　　“你可以这么理解。”詹姆斯伸出手，对着罗昊说道：“现在可以把东西给我了吗？你应该知道的，你们除此以外没有别的选择。”
<br/>　　说着詹姆斯指了指李毅峰和慕容飞所在那架“黑鹰”直升机，他们两个人已经被飞机上的美军士兵所控制，身上的武器也已经全都被卸掉。
<br/>　　“这两瓶P病毒我要了。”
<br/>　　就在这个时候，一道冷漠的声音从詹姆斯背后响起，紧接着詹姆斯觉得自己脖子上一凉，一把锋利的军刀架在他的颈动脉上。
<br/>　　詹姆斯侧脸看了眼这个神不知鬼不觉出现在自己身边的人，眼中几乎要喷出火来，咬牙切齿的说道：“幽灵刺客，蛰龙！”
