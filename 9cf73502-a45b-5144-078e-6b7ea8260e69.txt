　　第738章
<br/>　　程大雷在清风寺内忙碌，按部就班的执行自己的脱身计划，心中也并不如何着急。
<br/>　　而暂时被困清风寺的程大雷，并不知道现在外面发生了什么事。
<br/>　　随着他的出现，京州一地草木皆兵。由汝南城到长安城画一条线，这条线上的各家势力都如临大敌。
<br/>　　大家想的都是同一件事，一是程大雷有没有可能走出汝南城，二是假若他从自家门前路过，大家该如何应对。
<br/>　　能将其杀了最好，在整个帝国也是扬名立万的事情。但如果杀不了？过往的事情告诉大家，与他为敌从来没有什么好下场。
<br/>　　京州，平野城。
<br/>　　平野城位于京州腹地，这里土地肥沃，水源充沛，向来就有京州粮仓之称。
<br/>　　经营此地的是平野陈氏，陈家老祖宗是随着李氏太祖一起打江山的那批人。后来李氏坐了天下，陈家便被封在此地，他们也是为数不多愿意继续李氏皇族的豪门大族。
<br/>　　城主府内的演武场上，一年轻人赤着上身，在院中演练武艺。肌肉虬起，汗水顺着后背流淌下来。他的兵器是一对宣花斧，每只斧头各有三十八斤，在他手中使起来虎虎生风。
<br/>　　“梦儿。”
<br/>　　一个中年华服的男人出现在演武场，听到他的招呼后，年轻人又走了一趟把式方才停住身形，立在那里，胸脯微微起伏，汗水如撒花般砸在地上。
<br/>　　这华服男人便是平野城的城主，名叫陈霸祖。而这用斧头的年轻人便是他的小公子，有平野虎之城的陈梦。
<br/>　　将一副白毛巾丢给自己儿子，陈霸祖开口道：“你听说什么了没有？”
<br/>　　“那人要来了。”陈梦说着脸上挂出一抹狰狞的笑：“早就想碰碰他了，帝国很大，但容不下另一个用斧头的人。父亲放心，我不会让他活着离开平野城。”
<br/>　　陈霸祖摇摇头：“这一次你不许出手。”
<br/>　　“为何？”
<br/>　　“其中牵扯太多，并非一句话两句话能说清楚的。你只要记得，程大雷这次死在汝南城最好，如果他活着来到平野城，你不要动手，放他过去，反正前面还有人在等他。”
<br/>　　陈梦仰起头：“有我在，他便不可能从这儿过去。”
<br/>　　陈霸祖道：“这几天你就不用出门了，在府里好好操练武艺。世道很乱，以后有你出头的机会。”
<br/>　　说罢，陈霸祖迈步离开，陈梦盯着他的背影，眼里是满满的不服。
<br/>　　不得不说，程大雷是很抢风头的一个人，自他出世以来，很多年轻人都被压着出不了他。
<br/>　　那些年轻人明明一身本事，却没有出头的机会，提起程大雷便是咬牙切齿的恨。
<br/>　　这陈梦便是其中之一，号称平野虎，双臂有千钧之力，把式用得也熟。他还想靠着这次机会扬名露脸。陈霸祖的话未必不对，但他为什么要听呢。
<br/>　　站在原地心急如焚，心中急速思索着。忽然道了一声不好，程大雷纵然本事很大，可他也未必能走出汝南城。真如果他死在汝南城，自己哪还要与他交手的机会。
<br/>　　陈梦披了外套，将两把斧头绑在背上。事情宜早不宜迟，自己必须尽快赶到汝南城。如果程大雷还没走出汝南城，自己就帮他出来；如果程大雷已经闯出来了，自己就在众目睽睽之下杀了他。
<br/>　　不敢惊动府中人，他牵了坐骑，从后门溜走。结果刚走出马口，就被几个家丁堵住了。
<br/>　　大家看着陈梦：“公子，老爷让我们在这里等你。”
<br/>　　陈梦挠挠头，表情也有些无语。
<br/>　　……
<br/>　　消息比人走得快，程大雷还困在汝南城的时候，他出凉州的消息已经传到长安。传入皇宫中李乐天的耳中。
<br/>　　无论如何，李乐天是不希望程大雷出事的。虽然掌了军权，但毕竟只是京州一地，李乐天也是孤木难支。他现在很需要一个盟友，想来想去，程大雷是合适的。
<br/>　　双方各取所需，李乐天可以给程大雷名正言顺，而程大雷拥兵凉州，也能给予李乐天一定支持。
<br/>　　但如果程大雷死在来长安的路上，自己便没办法和他同盟。毕竟孤木难支，现在京州还有许多世家豪族对李乐天阳奉阴违，有些连阳奉阴违都不屑于去做。所以没有程大雷，李乐天也未必玩得转。
<br/>　　他差人把马孟起招呼到宫中，马超依旧是从前打扮，野性难训。关于如何安排他，李乐天有些头疼，不过无论如何，李乐天知道他是可以信任的。
<br/>　　“你点一千兵马去汝南城，将程大雷护送到长安。”
<br/>　　“程大雷？”马超睁大眼睛。
<br/>　　“听说过？”李乐天。
<br/>　　“当然听说过，据说很厉害。”马孟起磨拳霍霍，眼神炙热。
<br/>　　既然这个人如此了得，那自己可要碰一碰他，与他论个高低。
<br/>　　“你去可是去，可不要胡乱出手，把他平安护送到长安，你就算是完成了任务。”
<br/>　　“好好好，你还信不过我么。”
<br/>　　李乐天白了他一眼，马超的确可要信任，他是少数李乐天可要托付的。但对于他的脾气，做事风格，李乐天委实不太信得过。
<br/>　　马超嘻嘻哈哈一声，在他眼里也无所谓的君臣大礼，出了宫，点了一千鱼龙卫直扑汝南城。
<br/>　　关于程大雷这三个字，马超听说过不止一次。这次入京的藩王不少，程大雷倒是被提起最多的，大家显然很在意他。马超耳朵都磨起茧子，在他心里想来，这次无论如何，也要和程大雷碰一碰。
<br/>　　将他带回长安城没有问题，但必须先让他败在自己枪下，然后求自己保护他带回长安。
<br/>　　一千兵马赫赫出京，奔赴汝南城。
<br/>　　于此同时，平野虎陈梦也是好不容易混出了城主府，继而混出了平野城。之后，再也不藏着掖着，策马狂奔，就想早些赶到汝南城。
<br/>　　免得时间拖久，程大雷已经被魏牧冲杀了，那自己这番力气，岂不是白费了。
<br/>　　程大雷目前还算安全，依旧窝在清风寺，朝九晚五的执行自己的计划。
<br/>　　却也不知道，有许多人已经奔着自己而来。
