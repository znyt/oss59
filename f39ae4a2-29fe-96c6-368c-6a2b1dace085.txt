　　天际之上。
<br/>　　一道陨石般的流光，以肉眼难见的速度，疾速横跨着。
<br/>　　半个时辰后。
<br/>　　流光停下，露出了横天殿主魁梧的身影。
<br/>　　同时，身旁还有一道狼狈的身影。
<br/>　　身影，衣发凌乱，脸上尽是狼狈之色。
<br/>　　身影，自是萧逸。
<br/>　　而此刻的萧逸，除了满脸狼狈之外，还有满脸无奈。
<br/>　　别的前辈，以这般疾快速度飞行下，一般都会替后辈抵挡疾速下的罡风。
<br/>　　如承风前辈。
<br/>　　虽每次都忽然带着萧逸疾行，吓萧逸一跳。
<br/>　　可萧逸每次都仅仅是落地的那一刻，脚步有些不稳，除此以外，再无半分不妥。
<br/>　　而横天前辈，刚才带着他横跨了不知多少百万里路，却一直自顾飞行。
<br/>　　萧逸曾试过凝聚元力护身。
<br/>　　但以他实力下，凝聚的澎湃元力，却仅仅是凝聚瞬间，便在那疾速产生的风速下溃散。
<br/>　　“呼。”萧逸松了口气，整了整宛若刚进行过一场激烈大战般的凌乱衣衫。
<br/>　　一旁，横天殿主看了眼萧逸的狼狈身影，摇了摇头。
<br/>　　“小子，你就这点本事？”
<br/>　　“才飞那一会儿就这般模样，真让你去杀敌大战，岂非转瞬就陨落？”
<br/>　　萧逸闻言，翻了个白眼，懒得理会。
<br/>　　这横天殿主的实力之恐怖，单凭速度便能看出来。
<br/>　　此人的实力，绝对远在北隐宗大长老那等武道大能之上。
<br/>　　“好了，随我来。”横天殿主看了萧逸一眼，眼眸中闪过一丝不耐烦。
<br/>　　在他看来，总殿的传承考核，许多年才开启一次。
<br/>　　有的时候，若殿内的年轻天骄达不到要求，凑不足十人的话，甚至百年都难开启一次。
<br/>　　可每次开启，必是殿内天骄趋之若鹜，立刻赶来。
<br/>　　唯独这一次，总殿连下多道传令，都未能将萧逸唤来，甚至还要自己这个德高望重的前辈亲自去请人。
<br/>　　在横天殿主看来，萧逸无疑是个狂妄到了极点的人。
<br/>　　狂，并不一定是坏事。
<br/>　　可狂到目中无人的地步，就过分了。
<br/>　　这时，萧逸跟着横天殿主前行。
<br/>　　如无意外，应该是与其他九位修罗殿天骄汇合，一同开启传承之地。
<br/>　　萧逸边走，边打量着周遭。
<br/>　　这里，是修罗殿总殿内堂。
<br/>　　其格局和大小，与风刹殿差不多。
<br/>　　顺带一提，修罗殿总殿与风刹殿总殿的距离，并不多远。
<br/>　　起码放眼偌大个中域来说，这并不远。
<br/>　　两大总殿，几乎是紧挨着。
<br/>　　毕竟，两殿乃是联盟之殿。
<br/>　　当然，所谓不远，只是相对而言。
<br/>　　横天殿主横跨半个时辰的距离，怕是萧逸要全速飞行十天以上，甚至更久。
<br/>　　数分钟后。
<br/>　　二人前行的脚步，忽然停下。
<br/>　　前方，出现了一负手而立的老者。
<br/>　　强横如横天殿主，见了老者，连忙行礼，“参见殿主。”
<br/>　　“殿主怎会…”
<br/>　　老者摆摆手，示意横天殿主不必多言。
<br/>　　萧逸见状，亦连忙行礼，“参见总殿主。”
<br/>　　同样的，能让横天殿主躬身行礼，称一声殿主的，这世间，也只有修罗殿总殿主。
<br/>　　自然的，萧逸一眼认出了老者的身份。
<br/>　　另外，面前老者给他的感觉，与风刹殿总殿主一模一样。
<br/>　　如果说，风刹殿总殿主给他的感觉，是如天之飘渺；那么，面前老者给他的感觉，便是如整个大地般厚重。
<br/>　　面前老者的肉体力量，早已达到一个难以想象的地步。
<br/>　　萧逸甚至觉得，即便是这天地万道，亦不及老者的一拳。
<br/>　　老者那看似不如横天殿主般魁梧的身躯，却仿佛一座高不可及，横亘天地的亘古高山。
<br/>　　“萧逸是吗？”老者的语气，很是严肃。
<br/>　　不，准确来说是，老者浑身上下都散发着严肃的气息。
<br/>　　那威严的面庞下，即便只是平淡地说着话，亦肃然无比。
<br/>　　“是。”萧逸点点头。
<br/>　　“嗯。”老者点点头，“不必去参加传承了，直接去闭关室吧。”
<br/>　　“殿主，这…”横天殿主一愣。
<br/>　　老者威严的面容上，闪过一丝微微赞赏。
<br/>　　“能轻松参悟寂空剑道，得到整份传承的妖孽，再去参悟别的传承，也只是浪费时间。”
<br/>　　“还不如多放些时间到修炼上。”
<br/>　　“横天。”老者看向横天殿主，“剩下九位天骄的传承开启，由你主持，去吧。”
<br/>　　“这…”横天殿主迟疑了一下，又瞪了眼萧逸。
<br/>　　随后，还是点点头，道，“是。”
<br/>　　横天殿主离去。
<br/>　　原地，只剩老者与萧逸。
<br/>　　同样的，萧逸感到浑身不自在。
<br/>　　不过，对比之前初见风刹殿总殿主时，此刻却是镇定了许多。
<br/>　　“不错。”老者看着萧逸，点了点头，面露赞赏之色。
<br/>　　随后，老者脸上表情散去，化作了一贯的严肃。
<br/>　　“风刹殿那边，已将你的卷宗资料传于我手上。”
<br/>　　“你的天赋，不必再去考核了。”
<br/>　　“接下来，伸出手即可。”
<br/>　　“总殿主想…”萧逸迟疑了一下。
<br/>　　老者打断道，“若你不介意的话，我要查探你体内情况。”
<br/>　　萧逸迟疑着，没有动作。
<br/>　　武者的身体，可是武者自身的秘密，等闲绝不可能让外人查探感知。
<br/>　　老者皱了皱眉，“我若不查探你的肉体情况，不知你修的是何等体修功法，如何给你传承？”
<br/>　　说着，老者手指轻点。
<br/>　　空气中，一道氤氲光芒闪烁而过。
<br/>　　一本本功法武技书籍，凭空而现。
<br/>　　“罢了。”老者肃然道，“你既不愿，我也不逼。”
<br/>　　“你自己的身体，你自己最清楚。”
<br/>　　“自己选一本吧，随后自行去闭关室闭关即可，闭关室内，已准备好修炼之物。”
<br/>　　说罢，老者转身离去。
<br/>　　忽然，老者脚步顿了顿，“对了，出关之后，来寻我一趟。”
<br/>　　说罢，老者自顾离去。
<br/>　　萧逸站于原地，微微行了一礼。
<br/>　　随后，看向空气中漂浮着的这一本本书籍。
<br/>　　书籍上，所散发着的古老气息，证明着它们的不凡。
<br/>　　萧逸大致看了一遍，脸色一惊。
<br/>　　这一本本功法武技，随便一本，都是圣阶巅峰以上。
<br/>　　最差的一本体修功法，都远在雷罡战体之上。
<br/>　　萧逸倒吸了一口凉气。
<br/>　　十数分钟后，萧逸选好了一本功法。
<br/>　　当他拿起书籍的一瞬间。
<br/>　　哗…空气中，其余的一本本书籍，悉数化作氤氲云烟，不复存在。
