　　天地悠悠，时光如怅，岁月转瞬又是数万年过去了。
<br/>　　这一天，元始正在与两兄弟论道喝茶，忽然，他抬起头，看向邻里西昆仑方向，莫名其妙的说了一句：
<br/>　　“鸿钧要成圣了，圣人时代即将来临！”
<br/>　　“什么？”
<br/>　　身旁两人不明其理，感觉元始这话若有所指，但他们又不能弄个明白。但是下一瞬，他们就知道了这话的意思了。
<br/>　　下一瞬，天地之间传来一股强大的威压，在这威压之下，洪荒生灵全部都有一种对抗天威的感觉，所有的生灵都全部跪伏在地，身影挣扎。大神通者感觉这威压更胜，因为他们可以感受到这股威压本身并不是法力交织而出，而是一种气势，更确切的说，那是一种境界上的差距，就好似以大罗金仙的的境界去压一个小小的天仙。
<br/>　　只是几个呼吸的时间，能坚持一个呼吸与坚持两个呼吸，都有着巨大的差距。三个呼吸过去了，洪荒之中还没有跪倒的只有那么寥寥无几的几人，这几人都是日后在洪荒之中大放光彩的人物。
<br/>　　第四个呼吸，火云洞红云老祖暗叹一声，跪倒在地，又是一个呼吸过去了，万寿山镇元子也撑不住，跪倒在地，在第六个呼吸来临之际，血海冥河身边的元屠阿鼻两件一阵呜呜的轻鸣，冥河瘫倒在地，与此同时，北冥宫的鲲鹏老祖亦倒塌在地。
<br/>　　第七个呼吸即将到来之时，帝俊太一头顶的河图洛书，混沌钟亦无力地落下，这两个妖族的皇者也不得不屈服了，同时不周山的伏羲眼中亦闪过一丝落寞，跪在地上。
<br/>　　此刻，能坚持下来的，只有日后成就圣人的几位了。
<br/>　　第七个呼吸刚到，西方灵山准提怒吼一声，却也是坚持不住，同时，不周山的女娲也是面色苍白，跪倒在地。
<br/>　　第七个呼吸刚要结束，西方接引道人以无法坚持，跪伏在地。
<br/>　　第七个呼吸，就如同一个收割机，一下子就将洪荒之中的大神收割完毕。如今剩下还能坚持的，就只有昆仑山三清圣地的三清了。
<br/>　　却只见元始三人，元始头顶一个不周神印，对抗天地威压，鸿钧成圣的威压丝毫不得近元始身边，元始面色不改，端起茶杯，喝了一口，小脸之上笑眯眯地看着眼前还在努力对抗圣人威压的老子与通天。
<br/>　　只见老子头顶天地玄黄玲珑宝塔，丝丝玄黄之气散布下来，形成一个领域，对抗天地威压。通天手持青萍剑，剑意迸发，对抗天威，剑意不散，意志不倒，尽管他头顶大汗直冒，但他仍不放弃。
<br/>　　第八个呼吸过去了……
<br/>　　第九个呼吸过去了……
<br/>　　当第十个呼吸来临之际，突然，威压尽散，九天之上紫气回荡，天音飘飘，万花朵朵，馨香四溢，万兽齐鸣，道道仙音扶摇，玄之又玄。
<br/>　　天空之上，风云倒卷，更有万丈祥云浮空而起，天坠金花，地涌金莲，自大地之下涌起，顺着东部方向，一股浩瀚的紫气绵延无尽，遮蔽九霄，苍穹震动，日月星辰摇摆，无尽生灵恐惧莫名，朝着东方膜拜下去。
<br/>　　天地演化出条条大道，一时之间洪荒大地之上，草木凡兽，奇珍异宝数不尽的生灵被点开灵智，有的更是因此得以化形蜕凡。
<br/>　　“嘶！如此异象，紫气东来三万里，祥云万丈，瑞气浮空，地涌金莲，究竟发生了何事？！”
<br/>　　这一刻，除了元始之外的，大罗金仙至强之修，如老子、通天、女娲、镇元子……等人，一个个倒吸一口气，眼皮狂跳。
<br/>　　而就在此时，无数修士只看到，洪荒东部，一座仙山道场之处，一道巨大的圣道光芒，冲天而起，此光恢宏，撼动苍穹，滚滚威严扩散四面八方，紧接着，众修士可以看到，在那圣光之中，有一个面貌奇古，穿着玄色道袍的老者出现。他踏空而起，向着三十三天外而去，与此同时，九天之上，一道浩荡的大道歌声传来：
<br/>　　“高卧九重云，蒲团了道真。
<br/>　　天地玄黄外，吾当掌教尊。
<br/>　　盘古生太极，两仪四象循。
<br/>　　玄门都领袖，一气化鸿钧。”
<br/>　　“吾为鸿钧，今日证道成圣，百年之后，将于三十三天外紫霄宫中传道，有缘者皆可来。”
<br/>　　这个道者，不是鸿钧，又会是谁？此刻，他已成圣，却是向着三十三天外而去，将于那混沌之中开辟紫霄宫道场。
<br/>　　“鸿钧道人？成圣了！百年之后在紫霄宫讲道？”众生大惊，都在猜测这鸿钧道人乃何许人也，竟有如此威势，而圣人又是什么？
<br/>　　“咻！”
<br/>　　“咻！”
<br/>　　“咻！”
<br/>　　……
<br/>　　从洪荒各处冲出无数流光，虽然大多数生灵不知道发生了什么事儿，不过，许多大神通者都知道这是天大的机缘，不容错过，立刻从闭关之中苏醒，有的驾着祥云，有的化作长虹朝着混沌方向飞驰而去。
<br/>　　“大兄，我们去不去？”
<br/>　　老子两人将目光投向元始，想让他拿主意。
<br/>　　“去啊，既然有证道成圣者讲道，这么好的事儿怎么能不去？去取点真经也是好的，再说，就算取不到什么真经，拿点好处也是应该的，一个成圣者，应该有不少宝贝吧。”
<br/>　　元始自然是不会不去，他可是知道，他们三兄弟是注定的天道圣人，此刻不去，那不是出来变数吗？
<br/>　　老子与通天都翻了翻白眼，没有将元始后一句话放在心上，你去就是为了人家的宝贝？不过，听到要去听道，两人也觉得应该前去。
<br/>　　三十三天外，是无穷无尽的混沌世界！
<br/>　　洪荒众强者来到这茫茫不知边际的地方，这色彩永远属于灰蒙蒙的混沌。
<br/>　　这便是混沌！
<br/>　　天地乃是盘古从混沌之中开辟而出的，而开辟混沌的盘古大神，便是这混沌所孕育而生。
<br/>　　三千混沌魔神秉承三千大道法则孕育而生，天生掌控法则，为大道所眷顾。
<br/>　　在这混沌之中，除了三千混沌魔神能够自由的行走生活，其他任何生灵只要进入这混沌，便会被那最为原始的力量，混沌之气，侵蚀同化，不消一时三刻，便会被同化为苍茫混沌之气中的一份子。
<br/>　　此刻，那些前往混沌之中听道的修士们，正在陆陆续续地进入这混沌，只是这大道又岂是那么容易听到的。鸿钧将这道场设在混沌之中，自然是有他一分深意的，说是有缘者皆可前来，可这有缘有说的是什么呢？
<br/>　　将道场设在混沌之中，那就只有一个目的，那便是你有足够在混沌之中自保的修为，才可在这混沌之中活下来，能活下来还不够，还要在这茫茫混沌之中找到紫霄宫，才算有缘。此时，你自然是有缘人了，你才可在紫霄宫中听讲圣人之道。
<br/>　　只是这些修士还不知道这混沌的厉害，混沌又岂是这般容易通过的？
