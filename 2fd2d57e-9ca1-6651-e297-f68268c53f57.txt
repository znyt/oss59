　　苏莫心中杀意凛然，他对于敌人，从来不会心软。
<br/>　　不说太史轩派人追杀他之仇，现在他回到了神武学府，对方肯定也不会罢手。
<br/>　　毕竟，从派人追杀他那一刻起，双方就变成了死敌。
<br/>　　“走，我们去找他！”苏莫对聂乾说道，言罢，便转身向外走去。
<br/>　　“苏兄且慢！”
<br/>　　聂乾见此，急忙起身拦住了苏莫，沉声道：“苏兄，我之前不告诉你原因，就是怕你冲动，那太史轩不能得罪！”
<br/>　　“如何不能得罪？”苏莫淡淡的问道。
<br/>　　“太史轩是太史世家一脉的人，而且还是太史世家的嫡系子弟，势力庞大，我们惹不起！”聂乾语重心长的说道。
<br/>　　“你就能吞下这口气？”苏莫皱眉问道。
<br/>　　“不吞下又能怎样？”聂乾叹了口气，他被太史轩打伤，心中自然也是无比愤怒，但即便再愤怒，他也只能吞下这口气。
<br/>　　“苏兄，还是算了吧！”
<br/>　　这时，韩盈的声音响起，只见对方从洞府之外走了进来。
<br/>　　显然，韩盈听到了动静，才走出了自己的洞府。
<br/>　　“还记得我们上次离开学府，遭到两名武者的追杀吗？”苏莫望着两人，沉声问道。
<br/>　　“嗯！”两人点了点头。
<br/>　　“那两人就是太史世家一脉的人，奉太史轩和尤天韩之命追杀我们！”苏莫道，此事他之前并未告诉聂乾和韩盈。
<br/>　　“什么？”聂乾两人闻言一惊，随即纷纷脸色难看了起来。
<br/>　　之前太史轩逼问他们赵城和郭剑青之事，他们已经知道两人是太史世家一脉的人，但没想到却是有预谋的追杀。
<br/>　　“太史轩不会放过我们，所以，我们要率先出击！”苏莫冷漠的说道。
<br/>　　“可是……太史轩身份不凡，背景深厚，我们得罪不起啊！”聂乾和韩盈脸色变换，心中纠结不已。
<br/>　　他们知道苏莫实力强大，战胜太史轩问题不大，但是以后呢？如何能挡住太史世家一脉的报复？
<br/>　　若是只为了出一口气，而最终赔上性命，那就太不值了！
<br/>　　聂乾很韩盈两人，天赋在神武学府非常一般，又没有什么背景，平常都是谨小慎微，对于那些有背景有实力的弟子，均是不敢得罪。
<br/>　　这倒不是说两人胆小怕事，而是不想招惹无端的麻烦。
<br/>　　“得罪不起也要得罪，不杀他，他就会杀我们！”苏莫淡淡的说道，言罢转身向外走去。
<br/>　　聂乾和韩盈见此，相视一眼，随即急忙跟了上去。
<br/>　　随后，苏莫带着聂乾和韩盈，离开了火元峰。
<br/>　　到达外府南苑之后，苏莫直接向太史轩的住处飞掠而去。
<br/>　　不多时，他们便来到了太史轩的院落之外。
<br/>　　“太史轩，给我滚出来！”望着眼前精美大气的院落，苏莫立刻一声暴喝。
<br/>　　声雷滚滚，震荡四野，直接在院落上空炸开。
<br/>　　这一下，不仅是惊动了太史轩的院落，整个南苑都惊动了。
<br/>　　嗖嗖嗖！！
<br/>　　破空声响起，太史轩的院落中道道人影冲天而起，十几名青年直接飞到了院落上空。
<br/>　　十几名青年均是太史世家一脉的人，而太史轩正在其中。
<br/>　　“苏莫！”太史轩看到院落之前的苏莫，顿时眸中杀机一闪。
<br/>　　“太史轩，战界一决生死，敢否？”苏莫目光冷漠，直视太史轩，直接下达了战书。
<br/>　　“什么？”太史轩和一众太史世家一脉的人，闻听此言，顿时心中一惊，苏莫居然敢向太史轩挑战？
<br/>　　而一些被惊动的南苑弟子，远远的听到苏莫的话语，也俱是心中震惊。
<br/>　　众人仔细打量了苏莫一番，顿时心中惊讶，此刻的苏莫，居然已经达到了武帝境四重境界。
<br/>　　两年前的时候，苏莫不是才突破到武帝境界吗？
<br/>　　短短不到两年的时间，居然连升四重境界？
<br/>　　众人心惊不已，武者到了武帝境之后，修为晋升的速度大减，就算是绝世天才，想在一两年之内，提升一重修为境界，都非常的困难。
<br/>　　而苏莫，两年时间居然连升四重，怎么修炼的？
<br/>　　难道苏莫天赋如此可怕，或者是得了天大的奇遇？
<br/>　　苏莫身后的聂乾和韩盈，见苏莫上来就要与太史轩一决生死，亦是心中一震。
<br/>　　他们也是没想到，苏莫居然如此狂霸，直接就是生死一战，不留任何调解的余地。
<br/>　　“武帝境四重？”太史轩双眸微眯，随即冷笑一声道：“难怪敢如此狂妄，原来是有了如此修为境界！
<br/>　　“敢否？”苏莫面无表情，再次出声。
<br/>　　“既然你自己找死，那就怨不得我了！”太史轩森然一笑，他可是武帝境五重境界，而且战力强大，一般的武帝境六重武者都不是他对手，他怎么可能会惧怕苏莫？
<br/>　　“我在战界等你！”苏莫淡淡的留下了一句话，便转身向通宇大殿飞去，而后通过大殿内的入口，进入了战界之中。
<br/>　　“看我不虐死你！”太史轩嘴角勾起一丝笑意，便也立刻向战界而去。
<br/>　　其他太史世家一脉的人，以及很多被惊动的南苑弟子，亦是纷纷跟了上去。
<br/>　　苏莫挑战太史轩，如此战斗，怎能错过！
<br/>　　被惊动的南苑弟子，数量极多，人数接近万人，浩浩荡荡的飞进通宇大殿，将大殿中坐镇的导师都震惊了。
<br/>　　战界。
<br/>　　与以往没有任何不同，依旧是无尽的海水，十几座岛屿屹立八方为看台。
<br/>　　嗖！
<br/>　　苏莫进入战界之后，便直接向十几座岛屿的中央飞去。
<br/>　　此刻，战界中并没有什么人，因为无人战斗。
<br/>　　唰！
<br/>　　瞬息之间，苏莫飞到了十几座岛屿的中央，身形轻飘飘的落在了海面上。
<br/>　　而后，他转头看向后方，静等太史轩的到来。
<br/>　　聂乾和韩盈两人，一直跟着苏莫，他们飞到了周围的一座岛屿上，面上布满了忐忑。
<br/>　　他们知道苏莫的实力，绝对不弱于太史轩，但是若真的杀死了太史轩，怕是要捅了马蜂窝了。
<br/>　　嗖嗖嗖！！！
<br/>　　苏莫没有等多久，只是两个呼吸的时间，太史轩便来了，其身后还跟着大量的南苑弟子。
<br/>　　甚至一些另外三苑的弟子，见到如此阵势，也纷纷前来观看。
