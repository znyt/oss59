        每次到自己母亲的院子里，母亲冷夫人就在催傅夜沉和苏漫雪去把结婚证给领了，然后早些给她生个大胖孙子。
<br/>
<br/>        一提到苏漫雪，傅夜沉就是一肚子的火气，却只能假装若无其事。
<br/>
<br/>        冷夫人拉着傅夜沉的手感慨道：“我第一眼见小雪那孩子，就觉得小雪像你干妈的那个孩子。对了，小沉，你还记得你干妈给你生的那个小妹妹吗？那小家伙刚出生那天，你就爱不释手的抱在怀里亲她，还说要她以后给你当老婆。”
<br/>
<br/>        “不记得了。”傅夜沉淡淡地应了声，接着说道，“妈，您觉得苏漫雪像干妈的孩子，那筱颖岂不是更像？”
<br/>
<br/>        苏漫雪可是照着筱颖的样子整容的，这件事情冷夫人虽然不知道，但他傅夜沉心里可是一清二楚。
<br/>
<br/>        “那个筱颖怎么能跟我家小雪比！还是我家小雪漂亮！也不知道你弟弟小景是抽了什么疯，结婚这么大的事情居然不和你爷爷说一声，就跟一个来历不明的女人领了结婚证！”冷夫人没好气地呵斥道。
<br/>
<br/>        傅夜沉却帮着傅昼景说话：“妈，阿景有他自己的想法。”
<br/>
<br/>        “也对！不扯小景的事情了！说说你干妈吧！哪天，我跟你干妈约个时间，让她从国外回来，见见小雪这个干媳妇。哎，其实你干妈也是个可怜的女人。”冷夫人突然叹息道。
<br/>
<br/>        傅夜沉孝顺地安慰道：“妈，您不要太伤感了。对身体不好。”
<br/>
<br/>        “也是。只是觉得可惜……如果那孩子没被坏人抱走，你干妈一家子或许不会移民去国外，说不定，你和那孩子还是青梅竹马。”冷夫人说着说着，眼泪就出来了。
<br/>
<br/>        虽然不是自己的孩子被坏人抱走，但是当年，她看着自己的闺蜜哭得撕心裂肺，最后还差点送进精神病医院，她这当“姐妹”的也是跟着心力憔悴。
<br/>
<br/>        傅夜沉知道自己的母亲一直想生个女儿，可惜没机会了。
<br/>
<br/>        “小沉，你一定要好好地善待小雪。”冷夫人千叮万嘱道。
<br/>
<br/>        傅夜沉却只是答非所问：“妈，你早点休息，我先回偏院去了。”
<br/>
<br/>        “好！”冷夫人慈祥地点了点头。
<br/>
<br/>        傅夜沉从母亲那儿离开后，刚走回到偏院，一走进到大门口，就见庭院灯的橘色灯光下，童筱颖手里提着木桶，正在给院子里的草木浇水。
<br/>
<br/>        傅夜沉依稀记得，这些草木应该枯萎好久了，现在再看那些草木，一株株绿意盎然。
<br/>
<br/>        “这么晚了，还不进屋去休息吗？”傅夜沉冷声问了一句。
<br/>
<br/>        童筱颖听到他的声音，侧身抬眸，目光在半空中与他不期而遇。
<br/>
<br/>        “大哥，晚上好。”童筱颖落落大方地微笑着问候。
<br/>
<br/>        看来，经过这段日子的相处，她是真的已经把他当自己的哥哥看待了。
<br/>
<br/>        比起以前，他每次见她，都会强吻她，甚至还想把她压在身下爱抚。现在，他对她要矜持了很多。
<br/>
<br/>        更确切地来说，是他自从知道她已经成为了他的弟妹之后，就再也没有对她有过非分之想了。
<br/>
<br/>        就在傅夜沉礼貌性地点了下头，准备转身进屋的时候，身后童筱颖的声音又叫住了他。
<br/>
<br/>        “大哥，等等！”
<br/>
<br/>        童筱颖的呼唤，就像来自梦中那般亲切。
<br/>
<br/>        傅夜沉欣喜地转过身去，只见童筱颖端着一盆仙人掌朝他走了过来，并将手里的仙人掌递给了他。
<br/>
<br/>        “大哥，仙人掌能在夜间吸收二氧化碳，释放氧气。有助于睡眠。你把这盆拿屋里去放着吧！”童筱颖微笑道。
<br/>
<br/>        傅夜沉单手接过童筱颖递过来的仙人掌花盆，刻意隐藏着自己对她的感情，抿嘴一笑，淡淡地谢了声：“谢谢。”
<br/>
<br/>        童筱颖收回手，向傅夜沉挥了挥手后，便转身进了屋。
<br/>
<br/>        其实，她并非只送了傅夜沉仙人掌，就连苏漫雪屋里，她也送了。还有冷夫人和傅老太爷那儿，只不过，冷夫人和傅老太爷屋里，她送过去的是有助于睡眠的“长寿花”。
<br/>
<br/>        傅夜沉看着童筱颖离去的背影，又看了看手里的这盆仙人掌，心里虽然百般不是滋味，但是却很欣慰，同时也很羡慕阿景。
<br/>
<br/>        如果，哪天，阿景真如爷爷所说的那样，想要他的傅氏集团，那么阿景是否可以将筱颖让给他呢？
<br/>
<br/>        晚上各自在各自的屋内，那边童筱颖一如既往地复习功课，苏漫雪在屋子里给自己的脸部做保养，而傅夜沉则在看爷爷送给他的一些有关于公司高层管理的书籍。
<br/>
<br/>        童筱颖的功课复习到一半，手机突然响起来，听铃声不是傅昼景的专属铃声，这让她有点儿小失落。
<br/>
<br/>        不过，她还是拿起手机，看了一眼来电显示后，便接听了电话：“喂，嫂嫂。”
<br/>
<br/>        离开梁家已经有一个月了，但是，童筱颖却仍旧对哥哥诺司峻的女朋友唐倩兮的称呼改不了口。
<br/>
<br/>        “呜呜——筱颍——你哥哥他……我找不到他人了……”唐倩兮哭着向童筱颖叙说。
<br/>
<br/>        童筱颖惊怔了一下，慌忙问道：“到底发生什么事了？”
<br/>
<br/>        “我、我也不知道！我只知道，他那次不是和你一起回的老家嘛！然后，他和你一起回去后，确实又回我这边来了。但是，最近回来没几天，又回老家去了。我那天不还问了你，说你哥哥又回老家去的事情嘛！现在，这都一个月过去了。你哥哥也没回来，电话都打不通，通讯软件全都不在线，就连他公司那边，他也只是请了长假，不见人回来。我……我好担心，不知道该怎么办是好……呜呜——”唐倩兮说着说着，嗷嚎大哭了起来。
<br/>
<br/>        童筱颖顿时也替哥哥诺司峻感到担忧，慌忙说道：“嫂嫂，你先别哭，要不这样。我把我老家的地址给你，你先过去。”
<br/>
<br/>        “筱颍，你陪我一起去好不好？今晚就去？我已经苦苦熬了一个月了，这一分一秒都不想再熬下去了！我要找到你哥哥！呜呜——筱颍，求你了！”唐倩兮哀求道。
<br/>
<br/>        童筱颖被唐倩兮的哭声，哭得心都软了，于心不忍就答应了。
<br/>
<br/>        唐倩兮这才放下心来，止住了哭声。
<br/>
<br/>        童筱颖跟唐倩兮挂了电话后，又拨了一通电话给傅昼景。
<br/>
<br/>        傅昼景此时正坐在办公室里加班赶远东城军机部项目的设计图，这段时间，为了这个项目，他和季思妍一直在加班加点地忙活着。
<br/>
<br/>        所以，筱颖的这通电话来得不是时候。
<br/>
<br/>        “昼景，你可不可以，今晚抽空陪我和我嫂嫂唐倩兮回老家去一趟，我哥哥也就是诺司峻他……可能出了什么事。已经失踪了一个月了……”童筱颖心里其实很焦急，但在傅昼景面前，她尽量让自己保持冷静，语速放慢，稍稍温和一点。
<br/>
<br/>        但尽管如此，傅昼景听到她说这些，还是一脸不悦地皱起了眉头，反问：“你还回那去做什么？梁家跟你已经没有任何关系了不是吗？诺司峻也不是你的亲哥哥。而且，那个什么唐倩兮，是会所里的女人，没了你哥哥，她又会去另外找？你何必跟着她一起去那个什么老家，瞎搅和他们之间的事情呢？”
<br/>
<br/>        “她不知道我哥哥的老家很正常，毕竟那种地方穷乡僻野，作为男人不想让自己的女人知道自己出身低微有意隐瞒，也是人之常情啊！可我哥哥是真心待我嫂嫂的，我嫂嫂也是真心爱着我哥哥的。我哥哥出了事，我嫂嫂怎么可能不担心？”童筱颖顿时不满傅昼景所说的话。
<br/>
<br/>        梁家现在确实跟她一点血缘关系也没有，但是，撇开诺母对她尖酸刻薄不说，可诺爸却是含辛茹苦地将她拉扯长大，不是亲爸胜似亲爸，不是吗？
