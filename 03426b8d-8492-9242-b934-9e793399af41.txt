　　苏莫钻进了山洞之中，进入了血煞山的内部。
<br/>　　血煞山的内部，空间非常的宽敞，通道无数，四通八达。
<br/>　　浓郁的血煞之气，仿若血雾一般，充斥在山峰内部，让得通道中视距不足五丈。
<br/>　　苏莫走了片刻，便在一处通道的拐角处停了下来，然后盘膝而坐。
<br/>　　望着手中的血阳花，苏莫嘴角勾起了一丝笑容，这一关，他想不得第一名都不行了啊！
<br/>　　随即，苏莫尝试着吸纳了一缕血煞之气入体，然后又通过手掌注入了手中的血阳花种子之内。
<br/>　　而后，他手中的血阳花种子，微微的起了变化，从绿豆大小，变成了黄豆大小，颜色变得更加的血红。
<br/>　　这种血煞之气，根本不需要炼化，吸纳之后直接注入血阳花种子之内便可。
<br/>　　“果然如此！”苏莫暗暗的点了点头，随即不再迟疑，立刻催动了吞噬战魂吸纳血煞之气。
<br/>　　通道中浓郁的血煞之气，仿佛受到了牵引一般，滚滚涌动，化为一条血色长龙，向苏莫体内疯狂的冲去。
<br/>　　进入苏莫体内的血煞之气，全部通过他的手臂，注入了血阳花种子之中。
<br/>　　血阳花的种子，以肉眼可见的速度暴涨，片刻之间便长到了海碗大小。
<br/>　　而后，种子破裂开来，其上长出了一株血红色的幼苗。
<br/>　　血阳花发芽了！
<br/>　　苏莫一刻不停，吞噬战魂的催动力度稍稍加大，吸收的血煞之气更为快速。
<br/>　　随着血煞之气的不断注入，血阳花的幼苗越长越大，很快便长到了一人之高，足有手臂般粗细，成了一刻血色的小树。
<br/>　　而后，血色小树的顶端出现了一个花苞，越长越大，涨到拳头大小之时，便绽放开来，化成了一朵血色的花朵。
<br/>　　这花朵成圆形，仿若是一个血色的骄阳，通体绽放出血色的光芒。
<br/>　　这便是血阳花！
<br/>　　苏莫没有停止注入血煞之气，血阳花越长越大，很快直径便达到了一尺大小。
<br/>　　一尺大小，便是这第二关的标准，他已经达到要求了。
<br/>　　而这个时候，一个时辰的时间已经过去了近一半左右。
<br/>　　苏莫面现沉吟之色，暗暗思忖要将血阳花催生到多大的地步。
<br/>　　他一直是微微催动吞噬战魂，并没有大力的催动，所以才用了这么长的时间。
<br/>　　以他现在吸纳血煞之气的速度，可以参考到血阳花的生长速度，在苏莫看来，这一百八十余名参加第二关的人，能有一半人通过这一关考核，就算是不错了！
<br/>　　思忖了片刻，苏莫觉得自己必须让血阳花涨到一丈以上，方有夺冠的把握。
<br/>　　那巫人王、巫玛和巫幽兰，天赋极强，想必三人的血阳花，必定长的极大。
<br/>　　尤其是巫人王，还是双生战魂，虽然苏莫不清楚对方的战魂有多强，但想来肯定不会弱。
<br/>　　若是他的血阳花不够大，说不定被对方拔得头筹了。
<br/>　　想到这里，苏莫立刻开始大力的催动吞噬战魂，通道中的滚滚血煞之气开始疯狂涌动，向他汹涌而去，片刻间便将他包围成了一个硕大的血茧。
<br/>　　至此，血阳花开始快速生长，以肉眼可见的速度变大。
<br/>　　时间缓缓流逝，半刻钟之后，苏莫的血阳花便长的了一丈大小。
<br/>　　“还不够，再大点吧！”苏莫继续猛烈吞噬血煞之气，又过了半刻钟，血阳花已经超过了两丈，仿佛是一个巨大的血色太阳，非常的惊人。
<br/>　　这个时候，苏莫放慢了吸纳的速度，血阳花已经足够大了，但为了保证能夺得第一名，他还是没有停止吸纳血煞之气。
<br/>　　旁边的一座山洞之中。
<br/>　　巫天御身上战魂已经释放了出来，强大的飓风战魂疯狂的卷动，将周围的血煞之气全部吸纳了过来。
<br/>　　他手上的血阳花，已经涨到了七寸大小，距离一尺大小已经不远了。
<br/>　　巫天御有些心急了，他没想到血阳花居然生长的这么慢，以他吸纳血煞之气的速度，这么久了才长到七尺大小。
<br/>　　他的战魂不弱，乃是高级战魂，而且由于是飓风战魂的缘故，吸纳血煞之气的速度，远超其他性质的战魂。
<br/>　　但就算如此，过去了这么久的时间，他的血阳花才长到七尺。
<br/>　　“血煞之气怎么突然变弱了？”这时，巫天御眉头一皱，因为他感觉到自己所在之地，通道中的血煞之气，突然变的淡薄无比。
<br/>　　来不及多想，略一沉吟，他毫不犹豫的起身，手捧着血阳花离开了此地，向山峰内部更深处飞掠而去。
<br/>　　时间已经不多了，他必须要加紧时间了，不然的话，恐怕很难达到一尺大小了。
<br/>　　整个血煞山之内，一百八十余名巫族天才，都在疯狂的吸纳血煞之气，催生自己手中的血阳花。
<br/>　　广场之上，没有人离开，所有的人都在等待。
<br/>　　众人都在猜测，这第二关考验，谁能独领风骚，夺得第一名。
<br/>　　毫无疑问，九成九的人都认为是巫人王。
<br/>　　双生武魂的巫人王，再加上特殊体质，绝对是最巅峰的存在。
<br/>　　诚然，巫人王的特殊体质，与血煞之气半毛钱关系都没有，但多多少少会有点用处吧！
<br/>　　“这一次，巫尘应该要被淘汰了吧！”
<br/>　　人群之中，巫源长老呢喃自语，巫尘能闯过第一关，没有出乎他的预料。
<br/>　　但这第二关，虽然大祭司说的轻描淡写，但难度却是相当的大，想要的一个时辰之内，将一颗血阳花的种子，催生到长成一尺大小的血阳花，一般的超级天才都做不到。
<br/>　　时间缓缓流逝，在无数人的猜测之中，一个时辰的时间到了。
<br/>　　“所有人，离开血煞山！”大祭司淡漠的声音响起，但听在血煞山内众天才的耳中，无异于天雷炸响。
<br/>　　山洞之中，听到大祭司的声音，苏莫立刻停止了吸纳血煞之气。
<br/>　　此刻，他手中的血阳花，已经涨到了三丈大小，非常的惊人。
<br/>　　“第一名跑不掉了吧！”苏莫面上露出了一丝笑容，他也是不知道别人能达到什么地步，所以才不停的吸纳血煞之气。
<br/>　　不过，他已经努力的克制吸纳血煞之气的速度了，若是他拼命的催动吞噬战魂，一个时辰之内，绝对能让血阳花涨到十几丈大小都不止。
<br/>　　随即，苏莫手掌一番，收起了血阳花，闪身飞出了山洞。
