　　混沌之中不记年，时间悄悄从指缝间流走，苍茫依旧，亘古不变！
<br/>　　只是，从前寂静无比的混沌随着苏醒的混沌魔神的增多，变得不再平静。入眼所见，虽然还是灰蒙蒙一片，但混沌风暴早已经形成飓风，席卷八方，整个混沌早已变得汹涌狂乱，不复之前的宁静。
<br/>　　其余魔神如何？元释不知，但是他自己自从上次闭关，沉醉在领悟大道因果法则的美妙之中，感受着自己那逐渐在增加的修为，陷入不可自拔中，忘了时间、空间，忘了一切。
<br/>　　当元释再次苏醒之后，入眼处依旧是那一片苍茫的混沌，滚滚气流在翻滚，没有丝毫变化。但是，元释的体内却早已发生了翻天覆地的蜕变！
<br/>　　在大道法则之力的淬炼之下，元释那原本就极度可怕的魔神真身，又不知道强横了多少，有着百万丈之高，耸立混沌，浩瀚无垠，一股苍茫的气息弥漫着！
<br/>　　元释有信心，仅仅凭着这混沌魔神之体，就可以轻轻碾压大多数突破的混元大罗金仙前六重天的强者，即便对方魂寄虚空，真灵不灭。但那又如何，这只是相对的，在绝对的实力之下，虚空也得破碎，真灵不灭就是一个笑话！
<br/>　　这还只是他的魔神之体的力量，在这次修炼之中，他受益最大的，还是他的修为及道法神通。
<br/>　　原本他只是刚刚突破第三步，修为只是混元大罗金仙第一重天初期，但在那大道玄妙之下，他的神魂、意志再次接受大道的洗礼，在配合那神秘的鸿蒙元始印，借助此印的本源之道，助他悟道！
<br/>　　他沉浸在那股道韵之中，不知过了多少岁月，身体自动吸收混沌之气，随着大道因果法则的领悟加深，修为也随着自动提升，醒来之后，他惊骇的发现，自己的修为居然已经彻底的将自己的大道道基，因果之道，领悟到了一个极致，修为不知不觉中达到了一个很大的高度，混元大罗金仙九重天巅峰！
<br/>　　混元大罗金仙九重天为混元后期，只差一步，就可以突破到混元大罗金仙第十重天，混元圆满之境，踏入那第三步最后的境界！（第三步混元十二重天，前三重天为混元初期，四至六重天为混元中期，七至九重天为混元后期，十至十二重天为混元圆满境！）
<br/>　　要知道，修行之路，越到后面就越发的艰难，每进一步，就是天差地别！同样是混元之境，九重天与十重天根本不可同日而语，十个九重天也不可能是一个十重天的混元大罗金仙的对手，这是混元后期与圆满境的差别，后期还在第三步中前进，而圆满境已经是在为第四步做准备了。
<br/>　　是以，别看元释现在是混元九重天巅峰，与那十重天只是一步之远，但想要突破到十重天，没有逆天的机缘，真不知道要到何时才能突破？
<br/>　　但仅仅是混元大罗金仙九重天，就已经难倒了大多数的混沌魔神，要不是元释是前十孕化的混沌魔神，还有着鸿蒙元始印这件神秘莫测的鸿蒙至宝，他也不会这么快就突破到混元大罗金仙九重天。
<br/>　　不知道多少人还在为此目标奋斗着，就是那排名前十苏醒的混沌魔神，突破到此境界的也少之又少。当然，盘古这个大道之子除外，那个是有着大道帮忙开外挂，想不突破都难。估计那厮现如今早已迈入那混元圆满境界，为突破最后的第四步做着准备。
<br/>　　当然，修为还只是元释如今实力的一部分，他还有着强横的混沌魔神真身，以及混沌魔神的本命法则、天赋神通，以及自己这期间领悟的道法及神通。
<br/>　　在领悟本命大道因果法则期间，元释也不是只领悟法则，不创神通。要知道，混沌之中，危险重重，混沌劫更是所有魔神的末路，只有法则，没有神通护道，是走不远的。
<br/>　　元释的本命法则神通是“执掌因果”，随着领悟大道因果法则之道的加深，本命神通的威力及范围也就越大。起先只能掌控第一步及第二步强者的因果，但随着领悟法则的加深，现在已经能够开始掌控第三步混元强者的因果了，当然，这些强者的修为要比他自己低。总之，他能够掌控比自己修为低的修士的因果。
<br/>　　若是他想要突破到天道境，就必须掌控整个天道因果法则本源，想要突破到最后一步，只有执掌整个大道因果法则本源。想要突破这些，就必须要面对来自天道及大道之劫，天道及大道都不会眼睁睁地看着多一个与他们等同地位的人出现，来削弱他们的权力。到时候要面对的危险强度，可想而知。至少元释知道，最后连盘古都失败了。
<br/>　　元释闭关期间创出的法则神通有“强结因果术”、“因果绞杀”、“断因果”、“因果业杀”、“因果道眼”、“元心印”、“五行印”。
<br/>　　“强结因果术”，顾名思义，强行与人结因果，除非对方实力强大到能够强行斩断此因果，不然不怕对方不还这段因果。当然，这“强结因果术”不可能随便施展，这需要一定的媒介。
<br/>　　“因果绞杀”，利用对方的因果之线，绞杀对方，前提是，对方身上缠有因果。
<br/>　　“断因果”，断去比自己弱或者与自己实力差距不大强者的因果，有两种结果，一是仅仅断去因果，不影响个人，二是通过断因果来灭杀对方。
<br/>　　“因果业杀”，因果是一种无形的力量，如影随形，如声斯响，善因有善果，恶因有恶果，身缠因果，既有“业”，即善业、恶业、无记业。这种看不见、摸不着的业，却拥有着强大的力量。拥有业，既有业力，吾等人生是受业力支配的。“因果业杀”就是利用这种业力来杀死对方。
<br/>　　“因果道眼”，因果法则之道形成的道眼，可以看破世间因果。
<br/>　　“元心印”，乃元释利用鸿蒙元始印结合前世可以看破对方内心的神通“盗梦神通”推演出来的神通，但较之之前的神通不知强大几何，而且还少了在梦中施展这个限制条件。
<br/>　　当然“元心印”这神通已经初步涉及到“心道”的边缘了，还不完善，这需要足够的心道修为来施展。“心道”是一种不在三千大道之中的道，但他却又是实实在在存在的，每人都拥有它，但就是不能修炼它。如，修炼即修心，没有一定的心境修为却拥有强大的实力，只会沦落为实力的奴隶，为实力控制。心魔就是心境修为不到家才会出现的。
<br/>　　当然，心道还有更广阔的用途，这只是心道最基本的一个运用。众人都不知如何修炼心道，或者根本不知道有心道这一种道。
<br/>　　心道浩瀚无垠，最是难以捉摸！
