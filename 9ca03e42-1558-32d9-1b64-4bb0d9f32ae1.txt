        童筱颖苦笑：“我不是苏漫雪，我是童筱颖。”
<br/>
<br/>        “苏小姐病得不轻。”刘管家自顾自地说。
<br/>
<br/>        童筱颖从床边起身，却因为蹲坐得太久，而有些发黑眼晕。她打了个趔趄，直扑在刘管家的脚边。
<br/>
<br/>        她吃力地拽住刘管家的裤腿，一脸狼狈地抬起头来，看着刘管家，苦苦哀求道：“刘、刘叔……我求你，你放我离开这里，好不好？你一定知道，我压根就不是苏漫雪，对不对？”
<br/>
<br/>        “苏小姐，您现在的演技可是越来越高超了。如果有机会，我会跟傅老太爷说说您的特长，指不定送去娱乐圈里混还能大火一把！”刘管家嗤之以鼻。
<br/>
<br/>        童筱颖咬了咬下唇，微微闭了闭双眸。
<br/>
<br/>        与其求别人，不如靠自己！
<br/>
<br/>        她得想办法离开这里，苏、漫、雪！等她离开这里，她一定不会放过这个女人！
<br/>
<br/>        被囚禁在别墅里的童筱颖开始尝试着跟刘管家套近乎，别墅里的家务，她都会主动帮刘管家做，尽量讨刘管家的欢心。
<br/>
<br/>        “苏小姐这是想通了吗？”然而，刘管家似乎并不领情，站在一旁，看着正在拖地的童筱颖，还不忘冷嘲热讽。
<br/>
<br/>        童筱颖只是听着，什么话也不说，一笑而过便是。她现在必须忍气吞声，等逮着了机会，能拿到刘管家的手机，她便跟傅昼景打求助电话。
<br/>
<br/>        其实，她现在想得最多的就是，傅夜沉能来这别苑就好。
<br/>
<br/>        傅夜沉来了，一定知道她不是苏漫雪，一定会放她回到傅昼景身边。
<br/>
<br/>        更多的，童筱颖在期待，傅昼景不要被苏漫雪骗了就好。
<br/>
<br/>        刘管家每天上午有出去买菜的习惯，但别苑里还有一个冷面的女保镖在这监视着她。
<br/>
<br/>        童筱颖拖完地后，便坐在客厅的沙发上想，到底要怎么和外界取得联系？
<br/>
<br/>        不如，把那女保镖打晕了，逃出去？
<br/>
<br/>        童筱颖心里想着，抬眸看向一旁满脸冷漠的女保镖，顿时不寒而栗。
<br/>
<br/>        只怕她还没将她打晕，她便被她给“收拾”了。
<br/>
<br/>        左也不行，右也不行，童筱颖一筹莫展。
<br/>
<br/>        “保镖小姐，我们来聊天吧！”发了一会儿呆，童筱颖一手托腮，看着一旁占得笔直的女保镖说道。
<br/>
<br/>        哪知，这个女保镖压根就不正眼看她一下。
<br/>
<br/>        童筱颖有些郁郁寡欢。
<br/>
<br/>        而苏漫雪成功离开依山别苑后，拿着童筱颖的身份证和银行卡，去了一趟银行。
<br/>
<br/>        她还以为，傅昼景会给童筱颖很多钱，却没想到，童筱颖的银行卡里居然只有四万多块钱，连塞她牙缝都不够。
<br/>
<br/>        就在苏漫雪感到恼火的时候，童筱颖的手机铃声响起，一个滨江城的陌生来电，让苏漫雪毫不犹豫地接听了电话。
<br/>
<br/>        “筱颖，我是季思妍，傅昼景的秘书。我们还义结金兰了，过了这么久才第一次跟你联系。想必，你还记得我吧？”季思妍柔情似水地说。
<br/>
<br/>        苏漫雪虽然听得一脸茫然，却故作淡定，用童筱颖的嗓音，反问道：“你找我有什么事？”
<br/>
<br/>        “你学校里放假了吧！什么时候回滨江城？傅总他最近很忙，派我去高铁站接你。”季思妍微笑道。
<br/>
<br/>        苏漫雪忽然间有了个主意，她得用“童筱颖”的身份，从傅昼景的身上捞点钱回来用。
<br/>
<br/>        “那好啊！我今天就回来，你到高铁站去接我！”苏漫雪咧嘴一笑。
<br/>
<br/>        “好！”季思妍微笑着应了声。
<br/>
<br/>        苏漫雪挂了电话后，从童筱颖的银行卡里取了钱，给自己换了身行头，随即便订了一张去滨江城的高铁票。
<br/>
<br/>        在高铁站的出站口，季思妍接到了“童筱颖”。
<br/>
<br/>        只是，她看到“童筱颖”的第一眼，莫名其妙地产生了一种是另一个人的错觉。
<br/>
<br/>        高铁站的出站口，人来人往，一个皮肤如白瓷，如瀑布般的长发披散在肩头的女人引起了不少人注意。
<br/>
<br/>        季思妍一眼就认出了“童筱颖”，却又怀疑自己是不是认错了人。

<br/>        她脚上踩着白色细高跟，身形高挑纤细，走起路来，臀部微扭，脚下带风。

<br/>
<br/>        季思妍带着心里的疑惑，朝苏漫雪走了过去。

<br/>        苏漫雪四下张望，见有个穿着黑色西装裙的女人从人群中朝自己走过来，心里便断定这个女人很有可能就是起初那个给自己打电话的“季思妍”。
