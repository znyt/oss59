翌日清晨。
<br/>
<br/>        陆寒霆比往常起晚了，昨晚他折腾了很久，起床冲了好几次澡才压下身体的躁动，今天他脸色不太好，看着很阴沉。
<br/>
<br/>        他先去隔壁房间看奶奶，但是推开房间门，床上空空的，奶奶不见了。
<br/>
<br/>        陆寒霆瞳仁一缩，薄唇直接抿成了一道泛白的弧线，奶奶去哪里了？
<br/>
<br/>        这三个月奶奶都是待在自己的房间里，不愿意出去的。
<br/>
<br/>        “吴妈，奶奶呢？”
<br/>
<br/>        陆寒霆迅速拔开长腿下了楼，很快他脚步倏然一顿，停在了客厅里，因为他看到了奶奶。
<br/>
<br/>        老夫人现在在厨房里，因为久病的原因，老夫人坐在轮椅上，腿上盖了一条柔软的毛毯，外面璀璨而温暖的晨曦透过厨房的窗户洒了进来，老夫人在那里惬意的晒着太阳。
<br/>
<br/>        陆寒霆心头压着的大石瞬间放了下来，刚才他还以为奶奶出了什么意外。
<br/>
<br/>        他又觉得很诧异，奶奶怎么愿意出来晒太阳了，这可是奶奶第一次出房门。
<br/>
<br/>        这时就听里面的老夫人笑眯眯道，“绾绾，你是不是还在生我气啊？”
<br/>
<br/>        很快耳畔传来了一道清丽软糯的嗓音，“哼奶奶，我不理你了！”
<br/>
<br/>        陆寒霆喉头一滚，他听出来了，这是…夏夕绾的声音。
<br/>
<br/>        昨晚她爬上他的床，扰的他一帘春.梦，他还以为她走了，没想到她竟然还在。
<br/>
<br/>        陆寒霆拔腿上前，很快就在厨房里看到了那道纤尘绝丽的身影，今天的女孩穿了一件那种很嫩的草绿色毛衣，毛衣是娃娃领的，很宽大，落在她膝盖上方，下面是黑色的铅笔裤，八分的，露出精致纤细的足踝，她今天的穿着青春又俏妍，扑面而来的少女气。
<br/>
<br/>        现在她身上系了一条围裙，正站在流理台前准备早餐，纤白的小手里拿着勺子，搅动着锅里已经熬得很稠的小米粥。
<br/>
<br/>        一头清纯的乌发散落肩头，在一边掖到了雪白的耳后，陆寒霆的角度可以看到她半个侧颜，她拧着秀眉，粉嫩的两腮都生气的嘟了起来，奶凶奶凶的，“还有吴妈，这一次奶奶你拉着吴妈一起骗我！”
<br/>
<br/>        吴妈在一边洗水果，闻言她迅速讨好的笑道，“绾绾，昨晚我…我是眼花了，将你带错房间了，老夫人真的没有指使我。”
<br/>
<br/>        这句话大概是真的，老夫人真的没有明着指使，只是暗暗的点拨了一下，这都几十年的主仆了，吴妈当然是瞬间会意了。
<br/>
<br/>        喵~
<br/>
<br/>        喵喵~
<br/>
<br/>        耳畔传来了两声奶奶的猫叫，陆寒霆见小圆圆也在，小圆圆现在窝在夏夕绾的脚边，对着老夫人和吴妈叫了好几声，意思是你们欺负我的女主人，我也生气了。
<br/>
<br/>        整个厨房里其乐融融，所有人脸上都挂着笑，里面的小米粥香气都跑了出来，满屋子的米香。
<br/>
<br/>        陆寒霆高大英挺的身躯僵在了原地，他没有想到自己会看到这一幕，这三个月来奶奶都病着，所有人都很压抑，一个家里冷冷清清的，虽然配有星级大厨，但是一点烟火气都没有。
