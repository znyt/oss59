　　那当然不是谢雨晴，但是真的谢雨晴，他也见不到了。
<br/>　　她是爱过自己的，现在大概仍然爱着。
<br/>　　想到这一点，他就觉得心脏很疼，干脆就在马路边坐下去，往日的一幕幕浮上心头，在一起旅游的那几天里，平时大大咧咧的谢雨晴表现出了罕见的温柔，早晨会给他买好早饭，会给他洗衣服，每天早上强迫他喝一杯牛奶，睡觉之前帮他脸上涂防晒霜。
<br/>　　他就像个大男孩，欢喜地享受着这一切，在涂脸的时候，两人之间的距离是那么近，他一抬头就能看到她的脸，她的睫毛，她的洁白的脖子。
<br/>　　这时候她就会笑一笑，或者在他脑袋上敲一下，她把他当孩子，当弟弟，告诉他，自己也是第一次对人这样，她以前也不知道自己还有这技能。
<br/>　　是叶少阳帮她开发了新技能，只是，时间太短了，只有五天，她就收拾起刚学会的技能，一路南下，消失不见了。
<br/>　　醉笑陪公三万场，不诉离殇。
<br/>　　就像急病突发，之前还好好的叶少阳，一下子就被伤感击倒了，捂着心口，像个酒鬼似的坐在地上，龇牙咧嘴。开始他自己也觉得奇怪，后来才想明白，原来那些伤感的情绪，并没有随着时间流逝而淡化，而是一直都在，就在内心深处，逐渐积累，撑着自己的心脏，撑不出就爆发了，一发不可收拾。
<br/>　　不光是谢雨晴，还有周静茹、覃小慧，还有走了不久的碧清，她们都不见了，匆匆相逢，又匆匆分开。
<br/>　　他是最怕离别的人，最怕萍水分离再不相见，最怕天涯海角碧落黄泉。
<br/>　　他害怕离别，但离别总是找上他。她们都走了，下一个离别的，又会是谁呢？
<br/>　　相聚皆是缘，缘尽莫强求。
<br/>　　叶少阳抹了一把脸，草泥马，居然哭了。
<br/>　　（非凑字，文青了一把，大家理解则个）
<br/>　　早上醒来，自己躺在酒店的床上，叶少阳仔细回忆了一下，昨晚自己好像是半夜才回来的，回来后还喝了房间冰柜里的两瓶啤酒，然后怎么睡着的就忘了。
