　　吴三给了林峰一个物品清单，上面列举了吴三近期观察后发现的极具有升值价值潜力的东西。
<br/>　　林峰点了点头把这个东西给手写了几分然后给李广陈勾他们，他们主要负责收集这些物品。
<br/>　　“林峰，在不在基地。”
<br/>　　刚离开店，林峰就接到了寒鸦的电话。
<br/>　　“会长，我在基地。”
<br/>　　“你来一次总部吧。”
<br/>　　寒鸦说完就挂掉了电话。
<br/>　　林峰虽然不知道寒鸦这个时候给他打电话干什么，但还是过去了一趟。来到总部后发现十二个干事都已经到齐了。
<br/>　　“战神来了。”
<br/>　　“好小子。”
<br/>　　所有人都热情的和林峰打招呼，先遣部队的事情他们不知道具体发生了什么，但或多或少都知道了点，毕竟他们或多或少也都和军部有点关系。
<br/>　　“林峰，我一开始还以为我叫不动你了，你现在在军部之中的名气很大啊，许多人都叫你战神。”
<br/>　　寒鸦打趣道。
<br/>　　“行了吧，我可不敢自称战神。”
<br/>　　林峰摆了摆手。看了眼其余人，十二个干事都到齐，这肯定是有大事。
<br/>　　“说正事吧。”
<br/>　　寒鸦点了点头，也不墨迹，大家都不是墨迹的性格，确切来说，末世来了，让每个人都不在墨迹了，墨迹就是死。
<br/>　　“最近军部宣布了据点计划，不知道你们听说了没有。”
<br/>　　“据点计划？”
<br/>　　每个人都是诧异的看着寒鸦。
<br/>　　寒鸦道：“所谓据点计划，实际上就是在基地外面建造安全可靠的据点，这一个计划主要有实力最强的军部来牵头，战士会和基地内的地下势力可以选择参加不参加，目前，地下组织第一的天神帮第二的骷髅会以及第三的火神帮都是明确表明了会参加据点计划，因为军部表态了，只要是我们自己打下来的据点，那么不管多久以后，这个地方就是我们自己的了，我们有这一片土地的使用权，永久性质的。”
<br/>　　“什么？永久性质？这意思不就是相当于占地为王了吗？？”
<br/>　　刘峰震惊道。
<br/>　　“差不多就是这个意思。”寒鸦笑道。
<br/>　　“我去，这可是大手笔啊。”每个干事都震惊了，在地球上，什么最珍贵？那肯定是土地啊。
<br/>　　“当然了，你们不要把这件事情想得太简单了，基地外面都是怪物丧尸，要想开辟出一个固若金汤不被毁灭掉的据点，不简单。”寒鸦道。
<br/>　　“会长，这可是大好事，我们战士会无论如何也要参加。”刘峰道。
<br/>　　每个干事都明确表态要参加。
<br/>　　“好！我初步选定的地方就是这个地方！黑龙镇，这个地方在基地南边以北大概十五里地方，和基地之间的道路畅通，可以说是非常不错的一个地方，唯一的麻烦就是这个黑龙镇内有一头A级怪物盘踞着，足足有九阶实力，其次丧尸数量保守估计，五千头！”
<br/>　　寒鸦把地图拿出来，摆在众人面前。
<br/>　　“A级怪物？”
<br/>　　寒鸦道：“A级怪物也是军部划分的一个具体等级，从最强到最弱分别为S.A.C.D.E.F这五个等级，目前A级怪物是九阶的存在，盘踞在黑龙镇里的A级怪物是一头变异响尾蛇，实力属于很强的一类了，其次B级的怪物也有好几头。”
<br/>　　“这么多高等级怪物。”
<br/>　　一群干事都犯难了，要知道他们几个干事也不过才五阶罢了，偶然几个六阶也是最近才突破的，最强的也就是韩风的七阶。
<br/>　　当然他们以为韩风是最高的，但却不知道此时此刻的林峰等级上已经足足八阶了，仅次于寒鸦。
<br/>　　“所以这一次，我们战士会包括我和罗敏副会长都会出动，罗敏副会长也已经突破到了八阶，再加上韩风的七阶，以及六阶的林峰。”
<br/>　　寒鸦瞧了眼林峰笑道。
<br/>　　林峰摸了摸鼻子，他知道对方看出来了自己不是六阶了。
<br/>　　这一次是全体的大规模行动。
<br/>　　林峰虽然可以从军部调兵过来，他有这个权利，但是这怎么说也算是他们战士会自己的事情，军部现在肯定还在忙着肃清内部的事情，自己都忙不过来了，所以林峰没有厚着脸皮去调兵。
<br/>　　不过战士会的上千个成员，在这一次算是倾巢出动了。当然了，这一次任务不是强制的。
<br/>　　这就是民间组织和军部的最大区别，军部要去执行个任务，那可不会给你选择去不去的余地。
<br/>　　不少人选择了不去，这群人有明智之选，虽然组织这一次提供的福利非常的诱人，但他们知道自己实力太弱，去了也是送死，不过更多的人还是选择去。
<br/>　　因为去了，若是还能活着回来，那就是战士会的大功臣，日后都是有一笔很丰富的奖励。基本上可以活着不愁了。
<br/>　　浩浩汤汤一大群人，最终足足一千个人决定去了。十二个干事，每个人负责带一百个。不过林峰没要他的一百个小弟。
<br/>　　一来林峰不喜欢带一群人，这样目标太大，人多不好管理，二来，林峰的秘密，不想太多人知道，毕竟一会坦克一会无人机的。仅仅异能两个字，不好解释，毕竟异能是伪科学，而不是神乎其神的魔法。
<br/>　　这一次大规模行动林峰谁都没带在身边，哪怕是后宫之主陈勾都没有带在身边。
<br/>　　李广和陈勾要帮助吴三去搞些店里需要的材料，有些怪物的材料值钱，有些怪物的材料不值钱，所以需要人去特定的杀这些怪物。李广和陈勾他们做的就是这个事情。
<br/>　　而且最主要的是，太危险了，林峰一个人可以自保，带了人的话就不一定了，这一次任务，大家虽然元气满满，实际上，可以说绝对要死很多人了。
<br/>　　从安排好作战计划到全军出击，之间过了足足七天时间，这七天时间自然是各种辎重的准备了。
<br/>　　寒鸦调动一些关系从军部借一些火炮。
<br/>　　林峰这七天也哪里没去，离开基地捡空投后就第一时间回来。
<br/>　　七天下来也存了大概五十个D级空投了。以及五个C级空投。
<br/>　　林峰把五十个D级空投全部合成C级的，总共是十个C级空投。
<br/>　　林峰一次性全给他开了。
<br/>　　第一个C级空投里面全都是吃的。
<br/>　　老坛酸菜面都好几箱了。
<br/>　　林峰直接把这老坛酸菜面拿去卖了，林峰不喜穿吃这玩意，苏云陈勾李广他们也都不喜欢吃这个味道。
<br/>　　干脆林峰拿两箱去卖，一桶老坛酸菜面标价五十晶石。
<br/>　　本来老坛酸菜面的价格还要更高的，但因为这段时间军部科研部的发力，导致粮食问题没有之前那么严重了。
<br/>　　许多怪物的肉，甚至一些植物，都被发现可以食用，大大的缓解了粮食问题。
<br/>　　最后整个市场原本混乱的价格，也趋近于稳定，林峰旋即继续开空投。
