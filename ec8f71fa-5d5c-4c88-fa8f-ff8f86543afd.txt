        &lt;h3&gt;第三卷第三十四章  欢乐三人行2&lt;/h3&gt;
<br/>
<br/>        人生有很多的际遇，是错过，也是重生，是美好、也是困顿。
<br/>
<br/>        在浩瀚无垠的苍穹之下，我们都是一粒微尘，不小心被风带动，走了一程又一程，遇见了美好，也有不堪负重的前行。
<br/>
<br/>        生活，踩在脚下是路，哪怕坑坑洼洼、哪怕荆棘丛生，揣着一颗光明的心，再苦再不堪的人生也会走出一片温馨的光阴。
<br/>
<br/>        陈子昂很感激她生命之中遇到的一切人，所有的人都给予了她帮助，包括张函的母亲和陆玉，如果没有当初她们去找她，可能她不会有后面精彩的人生，痛苦也罢、艰辛也罢，都已经被她踏平。
<br/>
<br/>        在时间的洪流中，没有什么是能够被长久的留存，包括仇恨、时间会淡化一起。
<br/>
<br/>        陈子昂是一个小人物，她不是圣女，在那些痛苦、困难的日子里，她也曾经怨过，怨命运的不公，埋怨张函不找她，对张函有多爱其实她的心里就对张函有多抱怨。
<br/>
<br/>        随着她努力倔强的奋斗，时间将抱怨已经消灭，留在心中的只是曾经的美好，她早已经接受了生活给予她的一切，她只有不停的奔跑，才不会被时代和时光的洪流淹没。
<br/>
<br/>        江俞轩和陈子霖已经不再拌嘴了，江俞轩不停的问陈子霖，陈子昂小时候的事情。
<br/>
<br/>        而陈子霖也不停的问江俞轩，陈子昂这几年的生活，他们都在陈子昂生命中的某一个时间段里缺失过。
<br/>
<br/>        俩个大男人你一句我一句就着陈子昂聊开了，留下陈子昂静静的站在广告墙前沉思，时光在此刻是温柔的、美好的、安宁和谐的。
<br/>
<br/>        山风拂过，林间传来了树木之间的窃窃私语，小鸟清唱，秋高气爽，阳光落在林间，交叠着明明暗暗的光影，俊男靓女站在那里，时而微笑，时而沉思，这是一幅活生生的人物山水画。
<br/>
<br/>        陈子霖对江俞轩说：“俞轩，我们家是农村的，兄弟姊妹七八个，父母亲一个字都不认识，我听我父亲说，他原本还有一个妹妹，只是当初生活困顿，爷爷养不活俩个孩子，将我姑姑送人了，父亲就兄弟一个人，感觉孤单，就和妈妈一起生了我们这几个。
<br/>
<br/>        父亲和母亲吃了没有读书的苦，日出而作，日落而息，辛辛苦苦供我们几个都读书，希望我们能有一份体面的生活。我们家四个大的，都是有工作的人，那个时候父母亲年轻，有能力干活，等我们四个小的的时候，就已经力不从心了。”
<br/>
<br/>        陈子霖看了一眼陈子昂：“在别人的家庭，小的是最被疼爱的，而我们家子昂从小就缺吃少穿，上学的费用都是贷款，原本想着能考上大学，谁知道造化弄人，她这一生都与大学无缘。曾记得子昂豪言壮语的说：哥，我要考上电子科技大学，我要去做研究，要做时代的弄潮儿，要为我们的社会做一份贡献。”
<br/>
<br/>        “虽然她没有上大学，可是她已经实现了她的梦想啊。”江俞轩对陈子霖说。
<br/>
<br/>        “你知道你现在手中的项目所需的软件都是哪里来的吗？”
<br/>
<br/>        “不知道。”
<br/>
<br/>        “我们团队自行开发研究的，我到公司的时候，陈子昂的公司只有十几个人，但是有一大部分是做开发的，当时她拿到了一个单位的信息化建设，算是第一个做这个事情的。她一边带着大家做开发，一边跑业务，京城里的大街小巷没有几个她不熟悉，那个单位的门朝东还是朝西，她都知道。有时候她被人一次次的拒绝，她一次次的上门。”江俞轩摇了摇头。
<br/>
<br/>        “我们有一次去一个单位，都跑了好多次，最后被人家给赶出来了，你知道人家当时是怎么赶我们的吗？”
<br/>
<br/>        陈子霖也像江俞轩那样摇了摇头，陈子昂也带着他跑过业务，期间的困难他是知道了。
<br/>
<br/>        “人家直接将我们的资料砸在子昂的脸上，将子昂的脸都砸烂了，并且说：这脸真是不想要了？给你个脸色，你就不知道你是个，什么东西，滚！”
<br/>
<br/>        陈子霖紧了紧拳头，他和子昂跑业务的时候还没有遇到过这种情况，要是遇到了，他会动手的。
<br/>
<br/>        “我差一点就动手了，子昂拉着我，对我说：能吃苦中苦，方为人上人，韩信当年能受胯下之辱，何况我们这样的小人物呢？他拒绝了我们，其实就是拒绝改变，我们虽然要生存，要赚钱，但是，俞轩，你要知道，我们做的是一份事业，一份能促进社会向前发展的事业。”
<br/>
<br/>        “可是，他动手了。”我说。
<br/>
<br/>        “这个世界上什么人都会碰到，难道你要和每一个人去计较？这样累不累啊？或许他今天刚好心情不好，需要发泄，而我们刚好碰到了。”
<br/>
<br/>        “嗨，陈子霖，你都不知道你妹妹心怀有多宽广，看着她我是又心疼又欣慰。这么几年我们俩个就这么磕磕碰碰的走过来。社会发展太快，我们公司也发展太快，所有的资金都是她去跑的，这几年我们已经有了不小的规模，资产上亿。“
<br/>
<br/>        “我们这是赶上了好时代，当初在中关村的时候，中关村政策很好，我办理公司只要在那里找好地方，不到一个礼拜就注册好了，当时it也是在中关村兴起不久，每天的人流量大概十万左右，我从一个门外汉步入中关村，其实靠的就是我的勤劳，我努力的学习新的知识，每天碎片化的学习，给客户送货我也在学，走在路上我也在学，我阅读大量的电子信息类的书籍，用知识来储备自己，要不然也不会有我今天安宁的站在这里和你们聊天。”陈子昂接过江俞轩的话说。
<br/>
<br/>        她看了一眼远处，青山绿水，江山如画，他们置身在这样一个和谐安宁富足的时代里，有何理由不努力？
<br/>
<br/>        人生有很多机遇，而机遇其实就是留给有准备的人的，如果她不走出去，如果她只是一个小学生的知识水平，试想，她还能站在信息行业里叱咤风云吗？
<br/>
<br/>        互联网大佬们的崛起，哪一个不是充满着传奇，哪一个不是辛苦创业？哪一个不是在努力的坚持？
<br/>
<br/>        所以，知识改变命运不是说说而已，尤其是像她这种农村出来的孩子，更是需要努力的学习，学习了大量的知识，开拓了眼界，启发了思想，她才能越走越远。
