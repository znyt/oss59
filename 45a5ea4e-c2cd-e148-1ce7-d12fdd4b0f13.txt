　　苍龙所经过的地方，满目疮痍，碎石飞射，虽然它没有灵智，但半步元婴的修为可不是盖的，肉身更是恐怖至极至极。
<br/>　　哪怕怀空等人的神兵，斩击在它身上，也难以伤它分毫，他们实力都不弱，在苍龙的攻击下，只是受了一点轻伤，并不算严重。
<br/>　　“昂昂昂......！”
<br/>　　苍龙带着恐怖的力量再次袭来，慕清霄收敛气息，隐匿在巨石后方，苍龙也并未注意到他，而是一路直追怀空等人。
<br/>　　伴随着它一声长啸，忽然抬起龙首，龙口张开，一道炽热的火焰喷射而出。
<br/>　　突如其来的火焰，让怀空等人大惊，连忙调动全身灵气，布满全身，以此来抵挡苍龙的火焰。
<br/>　　火焰温度高得出奇，所过之处漆黑一片，寸草不生，甚至有的巨石直接化为粉末，随风而散，可见火焰的恐怖。
<br/>　　慕清霄望着苦苦坚持的怀空等人，嘴角不由的抽了抽，不愧是半步元婴，实力真是恐怖如斯，哪怕没有灵智，也不是那么好对付的。
<br/>　　霎时间，战局瞬息万变。
<br/>　　步惊云疏忽大意之下，居然让苍龙一爪子压在地上，然而就在苍龙想要了解步惊云之际，聂风及时赶来，在苍龙口下救出步惊云。
<br/>　　在这种关键的时刻，断浪等人也没有闲心情管聂风，毕竟苍龙的实力太过恐怖，多一个人就多一份力量。
<br/>　　然而苍龙也并不打算就此让过步惊云等人，数百米的龙躯直接碾压过来，形成一道包围圈，将步惊云等人全部环绕在其中，龙口再次张开，恐怖的火焰再次喷射而出。
<br/>　　待苍龙吐息过后，帝释天站起身来，道：“神母，待屠龙过后，你要找机会杀了聂风，这是你戴罪立功的好机会，你可要好好把握。”
<br/>　　此时，慕清霄已经回到战场上，但他却并未使用全力，虽看似威力惊人的攻击，实则却在敷衍了事，他和步惊云总共八人，也只能和苍龙势均力敌。
<br/>　　“看来还是本座低估了这神兽的威力，你们八人是当今武林奇才，虽然拥有神兵，却互相克制，在加上你们阳刚之气太重，很难发挥它们的威力。”
<br/>　　“照这样下去，你们恐怕就要死于苍龙爪下，现在唯一的办法，就是集齐你们的力量，加上本座千年寒冰绝学圣心诀，合为一气，如此刚柔并济，一定能够杀了那头畜生。”
<br/>　　话语刚落，帝释天也不再观望，纵身而下，步惊云等人相视一眼，调动体内大部分灵气，然后汇聚在帝释天身上。
<br/>　　旋即，以帝释天为首，九人的力量合为一气，刚柔并济，化作巨剑直指苍龙。
<br/>　　苍龙盘旋于天际，慕清霄等人御剑而行，化作一柄锋利的巨剑，追击而去，剑光爪影，几百回合下来，苍龙的龙躯上已经满目疮痍。
<br/>　　由帝释天为首，居然能够击伤苍龙，步惊云等人也不再犹豫，调动体内的灵气，不断传送过去，只要苍龙不死，他们的生命就要受到威胁。
<br/>　　慕清霄嘴角则勾起一丝邪异的笑容，他当然知道，帝释天这家伙老谋深算，但自己又何尝不是在暗算他。
<br/>　　“昂昂昂......！”
<br/>　　受伤的苍龙，性情变得更加凶猛，一双龙眸变成赤红色，伴随着嘹亮的长啸声，直接扑了上来。
<br/>　　八名巅峰高手的力量汇聚于一身，帝释天自然无所畏惧，哪怕苍龙恐怖如斯，他都有一战之力。
<br/>　　下一秒，帝释天圣心诀的气息释放而出，在圣心诀的牵引下，慕清霄等人的力量渐渐融合在一起，最后化作一柄蓝色的巨剑。
<br/>　　“畜生，你修为再高，本座也要取走龙元。”
<br/>　　话语刚落，帝释天为首，巨剑直指苍龙，带着划破天际的威势，化作一道湛蓝色流光，破空而去。
<br/>　　苍龙感受到这一剑中的威力，长达数百米的身躯，在空中不断扭动，速度开始增长，可任由它速度多快，帝释天依旧紧追不放。
<br/>　　神剑带着恐怖的威能，仅仅几秒的时间，就追上苍龙。
<br/>　　在众人的控制下，巨剑直接从左侧刺出，伴随着一连串的火花，以及鲜血的喷洒，巨剑直接刺进苍龙的命穴中。
<br/>　　那坚不可摧的鳞甲，也未能抵挡住这一剑的威力。
<br/>　　“昂昂昂！”
<br/>　　身躯让巨剑贯穿，苍龙在剧烈的疼痛下，直接撞在坚硬的山峰上，瞬间失去行动能力。
<br/>　　命穴被穿透，苍龙的生命不断流逝，龙躯也变得软趴趴的，龙眸瞬间黯然不少，其中隐隐可见绝望之色。
<br/>　　望着趴在地上动苍龙，帝释天满意的点头，当步惊云等人将力量剥离开来，却发现刚刚所调动的力量，丝毫没有反馈回来，脸色顿时变得苍白。
<br/>　　他们都知道，自己中了帝释天的奸计。
<br/>　　而在这八道身影中，唯有慕清霄神色平淡的站在哪那里，气色也亦如往常，没有一丝虚脱的现象。
<br/>　　将目光从苍龙身上移开，帝释天目光落在步惊云几人身上，发现慕清霄气色亦如往常后，神情有些意外。
<br/>　　显然，他也没想到，后者居然还能够站直身板。
<br/>　　意外归意外，帝释天自然毫不在意。
<br/>　　毕竟，在刚刚的屠龙战斗中，他可没有耗费一丝的力量，步惊云等人全部虚脱，而他汇聚着八人的力量，比往常还要强上一些。
<br/>　　“很好，非常的好，本座终于可以得到龙元了，你们八人的功力，全部都汇聚在我手中，以为本座不知道你们打的什么注意，想要龙元，痴心妄想。”
<br/>　　话语刚落，帝释天就化作一柄利剑，直接穿透苍龙躯体，从中获取到一颗赤红色，篮球大小圆球，圆球上散发着惊人的生命力。
<br/>　　这就是所有人都梦寐以求的龙元。
<br/>　　而步惊云等人，望着帝释天获得龙元，眼中尽是愤怒，但他们束手无策，只能盘坐在原地，开始调息，盼着早日恢复，夺回龙元。
<br/>　　拿着手中的龙元，帝释天激动道：“我终于要成为最强的神了，就凭你们也想获得龙元，就在那里好好调息吧。”
