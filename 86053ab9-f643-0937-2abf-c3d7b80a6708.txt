　　第二天众人依旧是如同平时一样早早的起床锻炼，越是到了比赛的时候，越是不能掉以轻心。
<br/>　　每天晨练可以让他们保持很高的专注度和精力，最重要的是活力满满，这就是长久以来晨练的好处。
<br/>　　有些东西并不是一蹴而就的，可能刚刚开始选择晨练会有些累，但是度过了那段时间后，那么你就会感觉到一种全新的人生。
<br/>　　如果还在高中时代的话，可能一整天你也不会犯困，这就是一个良好习惯带来的收益。
<br/>　　叶帝他们自然懂得这个道理，晨练会让他们在比赛中的专注度变的更加的高，而且还可以让他们的身体素质变的更好，让他们足以应付更长时间的比赛，所以他们自然是不会放弃掉如此好的一个习惯。
<br/>　　晨练过后，众人便是继续开始了一天的训练，比赛室在下午开始，所以他们这个时候自然是不会浪费时间早早的坐在电脑前等待。
<br/>　　而且今天也不是揭幕战了，他们也不必去现场观战，只需要在家里就好了，毕竟这样更节省时间一些。
<br/>　　今天是第二组的比赛，这里面有着KA战队，有着beng这名十王者的选手，所以大家自然是想要探探底。
<br/>　　能够成为十王者选手，自然是有着他的实力在里面，接下来像是法国的EP战队还有法国的Sky战队都是叶帝他们关注的对象。
<br/>　　一个队伍里面有着emp这样一个ad选手，另外一个队伍则是拥有着辅助long在内，都是时间顶尖级别的选手。
<br/>　　至于long去法国，这就相当于以一个外援的身份去的，在叶帝没有出现的时候，华夏的战队同样请了很多外援，现在许多战队依旧有外援，这是一个道理。
<br/>　　快要到下午1点钟的时候，吃过午饭的叶帝等人已经是早早的坐在了客厅，等待着比赛的开始，今天他们关注的比赛只有这第一场KA战队的对决，接下来第二组其他一些战队的比赛倒是入不了几人的眼。
<br/>　　“这场比赛的时间排的挺好的，这样咱们看完比赛后也好早早的继续打排位什么的了。”坐在沙发上，香蕉开口说道。
<br/>　　“不过明天就没有这么好了，第三组的TM战队是在3点钟那场，所以到时候咱们要抽时间过来看了。”萝卜跟着开口。
<br/>　　“其实真不想看比赛啊，还不如一气让咱们打完呢，这种打一天休息5天的感觉真的是太难受了。”蒋念这个时候也是跟着抱怨了起来。
<br/>　　“快了，你们不是在第五组了么，还有三天就要轮到你们了，不要急。”叶帝笑着开口安慰道。
<br/>　　在几人说话的时候，场上KA战队和西班牙的UB战队的比赛已经是开始了。
<br/>　　UB战队在60支战队中根据官方的排名，他们的综合实力是在24名的，也算的上是一支实力比较强劲的队伍了，毕竟能够从那么多队伍中脱颖而出，闯入到世界赛中就已经说明了他们的实力了。
<br/>　　以前的世界赛可是600支队伍那样的大规模比赛啊，他们能够闯入到世界赛，那就是说他们已经淘汰掉了540支战队，这样的实力已经是非常的可观了。
<br/>　　只不过当比赛打起来的时候，他们与KA战队的差距便是显现了出来，虽然UB战队这边已经是很针对beng了，但是到了这个时候beng作为一名十王者，终于还是展现出了他的风采。
<br/>　　9分钟的时候对方的打野过来抓他，愣是被beng的妖姬天秀两人，最终还反杀了对方的打野。
<br/>　　这一波之后，beng的妖姬便是已经无人可挡，最终打出了6-0-5这样的豪华战绩，到了这一刻所有的粉丝对于十王者的选手终于是又有了一个新的认识。
<br/>　　“无敌，真的是无敌，beng的这个妖姬真的是太秀了，这一届的世界赛大家都打开了啊。”看着beng的表现这么好，香蕉满脸的感叹。
<br/>　　“越是这样才越有意思啊，江山代有才人出，各领风骚几年，这是咱们最后的时代了，未来还会有很多小艺和清浅这样的选手的。”叶帝脸上满是兴奋，对于接下来的比赛充满了期待。
<br/>　　“这几天大家也不能放松，还是和皇族他们打打训练赛吧，皇族的实力也是世界顶尖的，这样的话你们也能保持你们的手感，说实话他们没有进入世界赛真的有些可惜了。”李景禹跟着开口说道。
<br/>　　“皇族现在是最佳陪练了，别的赛区可是没有这样的资本的，让一个世界前8的战队来当陪练，恐怕也就只有咱们华夏赛区这么土豪了，现在想想都感觉66的。”说道这个，蒋念也是一脸的兴奋。
<br/>　　“估计现在白衣他们肯定特别郁闷，如果他们进世界赛的话，这个最佳陪练可就要沦为雷霆了。”夜雪狂歌这个时候脸上也是露出了揶揄的笑意。
<br/>　　这个时候既然一切都已经有结果了，他们自然是看的很开，而且这些话他们平时也都在聊，毕竟从那个时代走过来的职业选手，大家的感情都是十分的好的。
<br/>　　所以这一次皇族虽然进不了世界赛，但是他们却是真心希望三支战队能够取得好成绩的，最佳陪练这句话就是楚白衣自己说出口的。
<br/>　　至于为什么他们三支战队不打训练赛，这就要考虑到一些隐私的东西了，毕竟他们将来会是对手的，如果真的打训练赛，他们肯定无法放开，要藏写东西。
<br/>　　但是和皇族就不一样了，之前大家已经说明了，这个训练赛是要严格保密的，所以双方完全可以全力发挥，这样的话对于他们也会更有帮助一些。
<br/>　　就这样在看过KA战队的比赛之后，蓝队直接去到了另外一个比赛室准备和皇族的比赛，下午是属于他们的时间，到了晚上则是雷霆的。
<br/>　　至于白队和皇族的训练赛则是被排到了明天，这个时候最忙的倒不是叶帝他们这些进入世界赛的战队了，而是被淘汰的楚白衣等人。
<br/>　　而叶帝一行人在蒋念他们去比赛室的时候，也是来到了自己等人的比赛室，虽然没有战队和他们打训练赛，但是这并不妨碍他们自己训练。
