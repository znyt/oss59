　　这生命力，精粹到了一种无法想象的地步，甚至是比起古帝身上的生命力，还要精粹。
<br/>　　一些古帝身上的伤势，如果吞噬了这些生命力，都足以彻底恢复过来，甚至是给一个毫无灵力的凡人来，稍微吸入一些，都足以活上数百年上千年。
<br/>　　这种精粹到夸张的生命力，在楚风眠的记忆之中，唯有一个东西，才可能拥有。
<br/>　　建木树枝！
<br/>　　建木。
<br/>　　传说中开天辟地的神树，支撑天地的根本，据说先天神兽，都是从神树之上，诞生而出的存在。
<br/>　　建木蕴含的生命力，就连先天神兽都足以孕育的出来，其中的生命力达到了一种无法想象的地步。
<br/>　　以至于后面建木，无数建木树枝遗留下来，经历千万年的时间，其中蕴含的生命力依然是达到了一种恐怖的地步。
<br/>　　曾经楚风眠神力境的时候，就得到了一棵建木树枝，也正是因此，后来楚风眠才可以从荒古门的神树秘境夺走了那神树幼苗。
<br/>　　可以说现在楚风眠肉身的强横，很有程度上，都是跟曾经楚风眠得到的那一枚建木树枝，有些关系。
<br/>　　当初楚风眠得到的那一株建木树枝，才不过是三寸大小，而眼前的这建木树枝，比起楚风眠曾经得到的那一株，要庞大百倍。
<br/>　　其中蕴含的生命力，也达到了一种不可思议的地步，给一名古帝得到，一旦是融入到了身躯之中，几乎就是遇到再严重的伤势，都可以瞬间恢复，不损失任何的力量。
<br/>　　而现在，楚风眠的身躯之中，可也有着一株神树，跟这建木树枝，本属同源。
<br/>　　楚风眠已经是清晰的感觉得到，身躯之中神树的渴望，如此之大一株建木树枝，足以是让神树的力量，再度蜕变。
<br/>　　“给我撕裂！”
<br/>　　楚风眠看到这其中建木树枝的一刻，也是同时出手，又是凝聚灵力，一道龙爪轰击而去。
<br/>　　这龙爪轰击在了刚刚打出的裂痕之上，一下子让上面的裂痕，更加扩大，这宫殿的大门，被全部打碎。
<br/>　　楚风眠也一举踏入到了宫殿之中，近距离的感觉着眼前这建木树枝。
<br/>　　建木破碎，大量的建木树枝遗留了下来，但是寻常见到的建木树枝，也就是三寸左右的大小。
<br/>　　像是这半人高的，却是无比罕见，越完整的建木树枝，其中蕴含的力量也更加强大。
<br/>　　“给我摄取！”
<br/>　　楚风眠震惊一下，当即出手，一把抓起了这建木树枝，准备摄取，以免夜长梦多。
<br/>　　也许是因为楚风眠身上有着神树的关系，这建木树枝，都没有反抗楚风眠，一下子臣服，被楚风眠摄取到了手心之中。
<br/>　　可楚风眠还没有来得及开心，突然之间一股危险的感觉降临在楚风眠的脑海之中。
<br/>　　刹那之间楚风眠只能暂时放下这建木树枝，身躯连连后退，在刚刚楚风眠所在的地方，却是留下了一道五彩刀锋。
<br/>　　如果刚刚楚风眠犹豫一瞬间，这五彩刀锋，只怕就将楚风眠当场给斩杀了。
<br/>　　“五行神刀？五行天的人？”
<br/>　　楚风眠已经是见识过这五行神刀，以至于如今一下子认了出来，他目光无比冰冷。
<br/>　　这出手的人，在出手之前，楚风眠都没有察觉到此人的存在，这种实力，境界绝对在楚风眠之上，定然是一尊九劫古帝在出手。
<br/>　　在强大的八劫古帝，无论如何也不可能无声无息之间的偷袭楚风眠，唯有九劫古帝，才有可能做到这一点。

<br/>　　显然刚刚就是他，突然出手，偷袭楚风眠。
<br/>　　“居然来得如此之快！”
<br/>　　楚风眠目光闪烁，这老者的到来，比起楚风眠想象之中的还要更快。
<br/>　　本来楚风眠还是准备抢在此人的前方，收取了所有的宝藏，随即离开的。
<br/>　　现在看来，这计划是行不通了。
<br/>　　“哦？不愧是小天主都在意的人，果然有着几分本事，可以躲开老夫这一刀。”
<br/>　　在楚风眠思考的时候，那老者也在打量这楚风眠，看着刚刚楚风眠躲过了他的那一刀，眼神之中都闪过了几分惊讶。
<br/>　　他身为一尊九劫古帝，出手偷袭一个小辈，本就是无人想到的事情了，现在楚风眠居然是足以躲开，更是超出了他的想象。
<br/>　　“看来，在这附近的五行天古帝，都是死在了你的手中了吧。”
<br/>　　老者盯着楚风眠，语气之中带有着几分冰冷道。
<br/>　　“不错，是他们找死。”
<br/>　　楚风眠冷声说道。
<br/>　　反正现在他跟五行天，早就撕破脸了，也没有什么不能承认的。
<br/>　　“找死？有意思，小辈，我看是你在找死！”
<br/>　　老者盯着楚风眠的目光之中，越发的冰冷，一股怒意在他的身躯之中酝酿。
<br/>　　被这老者目光盯着的一刻，楚风眠仿佛是被一尊饥饿的野兽所盯上一般。

<br/>　　既然是今日，避战不了，那就战吧。

<br/>　　但是看现在的样子，不杀了这老者，楚风眠也根本没有机会脱身去收取宝藏。
<br/>　　一不做二不休，先杀了这老者再说。
<br/>　　九劫古帝又如何？
<br/>　　楚风眠在踏入到了帝尊境后，实力已经是跟九劫古帝没有任何的差距了。
<br/>　　“始祖第一式！”
<br/>　　群龙咆哮，灵力凝聚在楚风眠的掌心之中，大吼一声，一掌凌空落下。
