　　轰隆隆！
<br/>　　呼——！
<br/>　　呼——！
<br/>　　……
<br/>　　风暴不断，万重混沌巨浪滚滚而来，化作混沌风暴，撕裂虚空。天地具皆昏暗无比。虚空在咆哮，混沌在嘶吼。
<br/>　　一股无与伦比的毁天灭地的灭世之威，酝酿而来。
<br/>　　混沌道君本想要躲开，但却已经来不及了。它看似很慢，但实际上却是极快。只因为这风暴太大了。
<br/>　　来不及躲闪，一股死亡气息扑面而来，无论他此刻有多强，在这一刻都察觉到了自己的渺小。
<br/>　　巨大的混沌风暴卷起无尽混沌，连混沌如此神秘坚固的地方，此刻的空间都破碎变成无数巨大的黑洞。
<br/>　　混沌道君控制不住自己，他的身体无论如何用力，都不能摆脱那其中的无比巨大的撕扯吸力。
<br/>　　混沌道君此刻毫不犹豫地将自己花费无数神材炼制的太虚神甲穿在身上，欲以此来抵挡那恐怖的力量。
<br/>　　太虚神甲是混沌道君以太虚神铁为主材，加入许多其他的混沌神铁，炼制而成的。坚固无比，就连他的先天至宝弑神枪，都不能将其损毁分毫，而且它还能穿梭虚空。就是这神甲帮他度过了无数次混沌危机。
<br/>　　但此刻，面对着无尽混沌风暴，道君的太虚神铁好像并不起多大的作用，不能使他在这风暴之中运动分毫。它能够帮他的，仅仅是御去一些力道，挡住一些混沌的侵袭。
<br/>　　毁天灭地的撞击声不断传入道君耳中，几乎要撕裂他的耳膜，同时，他感受到自己的身体正在不听使唤地朝着风暴中心而去。
<br/>　　咻咻咻……
<br/>　　巨大的气浪席卷而来，有着粉碎一切的威能。道君只感觉自己被什么东西狠狠地撞击个正着，随即他便听到自己的太虚神甲在咯吱作响。
<br/>　　他顿时脸色一变，来不及多想，他一咬牙，将整个人变得无比细小，几乎是化成了一缕混沌之气，随着黑洞被吞噬进去。
<br/>　　随即，他感觉到一股恐怖无比的大吞噬、大毁灭、大混乱、大破灭的气息向他袭来，这一刻，他对于混洞法则的理解，在飞速上升着。
<br/>　　滚滚道韵从他身上涌起，但随即便被黑洞吞噬一空。
<br/>　　一阵眩晕，他整个人在此刻失去了知觉。
<br/>　　天地覆灭，万物尽皆被吞噬，没有留下一点痕迹。
<br/>　　许久之后，混沌之中那紊乱的场域才渐渐平稳下来，随着场域的平稳，在混沌之中那洞开的巨大黑洞，也逐渐地收拢，慢慢消失不见。
<br/>　　最终，混沌恢复了平静，只留下条巨大无比的混沌深渊，时不时地还散发出惊人的毁灭气息。谁也不曾想到，就在不久前，此地还有一场巨大的混沌风暴发生过。
<br/>　　……
<br/>　　不知过了多久，混沌道君悠悠醒来。睁开眼的刹那，他便看到了七彩氤氲的混沌仙气，北极仙光，若不是他还看见了破碎的大地以及空间，他还以为此处是一处仙境呢。
<br/>　　他努力转动下眼珠，却察觉不到自身的存在，浑身上下呈现出一种麻痹的状态，什么感觉都没有。
<br/>　　他神念一动，便知道自己此时的状态了。只见他此刻全身破烂不堪，他的两只手臂、两只腿以及半边身体，都化为了虚无，他此刻只剩下了一颗头与一具半残的躯体，全身上下血迹斑斑，就连太虚神甲也变得破败不堪。
<br/>　　显然，此次他受伤颇重。尽管他当时已经拼尽全力保护自己了，但奈何那混沌风暴实在是太强了，受伤也是无可奈何的。此刻没有死亡就算是好的了。
<br/>　　尽管他此刻四肢不全，仅剩下半残的躯体，但混沌道君并不担心，只要还剩下一滴血，他都能重生，更不用说他还有一半的躯体。
<br/>　　只见他运转功法，躯体一阵阵蠕动，渐渐地，他全身散发出耀眼的混沌神光。眨眼间，他那残缺的躯体便渐渐长了出来，就连他的四肢也开始慢慢长出来。
<br/>　　不过几个呼吸的时间，混沌道君的身体便重新长了出来。
<br/>　　“呼——”
<br/>　　混沌道君深深呼出一口气，他的脸色此刻变得苍白无比，显然，这番疗伤，对他并不是没有负担。
<br/>　　“还好，本源并没有太过受创，只需要百年时间的修养，便可恢复。”道君有些庆幸道。此次真是死里逃生啊。
<br/>　　此刻，道君才有时间来查看他周围的世界。
<br/>　　他庞大的神念向着周围一扫，顿时就发现了周围的环境。
<br/>　　他看到自己此刻正躺在一座悬浮的大陆之上。这大陆不大，只有方圆万里大小。大陆之上，一片破败，大地好像被打沉过，上面也没有什么生物。
<br/>　　同时，道君皱了皱眉，他也发现了，此处的仙气极为稀少，仅有零零散散的一片，长生物质也很少。也就是说，在此处修炼之人，寿命会大幅度减少。
<br/>　　同时，他更是发现了，此处的天地规则道则并不同于混沌或者洪荒，显然，这里并不是混沌，而是另一个他没有见过的世界。
<br/>　　他如今的一身实力，被此方世界大道规则封印了许多，但尽管如此，他还是有混元中期的修为。他不由得心中一定，混元大罗金仙在一方世界，已经不算弱者了。
<br/>　　他的神念渐渐往远处伸去，渐渐地，他能够感受到，此地的道则有损，它并不完全，缺失了许多道则。同时，他也感觉到，此处空间为一特殊的空间，在他的上方，混沌道君隐隐感受到，有一方更加强大的规则世界，在它的下方，也有一方世界。
<br/>　　混沌道君若有所思，此处介于两方世界的夹缝之中。他发现，两方世界，有些类似于仙界与凡界的关系。只不过这里的凡界规则太强大了，而仙界规则却有些不符仙界之名。
<br/>　　“看来，此方世界许久之前，一定是发生过什么大战，导致世界的规则有损。”混沌道君若有所思道。
<br/>　　道君想了想，并没有出去，而是继续在此地疗伤，就算此地没有浓郁的仙气，但他有各种从混沌之中带出来的灵果，道果。虽说用来疗伤是大材小用，但此刻他也顾及不了那么多了。对他来说，时刻保证自己的巅峰状态，才是最为重要的。也正是这种时刻小心的态度，使得他在混沌之中度过了无数危难。
<br/>　　百年后，在道君用了许多顶级灵果疗伤后，他的伤势已经完全恢复了。
<br/>　　“呼——”
<br/>　　道君睁开眼睛，轻呼一口气，站了起来。感受到自身的伤势恢复完全，实力也能够发挥长正常，便道：
<br/>　　“不错，不错，混沌出品，必属精品。”
<br/>　　“如今也该出去看看此方新世界了。”说着，混沌道君一步迈出，便已消失在原地。等再次出现，他却来到一处空间节点之处。
<br/>　　“虽说以我如今的实力，两方世界的界壁都可以轰碎。但，既然这里有空间节点，那也省去了我许多力量。”
<br/>　　话音刚落，他便对着节点处轻轻一指。顿时，一道混沌灵光绽放，化为一点轰向节点。
<br/>　　“轰！”
<br/>　　空间洞开，面前出现一道巨大的仙光通道，道君一步踏出，便进入到通道之中，消失不见。
<br/>　　眨眼时间，通道再次消失不见，那空间节点也随之消失。仿佛从来没有出现过。
<br/>　　……
