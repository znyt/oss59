　　五天之后，长安。
<br/>　　李二与长孙两人尴尬而无奈的对视着，方老太监有些恭谨的站在一边，静静等着皇帝陛下的下一次召唤。
<br/>　　“二哥！”长孙皇后手里拿着已经看过两遍的《白娘子传奇》，哭笑不得的对李二说道：“看来我们都弄错了！”
<br/>　　“错就错了吧，反正也没出什么大事儿。”李二双手撑在双膝之上，沉声说道：“不过，这混小子这个方法似乎真的很不错，这段时间长安附近的百姓已经很少讨论天灾之类的事情了。”
<br/>　　长孙皇后将手里的书翻到里后一章‘白素贞永镇雷峰塔’，叹了口气说道：“是啊，妾身听说全都在讨论白素贞的儿子到底能不能考上状元！您说乾儿也是，好好的故事干嘛非要写一半呢，这不上不下的，好生让人难过。”
<br/>　　“这有何可难过的，你是他母后，让他把后面的写给了送回来不就行了，左右他在外面也是无所事事。”李二满不在乎的一摆手，对长孙皇后的那种纠结有些不以为然。
<br/>　　在大唐最尊贵的夫妻俩的讨论中，一场差点挑起儒佛两家争斗的事情就像是没有发生过一样。
<br/>　　李二在乎的是李承乾的想法与态度，只要那小子不是在故意胡乱折腾，那么出一点点小小的乱子其实也可以理解，必竟混小子出发点是好的。
<br/>　　而长孙皇后同样如此，她关心的是李承乾的安全和身体情况，至于有极少一部分人因为一部小说而产生心理上的郁结，同样不重要。
<br/>　　可是作为‘不重要的小事情’中的当事人，和尚们却十分郁闷，李二的不作为让他们感到很不能理解。
<br/>　　这一次的事件，和尚们完全就是受的无妄之灾，明明这事儿和他们一文钱关系都没有，可是偏偏庙却差点被人给砸了。
<br/>　　这都是因为那本破书，该死的，那书里的‘反面’人物只是一个个例啊！为什么要因为一个个例就影响到整个佛家体系，这帮读书人难道读书都读傻了？书都读到驴肚子里了么？
<br/>　　“怎么样，有没有找到书的作者？”兴善寺后院主持方丈禅房内，几个周边寺院的主持齐集一堂，讨论着这次的争斗事件。
<br/>　　“没有，长安周边的寺院全都查过了，没有一个叫‘明空’的弟子。”答话的是一个中年和尚，如果杨天和称心在的话，一眼就能认出来，这就是上一次被他们打劫的那个笨家伙。
<br/>　　灵感寺主持大师听了中年和尚的结论之后眉头一皱：“那这就奇怪了，难道这‘明空’真的是一个俗家之人么？但如果是俗家之人为何要起如此有禅机的名字呢。”
<br/>　　中年和尚摇头说道：“这就不知道了，只是近几代弟子当中根本没有‘明’字辈的，所以小僧认为这人根本就不是佛家弟子。”
<br/>　　中年和尚虽然在兴善寺中有些地位，但是如果和在坐的这些老僧们比起来，却是差了不是一点半点。
<br/>　　就以刚刚说话的灵感寺主持例，这灵感寺说起来建寺的时间并不长，是隋文帝开皇二年（公元582年）所建。但却是佛家八大宗派之一密宗的祖庭，当然，在大唐的时候也叫真言宗。
<br/>　　想想看吧，一个宗派祖庭的主持，那会是什么身份，中年和尚就是再牛逼，也不敢在他面前多有个一言半语。
<br/>　　“唉！”坐在另一边的草堂寺主持叹了口气，啧舌道：“那怎么办？如是书的作者不是佛家弟子，这却去哪里找他？”
<br/>　　中年和尚再次摇头，却没有接话，他知道这话不是问他的，如果没有什么好的主意或者办法，最好还是不要开口，否则被训斥一顿连个为他说好话的没有。
<br/>　　这个草堂寺主持虽然表现的不是那么咄咄逼人，但人家也不是善茬，这草堂寺名字不怎么响亮，但却是‘三论宗’的祖庭，而且草堂寺还是中国历史上第一座国立翻译场，因后秦皇帝出资请西域高僧鸠摩罗什在此翻译佛经而得名。
<br/>　　长安城内城外算在一起，寺庙数十，有四个佛门宗派发源于此，哪一个说起来都不是善茬。
<br/>　　此时此刻这些佛教大佬们聚在一起正火急火燎的为了《白娘子传奇》发愁，中年和尚自然不会自己找不痛快，没事儿去招惹他们。
<br/>　　不过很快这中年和尚就想起一件事情，犹豫了一下，抬头看看兴善寺的主持大师。
<br/>　　兴善寺主持大师也是也明白事理的，知道如果不让这中年和尚把话出来，怕是会引起其他诸人的误会，于是一声轻咳：“有什么话你就说吧，不要吞吞吐吐的。”
<br/>　　“主持，朝庭这段时间有了新举措，好像那些新出版的书里都加了一句话。”中年和尚纠结着说道。
<br/>　　“加了什么话，一次说清楚。”兴善寺主持恨声说道。
<br/>　　自己这个弟子什么都好，就是见不得大人物，看到大人物腿就哆嗦，连正常的思维都没有了。
<br/>　　“哦，里面加的是‘本故事纯属虚构，如有雷同，实乃巧合’！”中年和尚说完之后，心中很是忐忑。
<br/>　　这话说的好像多少有点扯淡的意思在里面，就像卖刀的商贩对人说：“这刀能杀人啊，所以如果杀了人我可不负责。”
<br/>　　果然，房间中几个老和尚听了之后被气的浑身直哆嗦。
<br/>　　“混账！”灵感寺主持狠狠一握拳头，咬牙说道：“简直欺人太甚！”
<br/>　　“阿弥陀佛！”兴善寺主持亦是宣了一声佛号，沉吟不语。
<br/>　　《白娘子传奇》给佛门造成的损失是十分巨大的，甚至可以说给一些信徒也造成了一定的心理阴影，这些东西绝不是用一句‘本故事纯属虚构’能抵消的。
<br/>　　可是现在佛门却没有在大卖，卖的几乎所有读书人人手一本。
<br/>　　而这些书卖的越多，对佛门的影响也就越大，读书人都是些执拗的性子，如果一旦被他们认为是不好的东西，那么激烈的抨击很快就会随之而来。
<br/>　　怎么办？到底应该怎么做？坐视不理？还是把那些新发行的书全都买回来？
<br/>　　老僧们思来想去最后发现，他们对这件事情真是无能为力，现实中最好的办法还真就是坐视不理，静静的等着这件事慢慢淡化下去。
<br/>　　当然，老僧们现在还不知道他们其实也就是躺枪，否则估计会直接气的喷出一口老血出来。
<br/>(本章完)
