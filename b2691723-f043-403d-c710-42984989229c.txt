　　第351章
<br/>　　程大雷是有准备动刑的，不施雷霆手段，怎显金刚面目。但其实，事情比他想得要顺利。
<br/>　　几乎用不着所谓的审讯手段，稍微一哄一吓，这些人便乖乖交待了。
<br/>　　所得到的消息，要比想象中多。
<br/>　　宋伯康自从就任凉州牧之后，便开始拉拢各方面关系，暗暗准备。程大雷这个冬天很忙，宋伯康也绝对没有闲着。
<br/>　　所做的一切，是针对程大雷而来。『能用则用，不能用则除掉』，这是宋伯康秉持的方阵，也恰恰是上面人的意思。
<br/>　　上面？上面是谁。
<br/>　　明帝，将军府，还是哪位权高位重的丞相大人。
<br/>　　透过宋游渠交待的话，程大雷发现一个惊人的事实。
<br/>　　这西北凉州，原来的守将是西北元帅林问天。但后来林问天死了，借着西北兵败，帝国杀了很多人。
<br/>　　人死了位置就会空出来，空出来的位置就要被其他人顶上，顶上的这些人是帝国的人，自然……也是相府的人。
<br/>　　西北是相府的西北。
<br/>　　莫鸣米负责采铁冶兵，自然有人负责垦田屯粮，也同样有人训练马匹。
<br/>　　帝国丞相，在西北经营诸多年，所图为何……
<br/>　　原因怕是连傻子都会想到，原因怕是连傻子都会不寒而栗。
<br/>　　这些事情本来不该程大雷知道的，甚至钟武才都没有资格知道。但宋游渠是宋伯康的儿子，话里话外自然会泄露一些出来。
<br/>　　如此，程大雷知道了不该知道的事。
<br/>　　本来安排得都挺好，将西北完全掌握在自己手中，一朝顺流而下，气吞江山。
<br/>　　可无端端冒出个程大雷，杀了莫鸣米，占了琴川关。在相府只手遮天的西北，狠狠扎上一颗钉子。
<br/>　　倒也不能说无端端冒出来，因为程大雷这个琴川关守将是明帝亲口封的。
<br/>　　思及此处，程大雷不由得对那位宫中的老人刮目相看。你若说他糊涂，他什么都明白，你若说他明白，他有时候却又蛮糊涂。
<br/>　　“大当家，这些人该怎么处理？”徐神机道。
<br/>　　“干脆一刀一个杀了，留着他们也是浪费粮食。”因醉酒闹事被关起来的高飞豹隔着牢门大喊。
<br/>　　“你给我闭嘴吧。”程大雷白了他一眼，目光在宋游渠几人身上扫过。
<br/>　　那是令人心惊胆战的目光，眼神如刀子，似乎能割破大家的喉咙。
<br/>　　程大雷忽然笑了，嘎嘎嘎的声音，像被握住脖子的鸭子。
<br/>　　“诸位，告诉你们一个好消息，我决定放你们走了。”
<br/>　　“真的？”宋游渠眼睛一亮，失去自由，他才发现自由真特么是个好东西。
<br/>　　“当然，本寨主说一不二。”程大雷道：“可我把你们放走了，你们会不会码人过来复仇？”
<br/>　　“一定不会。”宋游渠道：“只要程当家肯放了我们，今天的事情就当没有发生过。”
<br/>　　对程大雷的称呼，从『狗贼』变成『程当家』，但宋游渠心里却咬牙切齿：只要自己从此地脱身，必要杀程大雷以雪今日之辱。
<br/>　　“这只是你一面之词，也不要怪我信不过你们。”程大雷道：“不过我可以先放你们其中一人，回凉州向宋大人解释今日事情，等宋大人明白今天只不过一场误会，派人过来接你们回去。”
<br/>　　程大雷顿了顿：“你们谁先走？”
<br/>　　众人目光闪烁，如今身在此地，生死不知，谁先离开就能保一条命。
<br/>　　“宋公子，我先走一步，改日再接大家离开。”钟武才。
<br/>　　“嗳，钟兄脚力不行，还是我先走一步，可以速去速回。”一个白衣人道。
<br/>　　众人你一言我一语，吵成一片。生死之外无大事，什么荣华富贵，先活下来再说。
<br/>　　看着这一幕，薛问婷从心里涌起一阵反胃。如此争执，丑态毕露，岂不是被山贼看了笑话。
<br/>　　“都住了吧。”薛问婷突然喝了一声，道：“宋哥哥，你先走，你的话，伯父才会听。”
<br/>　　事实上，薛问婷明白，让自己先离开此地，才是最保险的。身为女儿身，在这山贼窝子里，要比男人多太多危险。她也明白，宋游渠绝不会让自己留在此地，定是安排自己先离开。
<br/>　　宋游渠微微顿顿，道：“好，婷妹你放心，不出三日，我必接你离开。”
<br/>　　薛问婷一怔，心凉了大半。
<br/>　　宋游渠目光看向程大雷，道：“程当家，与我松绑吧，我答应你的事，绝对算数。”
<br/>　　程大雷目光看着他，嘴角露出笑意，宋游渠恍然发现，这笑容是嘲讽的笑。
<br/>　　“你当真觉得，谁走谁留，轮得到你做主么？”程大雷难以置信道。
<br/>　　宋游渠一怔，就见程大雷长身而起，走到他面前，拍了拍他的脸颊。
<br/>　　“年轻人呐，你怎么会这么幼稚呢。”程大雷道：“你们这些人就是未来帝国的继承人么，帝国的前途还真是令人担心呐。”
<br/>　　宋游渠想不明白，从宋伯康出现在西北这一日，他就成了程大雷的敌人。他们面带微笑，披着和平的袍子，可各自都握着刀子，准备找机会割断对方的喉咙。
<br/>　　这种状况下，程大雷好不容易抓住了宋游渠，薛问婷这样的大人物，怎么肯轻易放手。
<br/>　　这就是人质呐。
<br/>　　宋游渠等人想不明白这个问题，以为只要略略降低尊严的底线，比如面对程大雷时带上笑容，就可以平安离开这里。对这个乱世而言，他们太过幼稚。
<br/>　　程大雷离开牢房，踱步在初春的阳光下。
<br/>　　能抓到宋游渠一伙，对程大雷来说自然是意外之喜。但就算没有抓住，对程大雷来说其实也不算什么。手握一万精兵，据坚城而守，宋伯康就算真的攻来，程大雷也抗得住。
<br/>　　程大雷心里想的，还是如何磨练自身，降服体内的怪兽。只要能将那恶兽驯服，它的力量才是真正属于自己的。
<br/>　　最后，程大雷选择放钟武才离开，由他回凉州报信。
<br/>　　钟武才离开琴川关后，立刻像逃离魔窟一般，快马加鞭狂奔回凉州城。
<br/>　　在凉州城，他见到了凉州牧宋伯康，战战兢兢的把事情说了。
<br/>　　自然过程中自然添油加醋，删减了一些东西。
<br/>
