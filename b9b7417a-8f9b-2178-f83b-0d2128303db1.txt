　　“海眼就在前面了，小心点啊大伙，水里有不少巨妖！”徐文长提醒道。
<br/>　　“水妖？好玩啊。”
<br/>　　橙子说完，一头扎进了水中，果然，原本平静的水面猛然泛起巨浪，从四面八方朝她卷去，似乎下面有什么哦东西在朝她身边游，而且不止一只。
<br/>　　“等等我们！”
<br/>　　美华、小青小白三人争先恐后地跳下去，其余人就在岸边等着，也不担心，这四个家伙不是水鬼就是水妖，在水中的战斗力无比强悍，只看见一大片水花翻腾，过了有十来分钟左右，突然有两个庞然大物浮出水面，一条青蛇，一条白蛇，两条蛇体形都巨大无比，口中吐着长长的信子，虽然知道他们是小青小白，但看上去还是触目惊心。
<br/>　　“老大，你们都爬到我背上来，我背你们过去，这样快一点！”小青回头喊道。
<br/>　　“妹子们到我背上，其余人去小青背上。”小白强调，于是叶少阳一干人爬到小青背上，小九现出真身，跟四个婢女一起上到小白背上，瓜瓜和包子也爬上小白背上，因为他们是小孩，小白也不在意。
<br/>　　人都上去之后，两条蛇速度飞快地朝前方游去，叶少阳坐在小青的头上，两边水花飞溅，这种感觉不要太爽，叶少阳没冲过浪，但感觉冲浪应该就是这种感觉，而且速度没这快，也没这稳。
<br/>　　“爽啊！”四宝也兴奋地大叫，“山羊，你说咱们以前怎么没想到这么玩，小青，以后你多回人间啊，没事我们就去海边冲浪如何？”
<br/>　　“去你的，我又不是你玩具！”大蛇口中发出含糊的声音。
<br/>　　突然，一条金色的美人鱼从水中腾空而起，带起一滩绚烂的水花，叶少阳一行人呆呆地看着这如同童话世界般绮丽的一幕。
<br/>　　“老大，来我背上吗？”橙子大声喊，一头又扎进水中。
<br/>　　叶少阳看了小九一眼，不太情愿地说道：“不了你自己玩吧。”
<br/>　　转头问小青：“美华哪里去了？”
<br/>　　“她去驱赶那些邪物了，免得它们靠近。”
<br/>　　橙子也好久没玩水了，开心地在水中游来游去，不时从水面上跳出来，洒下一串水花。
<br/>　　两条大蛇，驮着众人，航行在这看上去无边无际的海洋之中。
<br/>　　这画面，简直美到了极致，以至于没有人开口说话，大家都沉浸在这美妙的体验之中。
<br/>　　多年以后，其中部分历经天劫而幸存下来的人里面，还都时常会想起今天这一幕。
<br/>　　小青小白根本就不用费力，因为水是不断往前流淌的，他们只需要漂浮在水面上就行。
<br/>　　“你们看！”
<br/>　　瓜瓜偶然回头的时候，突然发现了什么，大叫起来。
<br/>　　众人一起回头，看到远处有一些黑压压的人影，正在快速地靠近。
<br/>　　“是太阴山的人吗，怎么这么快就来了！”叶少阳看向徐文长。
<br/>　　徐文长皱眉看了一会，道：“没道理啊，而且看着人也不多，不太可能是鬼兵。”
<br/>　　叶少阳于是让小青小白停下来，等了一会，等他们靠近了才发现，根本不是太阴山的鬼兵，而是慕寒，带着一些龙虎山的弟子，后面还跟着慈心师太、王道乾等一干宗门的大佬，叶少阳一眼扫过去，没看到逍遥飞和静慧师太。
<br/>　　“见过徐师爷。”慕寒带领大家向徐文长行礼。
<br/>　　叶少阳注意到，他们脚下都踩着一只纸船，上面还有朱砂符文，一看就是灵符幻化而成，这种灵符自己也会，有符文加持，可以御水而行，速度也很快，只是之前觉得不需要弄这东西。
<br/>　　“你们来做什么？”徐文长脸色很不好看。
<br/>　　慕寒拱手道：“阴司有难，我等人间法师也不能坐视不管，况我乃是天选之子、人间法术界的盟主，这种事当然要贡献微薄之力，擅自行动，还望徐公见谅。”
<br/>　　徐文长微惊，忙问道：“你们从哪里听说的这次行动？”
<br/>　　“自然是有人告知，徐公，这事你不找我，但我等得知之后，定要要保卫阴司，前来助力，还请徐公应允！”
<br/>　　徐文长转头冲叶少阳道：“你看看人家这态度。”
<br/>　　“那你找他呗，找我干啥。”
<br/>　　“说你胖你就喘上了。”徐文长扫了慕寒等人一眼，道：“既然你们都来了，我也不好阻拦，不过此次行动十分危险，你们要做好准备，万一出什么意外，我可负责不了。”
<br/>　　慕寒道：“我等俱为自愿，不需徐公负责。”
<br/>　　徐文长再没说什么，对叶少阳道：“我们走吧。”
<br/>　　叶少阳却犹豫起来。
<br/>　　慕寒看到到这一幕，轻笑道：“莫非叶掌教怕我等抢功？”
<br/>　　“你以为你谁呀！”四宝一脸厌恶。
<br/>　　徐公道：“既然他们来了，就跟着吧，我也阻止不了他们。”
<br/>　　叶少阳于是拍了拍小青的脑袋，让他开路。
<br/>　　再往前不远，便能依稀看到一座岛屿，矗立在海水中间，很快就到跟前，这才发现了一个神奇的地方：四周的海水，都在往海岛方向移动，似乎这海岛是漂浮在水面上的，四周奔袭过去的海水，都被吸了进去，整个海面因此形成了一个巨大的漩涡。
<br/>　　如此奇景，令所有人惊叹不已，眨眼间大伙已经来到岛边，本想直接上岛的，结果快登岸的时候，小青咔的一声脑袋撞在一道无形的结界上，舌头都撞出来了。
<br/>　　“何人擅闯海眼禁地！”岛上传来一个悠长而严厉的声音。
<br/>　　众人伸头望去，只看到一片迷雾。
<br/>　　徐文长飞起在空中，亮出大帝的冥王令，朗声说道：“我乃轮回司师爷，受命前来开辟海眼封印，事急从权，个还请冲和子师兄放我等上岛。”
<br/>　　虽然岛上的人不知道他们的目的，当冥王令放出的通透灵光，是做不了假的，迷雾之中沉默了半晌，只听那声音说道：“你们上来吧。”
<br/>　　小青伸着蛇头，望前方探了探，结界已经没了，于是游上了海岛。
<br/>　　就在众人落地的时候，岛上的迷雾渐渐褪去了，露出了一片沙地，在岛屿的最高处，有一高台，一共九层，顶端耸立着一座巨鼎。
